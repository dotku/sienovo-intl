## 概述

USB‑XM2404 是一款基于 USB 2.0 总线的高速、高精度 4 通道同步采集卡，能够实现 **24 位** 精度、**156 KSPs**（千样本每秒）的采样率。该卡支持 **外部触发** 与 **软件定时触发** 两种采集方式，适用于实验室、质量检测、医学检测以及工业过程监控等多种场景。本文将围绕卡片的硬件结构、关键技术指标、典型应用以及使用注意事项展开详细说明，帮助读者快速上手并充分发挥其性能。

> **原文摘录**  
> USB-XM2404板是一款USB2.0总线高速、高精度4通道同步采集卡，高达24位精度、156KSPs采样率。本卡支持外触发和软件定时触发。4路模拟信号、1路外触发信号（下降沿触发）通过板上的IDC-20接头P1口输入。16路DI、16路DO通过IDC-40接头P2口输入、输出。

### 采集信号路径

- **模拟输入**：4 路差分信号经 **AD7765**（四片独立的 24‑bit Σ‑Δ ADC）转换为数字数据。AD7765 自带差分放大器和数字滤波器，能够在 **±5 V** 或 **±10 V** 双极性范围内工作，提供 **112 dB** 以上的信噪比（SNR）在 156 kHz 采样率下的最小值。  
- **存储**：转换后的数据写入板载 **4 M 字**（约 8 MB） SDRAM，支持连续采集，满足实时监控需求。  
- **数字 I/O**：16 路数字输入（DI）和 16 路数字输出（DO）通过 IDC‑40 接口（P2）实现，输出带锁存功能，上电自动清零，便于外部逻辑控制。

### 主控与接口

- **USB 主控**：采用 Cypress **CY68013A（FX2）** 低功耗芯片，提供 GPIF（通用并行接口）胶链接传输，确保高速数据流。板卡支持 **热插拔**，即插即用，兼容 Windows XP/7/8/10 系统。  
- **供电**：默认通过 USB 供电（5 V），若 USB 电流不足，可外接 5 V 电源；板载电源管理芯片自动判断供电来源，并具备防接反保护与 1 A 保险丝。

## 硬件结构与布局

1. **模拟前端（Analog Front End, AFE）**  
   - 每路模拟输入经 **差分放大器**（内部集成于 AD7765）实现共模抑制。  
   - 差分信号通过 IDC‑20 接口 P1 输入，支持 **4 通道** 同时工作或按需选择。  

2. **数字处理**  
   - AD7765 输出的 24‑bit 数据直接写入 SDRAM，FPGA（内部逻辑）负责 **数据打包、时序控制** 与 **USB 传输**。  
   - FPGA 采用 **Cypress FX2** 的 GPIF 接口，实现 **双缓冲** 机制，保证在高速采样时不出现数据丢失。  

3. **数字 I/O**  
   - 16 DI 与 16 DO 通过 IDC‑40 接口 P2 进行 **并行** 交互。DO 端口带锁存寄存器，能够在外部触发时保持输出状态，适用于脉冲计数或状态保持场景。  

4. **电源管理**  
   - **USB 5 V** 与 **外部 5 V** 两路供电通过 **OR-ing** 电路自动切换。防接反二极管与 1 A 保险丝提供基本过流保护。

## 应用场景

- **实验室测试**：利用高精度 24‑bit ADC，可对低噪声传感器、精密仪器输出进行波形捕获与频谱分析。  
- **产品质量检测**：在生产线上对电子元件的电压/电流特性进行快速采样，配合 PC 端软件实现自动化判定。  
- **医学检测**：对生物电信号（如 ECG、EEG）进行同步多通道采集，确保无相位差的高保真记录。  
- **工业过程监控**：结合 16 DI/DO，可实现对机器状态的实时监控与控制，如阀门开闭、报警触发等。

## 使用指南

### 1. 驱动与开发库

- 出厂随卡提供 **Win10/Win8/Win7/Vista/XP** 下的驱动程序、DLL 动态链接库以及 **DEMO** 示例。  
- 示例代码覆盖 **VB、VC、LabVIEW** 三大平台，用户可直接改写为自己的采集逻辑。  

### 2. 基本连接

1. 将卡的 **USB** 接口插入 PC（或笔记本）USB 2.0 端口。  
2. 若 USB 供电不足（部分笔记本供电限制在 500 mA 以下），请使用 **外部 5 V** 电源适配器并确保正负极正确连接。  
3. 使用 **IDC‑20** 接口将四路差分模拟信号（或单端经差分转换）接入 P1，外部触发线（下降沿）同样接入 P1 的第 5 脚。  
4. 通过 **IDC‑40** 接口将数字 I/O 线接入 P2，注意 DO 端口在上电后会自动清零。  

### 3. 软件配置

- **采样率**：在驱动提供的配置界面或 API 中设定目标采样率（1.95 kHz ~ 156.25 kHz）。  
- **通道使能**：可单独启用任意通道，或一次性开启全部 4 通道。  
- **触发方式**：选择 **软件定时触发**（适用于周期性采样）或 **外部下降沿触发**（适用于事件驱动采样）。  

### 4. 数据读取

- 通过 DLL 提供的 **ReadData** 接口读取 SDRAM 中的采样数据，返回 **24‑bit** 原始值（左对齐至 32 位）。  
- 若需要实时显示波形，可在 PC 端使用 LabVIEW 或 Python（通过 ctypes 调用 DLL）实现 **双缓冲** 读取，避免因 USB 传输延迟导致的卡顿。  

### 5. 参数存储

- 卡片内部拥有 **4 K 字节 EEPROM**，用户可写入自定义参数（如板卡 ID、校准系数）。  
- 在同一台 PC 上插入多块 USB‑XM2404 时，系统会通过 EEPROM 中的 **ID（0~15）** 区分不同卡片，确保数据流向正确。

## 维护与保养

- **温度范围**：0 ℃~70 ℃ 为工作温度，建议在此范围内使用，避免高温导致 ADC 稳定性下降。  
- **防潮**：相对湿度 5%~95% 均可正常工作，但长期高湿环境可能导致 PCB 表面腐蚀，建议在干燥环境中存放。  
- **防误接**：外部电源防接反二极管与 1 A 保险丝已提供基本保护，仍建议在连接前检查电源极性。

## 结论

USB‑XM2404 通过 **USB 2.0** 与 **Cypress FX2** 主控实现了高速数据传输，配合 **AD7765** 四路 24‑bit Σ‑Δ ADC，能够在 **156 kHz** 采样率下提供 **112 dB** 以上的信噪比，满足高精度、多通道同步采集的需求。其丰富的数字 I/O、灵活的触发方式以及即插即用的驱动支持，使其在实验室测试、质量检测、医学信号采集以及工业监控等领域都有广泛应用。通过本文的介绍，读者应已掌握卡片的硬件结构、关键参数、基本使用流程以及常见的维护要点，能够快速将 USB‑XM2404 集成到自己的测量系统中，实现可靠、精准的数据采集。
