应用场景

随着我国高新 技术 的发展和国防现代化发展，航空、航天等领域需 要的大型光电子器件，微型电子机械、 光 电信息等领域需要的微型器件，还有一些复杂零件的加工需求日益增加，这些都需要借助精密甚至超精密的加工检测设备

##### 客户需求

高 精度 高效率

### 产品规格

处理器 Processor

Intel® 6th/7th Gen Core ™ i7/i5/i3/Pentium®/Celeron® Processor, LGA1151 Socket, TDP up to 95W

芯片组 Chipset

Intel® Q170

内存 Memory

Dual Channel, 4 x U-DIMM Slot, DDR4 2133/2400MHz, up to 64GB

存储 Storage

4 x SATA3.0, 1 x M.2 2242/2280 (SATA3.0/PCIe3.0 X1 supported)

显示 Display

Intel® HD Graphics

网口 Ethernet

1 x Intel® i219-LM GbE, 1 x Intel® i210AT/211AT (10/100/1000 Mbps)

音频 Audio

Realtek® ALC888S HD Audio codec, supports 2.1-channel

BIOS

AMI UEFI BIOS

外部接口 Rear I/O

1 x DVI-D: max resolution up to 1920 x 1200@60Hz 1 x VGA: max resolution up to 1920 x 1200@60Hz 1 x DP: max resolution up to 4096 x 2304@60Hz 2 x USB2.0, 6 x USB3.0 2 x RJ45 1 x RS232/422/485 (with auto flow control) 1 x Line-out, Line-in, Mic-in

内部接口 Internal I/O

1 x RS232/422/485 2 x RS232/485 2 x RS232 5 x USB2.0 (Type-A share USB signal with one USB2.0) 1 x 16 bit GPIO 4 x SATA3.0 1 x PS/2 (for Keyboard and Mouse) 1 x LPT

扩展接口 Expansion

2 x PCIe3.0 X16 (1 x 16 or 2 x 8 link) 3 x PCIe X4 (X4 link) 2 x PCI 1 x M.2 2242/2280 (SATA3.0/PCIe3.0 supported)

Nuvoton TPM1.2 (optional, need MOQ)

电源输入 Power Input

ATX

尺寸 Dimensions

304.8 x 244 mm (12" x 9.6")

操作系统 OS
