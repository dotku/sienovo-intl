众所周知，医学影像设备是医疗器械行业中最具发展潜力的细分领域之一，诸如CT扫描仪（简称：CT）便是医生用于多种疾病临床检查和诊治时的理想选择。

**医疗CT扫描仪的发展潜力及难点**

CT，即电子计算机X线断层扫描，通过计算机 控制 调整X线束、超声波等，利用高灵敏度的探测器可以全角度地对人体逐个断层进行扫描产生数字信号，最终在计算机平台完成数据整合而得到一个详细的人体影像图，具有扫描时间快、数据信息全面、图像清晰等特点，可以将患者体内器官的微小病变、骨骼密度的异常等都显示出来，辅助医生更高效、精确的进行病灶诊断和制定治疗方案，因此得到广泛使用。

目前，CT已占据我国医学影像设备市场中最大的细分市场。据点石资本刊文，在2020年，我国CT的销售突破了7200台，2021年也超过了7000台，这两年的CT销售台数超过了前5年(2014-2019)CT销量的总和，未来随着我国经济发展、人口老龄化问题加剧以及市民医疗服务需求的持续增长，CT的市场规模也将进一步增大。

然而，医学影像设备的技术壁垒和造价成本双高，且关键技术、核心部件等长期被国外垄断，国产CT等医学影像设备发展还面临着行业集中度低、市面上满足其高分辨率、高灵敏度、高稳定性、专业化服务需求的计算机硬件产品少的难题。针对于此，信迈在医学影像设备领域积极布局，不断加深与行业内客户的交流合作，打造了支持CT扫描仪等多种医学影像设备应用的嵌入式计算机产品。

**工业整机方案** 

针对需要高性能计算与图形处理，专业化与定制化服务的医学影像设备等领域，推出了EPC工业整机方案，产品集成高性能CPU，丰富I/O接口设计，具备出色的图形性能、网络通讯能力与工业级高可靠性，支持CT扫描仪等多种医学影像设备应用。

产品性能如下：

支持Intel 10代 Core i3/i5/i7 LGA1200 处理器和Intel H420E/Q470E芯片组，支持2\*SO-DIMM内存插槽，支持DDR4-2400/2666/2933MHz，最大支持64GB，支持高速大容量数据运算与存储；

CPU集成图形控制，支持1\*VGA，1\*HDMI，1\* DP ，最大支持4K分辨率；

支持2\*SATA3.0，支持1\*M.2 2280 KEY M接口接入SSD硬盘(Q470E 支持PCIE X4的SSD，H420E 支持SATA的SSD)；

支持5\*Intel I226、1\*Intel I219自适应RJ45网口，6\*USB3.0，4\*USB2.0，6\*RS232/RS485串口，1\*M.2 2230 KEY-E WLAN/BT(Q470E支持PCIE/USB信号)，可实现高速网络通讯和多机联网控制；

支持1\*Mic-In，1\*Headphone，1\*AMP(3.2W 4Ω负载，5.2W 2Ω负载)；

支持1\*PS2 KBMS 键盘鼠标；

支持看门狗功能；

适配Windows 10/11, Linux操作系统；

ATX电源供电，主动散热设计，支持在宽温、复杂电磁环境等各种条件下稳定运行；

尺寸为230mm x 190mm x 90mm (W x D x H)

支持桌面/壁挂安装方式。

当前国家正加快推进智慧医疗和新医改建设，医学影像设备、体外诊断设备、医疗机器人等先进医疗器械的需求高涨。紧跟市场趋势，持续医疗器械领域嵌入式产品研发，可为行业内客户提供X86架构和 ARM 架构多样化、定制化的嵌入式计算机产品及服务。
