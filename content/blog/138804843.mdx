智慧步道是合作伙伴基于智能终端主板IoT\-3288A推出的一种采用智能化技术的健康运动方式，基于物联网、人脸识别、大数据分析等技术，对人们的运动进行监测和数据采集，显示运动数据，包括里程统计、热量消耗等，生成运动成绩排行榜，让群众健身更便捷、更有效、更智慧。

**核心主板方案优势**

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138804843/03_21f86ed5b3a8d8a00c74419db331ee01.png)

         该方案的核心主板采用的是基于RK3288平台的IoT-3288A，具备以下显著的优势：

1、性能强劲：IoT-3288A采用瑞芯微RK3288四核处理器，主频最高达1.8GHz，拥有强大的多线程运算能力、图形处理能力以及硬件解码能力，可为智慧步道提供更强劲的核心支持。

2、系统成熟：信迈拥有十余年的系统研发经验，历经千锤百炼，系统和硬件以及应用软件之间的配合可以无缝对接，在兼容性方面表现突出。同时，信迈可提供系统定制服务，提供系统调用接口 API 参考代码，完美支持客户上层应用 APP 开发。

3、品质可靠：IoT-3288A采用沉金工艺，核心板金手指耐腐蚀，双螺柱固定，通过了各种压力测试，支持7x24小时长时间无故障稳定运行，稳定可靠，久经市场考验，且价格更实惠。

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138804843/04_eb68f1d5b1342723679cde035a696518.jpeg)

**主板配置**

**核心配置**

CPU：RK3288四核处理器，主频最高达1.8GHz

操作系统：Android7.1/11.0

内存：标配2G(4G可选）

存储：标配16G（8G/32G/64G可选）

**显示接口**

HDMI输出 ：1 个,支持 1080P@60Hz，4kx2k@60Hz 输出

LVDS 输出：1 个单/双路，可直接驱动 50/60Hz 液晶屏

eDP输出 ：可直接驱动多种分辨率的 eDP 接口液晶屏

支持双屏异显

MIPI CAM：预留 MIPI CAM 接口，最高支持 1300W 像素

**扩展接口**

USB ：7 路USB2.0

串口 ：1 路 TTL，2 路 232、1 路蓝牙复用串口、3 路GPIO、1 路 ADC

存储卡： 支持 SD 卡

**网络支持**

无线网络：内置WiFi；蓝牙4.2

有线网络：支持10/100M自适应以太网

移动网络：支持PCI-E接口4G上网及通话
