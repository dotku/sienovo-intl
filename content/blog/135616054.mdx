## 概述

本文聚焦于 **NI PXIe-4480** 声音与振动模块，这是一款专为高带宽音频、声学和振动测量而设计的 PXI 采集卡。通过本文，您将了解该模块的关键技术参数、可配置的输入选项以及在实际测量中的典型应用场景，帮助您快速上手并在实验或生产环境中发挥其最佳性能。

## 主要特性

### 1. 高速采样能力  
- **采样率**：标准模式下最高 **1.25 MS/s**（每通道），在时域模式（用于捕获突发性事件）可提升至 **20 MS/s**。  
- **带宽**：最低可达 **3.4 Hz**，满足从低频振动到宽频音频的全频段需求。

### 2. 多通道与增益配置  
- **通道数**：提供 **6 个独立输入通道**，每通道均可独立配置。  
- **增益设置**：内置 **四档增益**（对应不同输入范围），用户可根据传感器输出幅度灵活选择，确保信号在 ADC 的有效范围内，最大化信噪比。

### 3. 多种耦合与激励方式  
- **AC/DC 耦合**：软件可配置的 **AC/DC 耦合**，适配不同测量需求；AC 耦合用于抑制直流偏置，DC 耦合则保留完整信号。  
- **电荷测量**：支持 **电荷模式**（Charge Mode），适合电荷型传感器（如压电式加速度计）。  
- **电压激励**：提供 **电压激励**（Voltage Excitation），可直接为传感器供电。  
- **IEPE 激励**：内置 **IEPE（Integrated Electronics Piezo-Electric）激励**，简化对带有内置前置放大器的传感器的接入。  
- **TEDS 支持**：符合 **TEDS（Transducer Electronic Data Sheet）** 标准，能够自动读取传感器的校准信息，提升测量的可追溯性与便利性。

### 4. 兼容性与软件支持  
- 与 **NI LabVIEW**, **NI DIAdem** 以及 **NI SignalExpress** 完全兼容，提供丰富的驱动与 API，便于二次开发和自动化测试。  
- 支持 **PXI 3.0** 及以上平台，能够在同一机箱内与其他高速采集卡、时钟模块等共享时钟资源，实现同步采集。

## 典型应用场景

| 应用 | 关键需求 | PXIe-4480 解决方案 |
|------|----------|-------------------|
| **音频测试** | 高保真音频捕获、宽频带分析 | 1.25 MS/s 采样率 + 6 通道并行采集，满足多路音频信号同步记录 |
| **噪声、振动与粗糙度 (NVH) 分析** | 低频振动检测、宽频噪声谱分析 | 3.4 Hz 低通带 + 多档增益，确保低频信号不被滤除 |
| **水下声学测量** | 水听器（hydrophone）信号采集、长时间监测 | IEPE 激励 + TEDS 自动校准，简化传感器接入与数据校正 |
| **高速爆炸或压力冲击** | 瞬态高频冲击捕获 | 时域模式下 20 MS/s 采样，完整记录冲击波形的上升沿与峰值 |

## 使用建议

1. **通道配置**  
   - 在 LabVIEW 中创建 **DAQmx Task**，为每个通道分别设置所需的增益和耦合方式。  
   - 对于低频振动测量，建议使用 **DC 耦合** 并选择合适的低增益，以保留直流分量；而对音频信号则可切换为 **AC 耦合**，去除直流偏置。

2. **激励方式选择**  
   - 若使用 **压电式加速度计**，推荐启用 **电荷模式** 并配合 **IEPE 激励**，这样可以直接利用模块内部的电源，避免外部供电噪声。  
   - 对于 **水听器**，使用 **电压激励** 并读取 **TEDS** 信息，可自动完成灵敏度校正。

3. **同步采集**  
   - 当与其他 PXI 卡共同使用时，务必将 **PXIe-4480** 的时钟源设置为 **共享时钟**（如 PXI-6652），确保所有通道在同一时间基准下采样，避免相位误差。

4. **数据后处理**  
   - 采集完成后，可利用 **NI FFT** 或 **Wavelet** 分析工具进行频谱、时频图的可视化。  
   - 对于突发事件（如爆炸），建议在 **时域模式** 下先进行 **高速采样**（20 MS/s），随后在软件中进行 **降采样** 与 **滤波**，以提取关键特征。

## 实际测量示例

下面给出一个简化的 LabVIEW 示例流程，帮助您快速验证 PXIe-4480 的基本功能：

1. **创建任务**  
   ```labview
   DAQmxCreateTask("AcqTask", &taskHandle);
   DAQmxCreateAIVoltageChan(taskHandle, "PXIe-4480/ai0:5", "", DAQmx_Val_RSE, -10.0, 10.0, DAQmx_Val_Volts, NULL);
   ```

2. **配置采样率**  
   ```labview
   DAQmxCfgSampClkTiming(taskHandle, "", 1.25e6, DAQmx_Val_Rising, DAQmx_Val_ContSamps, 1000);
   ```

3. **设置耦合与增益**（以 AC 耦合、增益 2 为例）  
   ```labview
   DAQmxSetAIACCoupling(taskHandle, "PXIe-4480/ai0:5", DAQmx_Val_AC);
   DAQmxSetAIGain(taskHandle, "PXIe-4480/ai0:5", 2.0);
   ```

4. **启动采集并读取数据**  
   ```labview
   DAQmxStartTask(taskHandle);
   DAQmxReadAnalogF64(taskHandle, 1000, 10.0, DAQmx_Val_GroupByChannel, data, 6000, &samplesRead, NULL);
   ```

> **提示**：实际使用时请根据传感器手册调整增益、耦合和激励参数，以获得最佳信噪比。

## 结论

NI **PXIe-4480** 通过 **6 通道**、**1.25 MS/s**（常规）/**20 MS/s**（时域） 的高速采样能力，结合 **AC/DC 耦合**、**电荷/电压/IEPE 激励** 以及 **TEDS** 自动校准功能，为音频、噪声、振动以及水下声学等多种测量场景提供了灵活且高效的解决方案。合理配置增益、耦合方式和激励模式，配合 NI 软件生态，可实现从实验室原型到工业级部署的全流程测量需求。
