#### 支持Intel Socket G2 Mobile Sandy/Ivy Bridge i7/i5/i3 处理器 ,Intel QM67/QM77,2\*LAN,2\*USB2.0,2\*USB3.0,3\*COM,3U

◆ 支持Intel Socket G2 Mobile Sandy/Ivy Bridge i7/i5/i3处理器,Intel QM67/QM77

◆ 1\*DDR3 SO-DIMM内存

◆ 1\* VGA ,1\*HDMI

◆ 2\*LAN,2\*USB2.0,2\*USB3.0,3\*COM

◆ 支持PCIE和PCI扩展

CPU

Intel Socket G2 Mobile Sandy/Ivy Bridge i7/i5/i3

芯片组

Intel QM67/QM77

内存

1\*DDR3 SO-DIMM内存插槽

存储

RTM：SATA2-2.5寸硬盘/CF卡可选

显示接口

VGA+HDMI主板模块外部接口  
1\*VGA,1\*HDMI

网络

1\*Intel WG82574千兆网口,1\*Intel Wg82577千兆网口

USB接口

2\*USB2.0,2\*USB3.0

COM口

3\*COM

I/O接口

PBP-09C05:采用AT或ATX供电,5\*扩展槽  
PBP-09C05P 采用ATX供电,5\*扩展槽,支持PCIE和PCI扩展  
AFC-964 扣板2\*USB,1\*串口,1\*VGA,1\*SPEAK-OUT,1\*MIC-IN,1\*POWERSW,1\*CF卡插座  
ACP-134S 扩展板扩展出1\*2.5寸SATA硬盘接口  
ACP-309S 扩展板扩展出3\*2.5寸SATA防振硬盘接口  
AFC-953 扩展板扩展出2\*串口,4\*GPIO,1\*USB2.0,1\*LAN,1\*SPEAK-OUT  
AFC-983U 扩展板扩展出8\*USB3.0接口  
AFC-952 扣板2\*USB,1\*串口,1\*HDMI,1\*SPEAK-OUT, 1\*POWERSW,1\*CF卡插座

BIOS

AMI EFI BIOS

系统

Microsoft Windows XP,Microsoft Windows7,Linux

温度湿度

工作温度:-20℃～60℃  
存储温度:-30℃～80℃  
相对湿度:不大于93%（40℃）

尺寸

465.4mmx 250mmx 133mm

安装方式

标准3U无风扇可上架
