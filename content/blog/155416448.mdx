ARM平台：

超微型EtherCAT运动 控制器 ,基于ARM架构开发,配置双核ARM处理器，具有1 x LAN,1 x EtherCAT,1 x MicroSD和Slice IO扩展

*   ARM-based的EtherCAT运动控制器最大支持32轴
*   双核ARM处理器
*   1 x Ethernet LAN，1 x EtherCAT，1 x MicroSD
*   支持 Modbus TCP、TCP/IP通讯
*   提供low-code软体开发环境-MotionNavi Designer
*   提供可以多任务开发的研华BASIC编程语言
*   Retain memory(FRAM)
*   支持PTP、直线和圆弧插补、路径表

x86平台：

超微型EtherCAT运动控制器，基于PC架构开发，配置了Intel Atom处理器，具有64GB eMMC、2 x LAN、2 x USB 和Slice IO扩展 PC-based的EtherCAT运动控制器最大支援32轴 Intel® Atom® 四核x6413E处理器 4 GB DDR4运行内存 64GB eMMC存储 配有2 x GbE、2 x USB、1 x HDMI M.2 B扩展槽，可灵活扩展 内置TPM2.0，基于硬件的安全的考虑 支持PTP、直线插补和圆弧插值、路径表 支持C/C++、VS.Net、LabVIEW编程语言
