## 板卡概述

FMC_XM131 是一款面向高端雷达、通信与导航仿真以及干扰系统的 **4 通道 12.6 GSPS、16 bit DAC** 子卡模块。板卡采用 **FMC+** 标准（符合 VITA 57.4 规范），通过 **JESD204B** 接口与 FPGA 前端高速串行端口（GTH）相连，能够实现高精度、宽带宽的模拟信号输出。每路模拟信号经 **50 Ω 特征阻抗的 SSMC 射频连接器** 以 **交流变压器耦合** 的方式输出，最大输出功率 ≤ 0 dBm。

## 核心技术要点

### 1. AD9172 / AD9176 DAC 芯片

- **AD9172**：支持最高 **1.54 GSPS** 的复数输入数据率，适用于对采样率要求相对较低但仍需 16 bit 高分辨率的场景。  
- **AD9176**：最高 **3.08 GSPS** 复数输入数据率，能够满足更宽频段的仿真需求。两者均提供 **48 bit NCO**（数控振荡器），最高可实现 **6 GHz** 的频率合成，满足宽频段调频、调幅等复杂波形的生成。

每个通道均配备 **独立 NCO**，实现多通道独立频率、相位控制，极大提升了系统的灵活性。

### 2. JESD204B 高速串行接口

- **8 线、15.4 Gbps** JESD204B 接口通过 **FMC+** 连接器直接映射到 FPGA 的 GTH 端口，实现低延迟、高可靠的数据传输。  
- 采用 **JESD204B** 标准，可在 FPGA 侧通过 Xilinx IP 核快速完成协议层的实现，降低硬件设计复杂度。

### 3. FMC+ 机械/电气特性

- **符合 VITA 57.4** 规范的标准 FMC+ 子卡，使用 **ASP‑184330‑01（HPC）** 连接器，提供 8 VCC、12 V、GND 等电源与信号引脚。  
- 支持 **1 片 EEPROM** 用于存储板卡序列号、校准参数等信息，便于系统级管理。

### 4. 时钟与触发

- 板载 **HMC7044** 高精度时钟芯片，提供 **外时钟模式** 与 **板载可编程采样时钟** 两种方案，满足不同系统对时钟源的需求。  
- 支持 **外部触发**（输入/输出）信号，实现多卡同步输出，适用于需要严格时间对齐的雷达阵列或多通道仿真系统。

### 5. 电源与散热

- **+12 V** 供电，整卡典型功耗 **6 W**，最大电流 **3 A**（±5%）。  
- 散热方式支持 **自然风冷** 或 **金属导热散热**，可根据实际安装环境灵活选择。

## 软件支持

- **板级软件开发包（BSP）**：提供针对 Xilinx 开发板（如 **ZCU102、KCU1500、VCU108**）的完整驱动与例程，帮助用户快速完成硬件到软件的迁移。  
- **自主研发板卡**：兼容我司 **KU、ZU、VU** 系列板卡，支持程序的直接移植，降低二次开发成本。  
- **定制化服务**：可根据客户需求提供 **算法实现** 与 **系统集成**，包括波形生成、频率合成、时序控制等功能。

## 硬件集成指南

1. **卡槽安装**  
   将 FMC_XM131 插入符合 VITA 57.4 的 FMC+ 卡槽，确保连接器对齐并锁紧卡扣。  

2. **电源接入**  
   通过卡槽提供的 **+12 V** 电源供电，检查电流不超过 3 A，确保电源噪声在规格范围内。  

3. **时钟配置**  
   - 若使用外部参考时钟，连接外部 10 MHz 或 100 MHz 时钟源至 **CLK_IN** 接口，关闭板载 HMC7044 的内部时钟。  
   - 若采用板载时钟，直接使用 HMC7044 输出的采样时钟，确保时钟抖动符合 DAC 的要求（< 1 ps RMS）。  

4. **触发同步**  
   将外部触发信号（TTL）接入 **TRIG_IN**，或使用 **TRIG_OUT** 将本卡的触发信号传递给后续卡，实现多卡同步。  

5. **FPGA 端口映射**  
   在 Vivado 中实例化 **JESD204B IP**，将 **8 线、15.4 Gbps** 的高速串行端口映射到对应的 GTH 引脚。使用 Xilinx 提供的 **JESD204B 示例设计** 完成链路校准与数据验证。  

6. **软件初始化**  
   - 加载 BSP 中的 **DAC 驱动**，完成 AD9172/AD9176 的寄存器配置。  
   - 通过 **I2C** 或 **SPI** 接口写入 EEPROM 中的校准参数。  
   - 调用 **NCO 配置 API**，设定每路通道的频率、相位与幅度。  

7. **波形输出**  
   将待输出的复数数据流（I/Q）写入 DDR，使用 DMA 将数据推送至 JESD204B 链路，实现实时波形播放。

## 结论

FMC_XM131 通过 **高采样率、宽带宽、16 bit 高分辨率** 的 DAC 与 **JESD204B** 高速接口相结合，为雷达、通信、导航以及干扰系统提供了一个 **即插即用、易于集成** 的模拟信号输出平台。板卡的 **可编程采样时钟、外部触发** 与 **独立 NCO** 能够满足从单通道到多卡同步的各种复杂需求。配套的 **BSP** 与 **定制化服务** 进一步降低了系统开发门槛，使得用户可以更专注于业务层面的算法实现与系统验证。
