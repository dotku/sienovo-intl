用vim打开/etc/vimrc  
vi /etc/vimrc  
在最下面加入：  
set number  
set tabstop=4  
如果在vimrc文件里已经有了  
set ai  
只是被用双引号注释掉了，那么就把双引号拿掉就好了  
如果没有set ai，那么加在最下面。  
  
  
我用的是CentOS5.3，经过如上设置后，发现普通用户的vim已经设置好了，但是root用户当使用vi命令的时候，打开的是vi而不是vim，所以还不能自动缩进和显示行号，解决方法如下：  
打开root用户的.bashrc文件  
vi /root/.bashrc  
在里面加入一行：  
alias vi=vim  
保存，退出root用户，再登录，就好了
