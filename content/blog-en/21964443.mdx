[The break statement](http://zhidao.baidu.com/search?word=break%E8%AF%AD%E5%8F%A5&fr=qb_search_exp&ie=utf8)
[The break statement](http://zhidao.baidu.com/search?word=break%E8%AF%AD%E5%8F%A5&fr=qb_search_exp&ie=utf8) is typically used in [loop statements](http://zhidao.baidu.com/search?word=%E5%BE%AA%E7%8E%AF%E8%AF%AD%E5%8F%A5&fr=qb_search_exp&ie=utf8) and switch statements. When `break` is used within a [switch](http://zhidao.baidu.com/search?word=switch&fr=qb_search_exp&ie=utf8) statement, it allows the program to exit the [switch](http://zhidao.baidu.com/search?word=switch&fr=qb_search_exp&ie=utf8) block and execute the statements following the [switch](http://zhidao.baidu.com/search?word=switch&fr=qb_search_exp&ie=utf8) statement. Without a [break statement](http://zhidao.baidu.com/search?word=break%E8%AF%AD%E5%8F%A5&fr=qb_search_exp&ie=utf8), it would result in an [infinite loop](http://zhidao.baidu.com/search?word=%E6%AD%BB%E5%BE%AA%E7%8E%AF&fr=qb_search_exp&ie=utf8) that cannot be exited. The usage of `break` within `switch` statements has been covered in previous examples when introducing switch statements, so no further examples will be provided here.
When the break statement is used within do-while, for, or while [loop statements](http://zhidao.baidu.com/search?word=%E5%BE%AA%E7%8E%AF%E8%AF%AD%E5%8F%A5&fr=qb_search_exp&ie=utf8), it can terminate the loop and execute the statements following the loop. Typically, the break statement is always associated with an [if statement](http://zhidao.baidu.com/search?word=if%E8%AF%AD%E5%8F%A5&fr=qb_search_exp&ie=utf8). That is, the loop is exited when a certain condition is met.
Note:
1) The break statement does not affect if-else conditional statements.
2) In nested loops, a single break statement only exits the innermost loop.
