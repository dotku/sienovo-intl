FPGA + HiSilicon ARM solution, capable of simultaneously receiving both HDMI and VGA signals, and outputting either signal or a stitched image via configuration.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138613790/01_b62d8bf26d2218ddd2ba5137170a29f7.png)

Customer Application: Wireless Remote Control

Key Features:

1.  Supports any resolution format below 2K.
2.  Supports H.264 compressed images.
3.  Supports Wi-Fi/4G wireless transmission.
4.  Supports adaptive input image resolution.
5.  Supports stitching HDMI/VGA dual-channel signals of any resolution into a specified resolution image.
6.  Customizable features, expandable applications.
