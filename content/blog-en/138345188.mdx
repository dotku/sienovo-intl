### 100 S/s/ch, 4-Channel C Series Universal Analog Input Module

The NI-9219 is designed for multifunction testing. The NI-9219 can be used to measure signals from various sensors (such as strain gauges, resistance temperature detectors (RTDs), thermocouples, load cells, and other active sensors), as well as performing quarter-bridge, half-bridge, and full-bridge current measurements, and features built-in voltage and current excitation. You can independently select different measurement types on each channel.
