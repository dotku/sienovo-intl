High-precision 16-channel vibration + 2-channel speed (24-bit) terminal acquisition board

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/144676579/01_f326c16eeaf3458b9b4ca06037147250.png)

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/144676579/02_5a2412101e944e2296275204b5f96abf.png)

Utilizing a fully domestically produced processor architecture featuring the AG16KF256 domestic FPGA and T3 domestic ARM. The design is divided into two boards: a baseboard and a core board. The baseboard is responsible for the acquisition of 16-channel signals and 2-channel speed data, signal filtering, phase alignment, and other tasks, transmitting the processed signals to the core board.

The core board uses the T3 processor, which is responsible for subsequent algorithm computations, data communication, data storage, and other functions.

The core board plugs into the baseboard via connectors.
