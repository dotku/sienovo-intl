FPGA + Actions ARM solution, using a single video source signal to simultaneously drive two LCD screens for 3D immersive playback.

Customer Application: VR Video Player

Key Features:

1. Supports playback of multiple video file formats
2. Supports real-time switching between 2D/3D effects during playback
3. Supports playback of files from TF card / USB drive
4. Supports customized configuration
