The XM-AMR-5728 series controller is a general-purpose controller designed for mobile robots (AMRs, intelligent forklifts, etc.), providing core functionalities such as map building, localization and navigation, and model editing for mobile robots. This product integrates the core components of mobile robots into a single unit. Coupled with the powerful client software Roboshop Pro, it helps users quickly set up robots. In addition, SRC has gradually evolved into a controller for smart factory infrastructure, capable of controlling automatic charging stations, automatic doors, elevators, traffic lights, and more. Under a unified scheduling interface framework, it drives the intelligent transformation and automation expansion of the entire factory.

**SRC-3000FS**

**Safety-Rated AMR Controller**

The SRC-3000FS safety-rated controller is an integrated navigation and functional safety controller. It provides mobile robots with basic functionalities such as map building, localization, and navigation, as well as advanced features like multi-robot scheduling, automatic charging, and 3D obstacle avoidance. Furthermore, it complies with IEC 61508, IEC 62061, and ISO 13849 standards.

By using a composite arm AMR equipped with 3D vision for CST recognition and grasping, Xinmai Intelligent's 3D vision solution offers absolute advantages in deployment speed, grasping accuracy, and recognition speed compared to traditional 2D and 2.5D solutions. This delivers a disruptive advantage, changing the ingrained perception that composite AMRs are slow in efficiency and recognition, and helps to promote this mature solution to a wider range of applications.
