#### I. Core Domestic Replacement Chip: TL63104 Controller

1.  ‌**Technical Specifications**‌
    
    *   Supports full PCIe 5.0 speed (32 GT/s), offering 96+8 Lanes configuration, with an aggregated bidirectional bandwidth of up to 128GB/s (x16 mode)1.
    *   Supports multi-mode topologies:
        *   ‌**Basic Mode**‌: Expands PCIe interfaces, supporting multi-level cascading;
        *   ‌**NT Mode**‌: Enables multi-PCIe domain interconnection, supporting NTB and DMA features;
        *   ‌**P2P Mode**‌: Direct transmission between any ports, reducing latency and improving bandwidth utilization19.
    *   Integrates SM2/SM3/SM4 national cryptographic algorithm engines, meeting domestic security requirements1.
2.  ‌**Interface Expansion Capability**‌
    
    *   Upstream Interface: PCIe 5.0 x16 (gold finger);
    *   Downstream Interfaces:
        *   1× PCIe 5.0 x16 (10146070-113Z0LF connector);
        *   4× PCIe 5.0 x16 (G97V24332HR cable interface);
        *   2× PCIe 5.0 x4 (G97V21352HR cable interface)1.

Application Scenarios
Basic Mode
Suitable for situations requiring an increased number of PCIe ports, such as when PCIe Lanes are insufficient in CPU servers, the 63104 can provide effective expansion.

NT Mode
Excels in environments requiring high availability, for example, enabling real-time backup and rapid failover between servers in a data center via NTB, enhancing overall system stability and security.

P2P Transmission
Particularly suitable for the GPU server domain, significantly improving data transmission efficiency and reducing latency by directly connecting different GPUs or PCIe SSDs.
