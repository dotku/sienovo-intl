**ITX-F601 All-in-One Motherboard Product Specification Sheet**

1.  ****Product Overview****

The ITX-F601 is a high-performance all-in-one motherboard, designed and developed based on the domestically produced Phytium D2000/8-core processor. Featuring a low-profile I/O design, it can be used in industrial all-in-one PCs, industrial tablet PCs, and similar applications. With optimally arranged CPU, Mini-PCIe, and M.2 slots, and compatibility with Galaxy Kylin/UOS operating systems, it provides high performance, high stability, and high reliability for the overall solution.

1.  ****Product Features****

1.  Features Phytium Tengrui D2000 8-core CPU, main frequency 2.0~2.3GHz
2.  Motherboard integrates multiple display interfaces, supporting VGA, HDMI, LVDS
3.  Supports 2 USB 3.0, 4 USB 2.0, and 8 RS232 serial ports
4.  Supports 1 PCIe x16 expansion slot, side-insertion type
5.  Supports wireless communication modules such as WiFi/Bluetooth/4G via Mini PCIe interface
6.  Supports M.2 SSD (M-Key), compatible with PCIe and SATA protocols; supports 2.5'' SSD interface, SATA protocol
7.  Supports dual Gigabit Ethernet interfaces, with auto-adaptation
8.  DC 12V power input, supports overvoltage protection and reverse polarity protection

1.  ****Application Areas****

Applied in industrial all-in-one PCs, industrial tablet PCs, and ruggedized tablets.

1.  ****Specifications****

****Form Factor****

Mini-ITX 150*150mm

****CPU****

Phytium D2000 8-core processor, main frequency 2.0~2.3GHz, TDP=15W

****BIOS****

Domestically developed UEFI, 16MB Flash

****Memory****

Supports two SO-DIMM slots, supporting 8GB, 16GB, up to a maximum of 32GB memory

****Storage****

Supports 1 M.2 M-Key NVMe interface (PCIe and SATA protocols) and 1 SATA 3.0 7-pin interface

****Display****

Supports 1 VGA (optional pin header), 1 HDMI interface, 1 LVDS interface (pin header, with backlight control)

****PCIe****

1.  1 Mini PCIe interface, supports 4G/5G (USB/UART resources)
2.  1 Mini PCIe interface, supports WiFi/Bluetooth (PCIe and USB resources)
3.  1 M.2 M-Key interface, supports 2280 SSD (PCIe and SATA protocols optional)
4.  1 side-insertion PCIe x8 interface, utilizing an x16 slot

****USB****

1.  2 USB 3.0 interfaces
2.  4 USB 2.0 interfaces, in pin header form

****Network****

Supports 2 Gigabit Ethernet interfaces

****Audio****

Supports 1 audio input and 1 audio output, 3.5mm interface

****Serial Ports****

1.  1 debug serial port, TTL level, on-board pin header
2.  8 RS232 serial ports, on-board pin header
3.  1 RS422/RS-485 serial port, on-board pin header

****Watchdog****

Supports hardware watchdog

****Fan****

2 4-pin fans

****Operating System****

Galaxy Kylin OS V10, UnionTech UOS

****Power Input****

Rated voltage 12V, supports DC Jack or ATX terminal power supply (choose one)

****Typical Power Consumption****

Approximately 30W

****Operating Temperature****

\-10℃ ～ 50℃

****Storage Temperature****

\-25℃ ～ 70℃
