Currently, Codesys + EtherCAT drives offer significant advantages for motion control. Nowadays, bus-based motion control mostly uses this configuration.

Codesys is known as the Android of the PLC world, and domestic PLC manufacturers mostly use the Codesys kernel.

Such as: Inovance, Co-Trust, Hollysys, Invt, Delta.

Including international ones: Beckhoff TwinCAT 2, Schneider SoMachine, Rexroth, etc., all use Codesys.

Intuitively speaking, Codesys is a software PLC with pre-written motion control and communication functionalities.

In motion control, the PLCopen protocol is a programming standard for PLCs. TIA Portal's motion control also uses PLCopen Motion, and Codesys supports the PLCopen standard.

Additionally, Codesys also supports high-level languages; for example, C++ functions can be called within Codesys.

For another example, Codesys can also define multi-tasks, similar to parallel computing in high-level languages.
