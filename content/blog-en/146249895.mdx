NXP+FPGA-Based Rail Transit 6U Chassis Structure Remote Input/Output Module (RIOM)![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/146249895/01_b81499e9681a428aa1674282778c2a4e.png)

RIOM allows data to be input and output through nearby I/O sources. I/O sources can also be directly connected to the train computer (e.g., VCU), which can reduce cable usage and thus save costs.

Key Features
Supports analog and digital input/output. Configurable modules include DI, DIO, MDO, RDO, AIO, PTI, etc.

Interface Options
MVB RIOM devices support three interfaces: MVB/CAN/Serial Link; TRDP RIOM devices support two interfaces: TRDP/CAN.

Number of Modules
Supports up to 14 I/O modules.

Technical Parameters
Dimensions (W×H×D):
427mm×132mm×230mm
Weight:
8kg
Input Voltage:
110VDC
Operating Temperature Range:
–25℃～+70℃
Networking Method:
TRDP/MVB/CAN/ECN
