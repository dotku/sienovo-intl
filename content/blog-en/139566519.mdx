##### **XCZU7EG FPGA — SOC (ARM+FPGA) Architecture**

The Zynq®UltraScale+™ MPSoC family is based on the Xilinx UltraScale™ architecture MPSoC. This product family integrates a feature-rich 64-bit quad-core/dual-core Arm®Cortex™-A53 and dual-core Arm®Cortex™-R5 based processing system (PS), along with Zynq®UltraScale™ programmable logic (PL). It also includes on-chip memory, multi-port external memory interfaces, and a rich set of peripheral connectivity interfaces.
