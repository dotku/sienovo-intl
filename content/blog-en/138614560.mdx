Capable of CVBS/HDMI/SDI signal conversion.
Customer Applications: TV stations, stages, conference rooms.
Key Features:
1: Standard CVBS signal input, standard HDMI signal input, standard SDI signal input; adaptive input signals.
2: Optional 3G/HD/SD SDI signal output.
2: Optional common resolutions: 1080P/1080I/720P/480I/576I.
Special resolutions like 1080PsF/720P30/25/24.
3: Supports embedding external analog stereo audio signals into the SDI signal.
4: SDI output can be set to "variable frequency" or "fixed frequency".
5: Special functions are customizable.
