## I. Introduction

With the development of computer and integrated circuit technology, real-time image processing application scenarios have become increasingly diverse. Concurrently, as the demand for image resolution in various scenarios grows, the time consumed by image acquisition and processing systems also increases.

Solution: Change the hardware implementation method of algorithms, and select a resource-rich, high-performance platform for acquiring and processing large-scale video image data.

This paper designs an image acquisition and processing system based on ZYNQ UltraScale+ MPSOC, with the main contents as follows:

**(1)** Addressing the issues of poor real-time performance and high latency in image processing with common multimedia processors, the image acquisition and processing system designed in this paper is based on the MPSOC platform. It achieves high-speed acquisition of high-speed (1920×1080@60fps) image signals output by the domestic CMOS image sensor GMAX4002 via a MIPI D-PHY interface, employs FPGA for real-time ISP signal processing, and finally displays the results in real-time on a DP monitor.

**(2)** To enhance image quality in military vehicle environments, the designed Image Signal Processing (ISP) algorithm pipeline module is applied to the system data path.
