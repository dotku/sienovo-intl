**Key Product Features:**

- Supports 4-channel real-time HD AHD 1080P recording
- Records data to SD card (supports 2 high-capacity SD cards, up to 256GB per card)
- Supports GPS global positioning, optional module
- Supports high-speed automatic WiFi download function, optional module

- Built-in acceleration sensor, etc.
