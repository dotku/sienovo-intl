**QQ Communication Protocol**

**Protocol Description:**
The protocol consists of a message header (T) + sender (T) + receiver (T) + message type (T) + message length (L) + message content.
Sender and receiver are program types within the system: OICQ server 0x01, fax server 0x02, WEB server 0x03, print server 0x04, chat server 0x05, OICQ user 0x0A.
OICQ User to OICQ Server Communication Protocol Identifier (0x81+0x0A+0x01)
