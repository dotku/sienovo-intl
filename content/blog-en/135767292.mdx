1.60 GHz Quad-Core CPU, 4 GB DRAM, 4 GB Storage, -40 °C to 70 °C, Kintex-7 70T FPGA, 8-Slot CompactRIO Controller
The cRIO-9047 is a rugged, high-performance, customizable embedded controller, offering Intel Atom quad-core processing power, supporting NI-DAQmx, and featuring an SD card slot for data logging, embedded monitoring, and control. It includes a Kintex-7 70T FPGA with LabVIEW FPGA Module support for advanced control and coprocessing. This controller uses Time-Sensitive Networking (TSN) to provide precise synchronized timing and deterministic communication, making it ideal for highly distributed measurements. Additionally, it offers multiple connectivity ports, including Gigabit Ethernet, USB 3.1, USB 2.0, RS232, and RS485 ports. The USB 3.1 port can be used to add a local human-machine interface (HMI), as well as for programming, deploying, and debugging software, simplifying application development.
