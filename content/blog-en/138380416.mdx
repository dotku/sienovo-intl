1. This is a high-performance 6U VPX data processing blade based on Phytium's FT2000+ processor. The VPX computing blade complies with the OpenVPX 6U standard specification, and its signal definitions conform to the SLT6-PAY-4F1Q2U2T-10.2.1 requirements defined in the OpenVPX VITA65 standard.

2. System interconnect interfaces are achieved through the VPX backplane, including: 10GbE Ethernet interfaces for data plane interconnection; Gigabit SerDes and MDI interfaces for control plane interconnection; single-ended GPIOs for system interrupt and time synchronization; PCIe interfaces for expansion plane interconnection; JTAG interface for system debugging and maintenance; IPMB interface for chassis management; blade slot identification signals for identifying different slots within the chassis; and chassis/cabinet identification signals for identifying different chassis and cabinets.

3. The product features a 64-core high-performance processor based on the ARMv8 architecture, delivering a theoretical peak floating-point performance of 563.2 GFLOPS@2.2GHz. It is suitable for high-speed, compute-intensive applications such as radar signal processing, data processing, and HPC.

4. Key Functions and Performance

● High-performance computing: Multi-core high-performance processor based on general-purpose ARM architecture, with a theoretical peak floating-point performance of 563.2 GFLOPS@2.2GHz; onboard 32GB DDR4 memory;

● High-speed serial real-time buses: 10GbE Ethernet, SRIO, and PCIe 3.0 x16;

● Front I/O interfaces: LEDs, reset button, VGA, USB, J30J connector, and a Gigabit debug Ethernet port;

● The J30J connector includes essential debugging interfaces such as JTAG and serial ports;

● Cooling method: Supports air cooling;

● Industrial operating temperature range: -40°C to +55°C.

5. Operating Environment

Operating Temperature: -40°C ~ +55°C  
Storage Temperature: -50°C ~ +70°C  
Relative Humidity: 95% non-condensing  
Vibration: 5 Hz–2000 Hz, 1.5 Grms (operating); 2 Grms, 20–2000 Hz (non-operating)  
Shock: 20G, 11ms (operating); 30G, 11ms (non-operating)

6. Electrical Specifications

Supply Voltage: 12V primary power, 3.3V auxiliary power  
Power Consumption: ≤180W  
Cooling Method: Air cooling

7. Mechanical Specifications

Form Factor: Standard 6U size  
Board Dimensions: 233.35mm (W) × 165mm (H) × 49.4mm (D)

8. System Configuration

CPU: Phytium FT2000+ 64-core processor  
Memory: Onboard dual-channel DDR4 memory, up to 32GB capacity  
Storage: Onboard 64GB SSD; default installation of Linktop Technology 64GB MLC mSATA SSD (Ordering Info: SIM064-SX5AM)  
Display: 1 integrated VGA output  
Operating System: Pre-installed Galaxy Kirin Server OS (trial version)

9. Interfaces

Gigabit Ethernet: 4× Gigabit Ethernet ports + 1× Gigabit Ethernet debug port  
10GbE Ethernet: 2× 40GbE Ethernet interfaces  
RapidIO: 2× SRIO 5G x4 interfaces  
PCIe: 1× PCIe 3.0 x16 interface (Note: configurable as 2× PCIe 3.0 x8 or 4× PCIe 3.0 x4 interfaces)  
Serial Interfaces: 1× standard RS232, 1× standard RS422  
SATA: 2× SATA 3.0 interfaces  
USB: 4× USB ports  
VGA: 1× VGA port  
Slot Address Interface: Slot address Mark[0:7] single-ended input interface
