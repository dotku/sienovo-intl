1. ****Product Overview****

ITX-XMF201 is a high-performance edge computing gateway motherboard featuring the domestically produced quad-core Phytium E2000Q processor, achieving a localization rate of 95%. The board integrates 2 Ethernet ports, 2 CAN interfaces, 6 RS232 serial ports, 1 RS485 serial port, and 16 GPIO channels, meeting IoT gateway requirements for industries such as banking, rail transit, and power systems.

1. ****Product Images****

1. ****Product Features****

1. Onboard domestic quad-core Phytium E2000Q processor;  
2. One SO-DIMM slot supporting 16GB, 8GB, or 4GB, up to 16GB maximum memory;  
3. Onboard 2 Ethernet ports, 2 CAN interfaces, 16 GPIO channels, 6 RS232 ports, and 1 RS485 port;  
4. Supports operating systems including Kylin V10, UOS, and LINUX.

1. ****Specifications****

****Form Factor****

MINI-ITX, 170mm × 170mm

****CPU****

Phytium E2000Q quad-core processor, 1.5~2.0GHz frequency, TDP = 11W

****Memory****

One SO-DIMM slot, supports 4GB, 8GB, 16GB, up to 16GB maximum memory

****BIOS****

Supports Phytium UEFI / U-BOOT

****PCIE****

- One MINI PCIe interface supporting 4G/5G (with USB/UART resources)  
- One MINI PCIe interface supporting WiFi/Bluetooth (with PCIe and USB resources)  
- One M.2 2280 interface supporting NVMe (auto-adaptive with SATA)  
- One reserved PCIe x2 signal using x4 connector

****Display Interfaces****

Supports one VGA (with pin header), one HDMI interface, and one LVDS interface (with backlight control)

****Network Interfaces****

Total of 2 Ethernet interfaces:  
Two Gigabit RJ45 ports; CPU RGMII-based (one RJ45 shares location with USB2.0)

****SATA****

One SATA 3.0 interface

****USB****

- Two USB 3.0 interfaces (one USB3.0 signal for lower connector, one USB2.0 signal for upper connector)  
- Two 2\*-1PIN onboard pin headers (providing 4 USB2.0 signals)  
- Two reserved USB 2.0 interfaces sharing location with front-panel RJ45

****CAN****

Two isolated CAN interfaces with Phoenix terminals

****Audio****

Dual-channel audio interface support

****Serial Interfaces****

- Five RS-232 onboard pin headers  
- One debug serial port  
- One RS-485 onboard pin header (configurable as RS232)

****Watchdog****

Hardware watchdog supported

****Fan Connectors****

- One 4-pin CPU fan connector with monitoring and PWM control  
- One 3-pin system fan connector with monitoring

****Operating Systems****

Kylin Operating System V10 / LINUX

****Power Supply****

Single 12V DC input via adapter; supports one 2×2 4-pin power terminal

****Power Consumption****

No more than 30W

****Localization Rate****

95%

****Operating Temperature****

Commercial grade: 0℃ ~ 45℃; Quasi-industrial grade: -20℃ ~ 70℃

****Storage Temperature****

-40℃ ~ 85℃
