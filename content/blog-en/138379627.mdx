**FeiTeng D2000+X100 TYPE6 Core Board**

**Product Overview**

The FeiTeng D2000+X100 TYPE6 core board is an enhanced domestic controller core board. Its core CPU chip features a FeiTeng D2000/8-core industrial version CPU, FeiTeng X100 chipset, dual-channel DDR4L slots, PHY chip, and more.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138379627/01_87f8071a08eb0fed0fbd4e5afc7c8305.jpeg)

**Product Features**

l Based on FeiTeng D2000+X100 chipset

l Abundant PCIe expansion resources: one PCIe x16, two PCIe x2, four PCIe x1

l Memory supports two DDR4 SODIMM slots, up to 32GB

l Domestically produced\*\*\*

l Compatible with domestic operating systems such as Galaxy Kylin V10, UnionTech UOS20

**Product Specifications**

**Form Factor**

95mm\*125mm, standard COM-Type 6

**CPU**

FeiTeng D2000/8-core standard version CPU

**Chipset**

FeiTeng X100 chipset

**Memory**

2 DDR4 SODIMM slots, up to 32GB

**Display Interface**

3x DP interfaces, including eDP

**PCIe**

1x PCIe x16 or 2x PCIe x8, 2x PCIe x2, 4x PCIe x1

**Storage Interface**

3 SATA 3.0 interfaces

**Network Interface**

1x PHY YT8521 Gigabit Ethernet port

**USB**

4 USB 3.1 in COM C-D; 4 USB 2.0 in COM A-B

**Serial Interface**

6 UART serial ports

**CAN**

Supports 2 CAN bus channels

**SMBus**

Supports 1 SMBus monitoring

**GPIO**

8 GPIOs

**Operating System**

Galaxy Kylin V10, UnionTech UOS20

**Power Supply**

ATX

**Commercial Grade**

Operating temperature: 0℃~50℃; Storage temperature: -20℃~70℃

**Semi-Industrial Grade**

Operating temperature: -20℃~70℃; Storage temperature: -40℃~85℃

**Industrial Grade**

Operating temperature: -40℃~70℃; Storage temperature: -40℃~85℃
