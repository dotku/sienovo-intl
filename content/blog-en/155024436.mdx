The smart storage acceleration card is based on XILINX VERSALâ ACAP MPSOC. The EP side provides a 1-lane PCIe Gen4 x16 interface, while the RP side supports up to 2-lane PCIe Gen4 x8 or 4-lane PCIe Gen4 x4. It features onboard DDR4-3200MHz cache, offering advantages such as low latency, high performance, low power consumption, low cost, and customizability. It is the preferred acceleration platform for server computing acceleration and storage services, including instance storage, SSD hardware virtualization, storage encryption/decryption, erasure coding, compression/decompression, high-speed data processing, and application offloading.![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/155024436/01_115e39f78c054c83a257d3033727126d.png)

Virtualization Application Feature Specifications:

Ø  256 virtual NVMe PCIe Functions (4PF + 252 VF)
Ø  Hardware implements 512 IO queues, each virtual NVMe supports up to 64 IO queues
Ø  Direct connection mode, supports 4 * PCIe NVMe SSDs
Ø  Supports NVMe virtualization based on 16GB granularity mapping
Ø  High performance, latency increase is around a few microseconds (us)
Ø  Supports transparent encryption and decryption, supports virtual NVMe device-level key management
Ø  Hardware implements QoS, precise down to the queue level
Ø  Supports disk management, isolates NVMe SSD failures
Ø  Uses native NVMe drivers, provides PCIe path management channels![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/155024436/02_299b50f88e2e4612aab868e91a672245.png)
