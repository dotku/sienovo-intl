The degree of digitalization and intelligence in industrial applications is an important criterion for measuring new quality productive forces. STM32's latest generation 64-bit microprocessor, the STM32MP2, with its advanced computing power, rich interfaces, and high security, empowers high-performance and highly interconnected Industrial 4.0 applications.

**Four Key Features of STM32MP2 Empowering Industrial 4.0 Applications**

The first product in the STM32MP2 series, the STM32MP25, is now in mass production. It features a junction temperature range of -40 to 125 degrees Celsius, supports 10 years of 100% uptime, and, like other STM32 products, offers a 10-year rolling supply guarantee.

The STM32MP2 features a heterogeneous architecture and excellent performance, robust security features, advanced edge AI capabilities and multimedia functions, and a rich set of peripheral interfaces and on-chip functions, which can offload the CPU in applications.

**STM32MP25 Features a Heterogeneous Architecture with Big and Little Cores, Delivering Excellent Performance:** Dual 1.5GHz Cortex-A35 cores and a 400MHz Cortex-M33 core, both the big and little cores can be started independently. The Cortex-A35 big cores typically run Linux or Android-based applications and, combined with a 3D GPU, video codecs, high-speed peripheral interfaces, etc., can meet users' high-end and complex business requirements. The Cortex-M33 core not only has the ability to directly access on-chip SRAM resources but also efficiently accesses on-board DDR memory. When running critical applications such as hard real-time tasks based on bare-metal or Real-Time Operating Systems (RTOS), it can benefit from high data throughput and ample memory resources, thus ensuring efficient system operation and real-time response. The heterogeneous dual cores can achieve efficient inter-core communication based on on-chip shared memory and allow for flexible allocation of on-chip resources.

**Enhanced Security Features for Industrial 4.0 Applications:** The STM32MP25 has achieved SESIP Level 3 and PSA Level 1 target certifications. It integrates powerful secure cryptographic hardware, enabling memory and peripheral protection to prevent unauthorized access and control. It features a secure cryptographic accelerator, enhancing robustness against physical attacks. It supports runtime code isolation protection and device authenticity verification throughout the product lifecycle.

**Advanced Edge AI and Rich Multimedia Capabilities Empower More Industrial Scenarios:** The STM32MP25 integrates an NPU edge AI accelerator with up to 1.35 TOPS of computing power. It supports MIPI CSI interfaces with ISP (Image Signal Processor), enabling various AI applications, including machine vision. The STM32MP25 also features an integrated 900MHz 3D GPU. Developers can flexibly choose to run AI applications on the CPU, GPU, or NPU.

The 3D GPU supports resolutions up to 1080p. It features a MIPI CSI-2 camera interface with an ISP. The STM32MP25 also integrates full HD video codecs and offers a rich set of display interfaces, supporting RGB, LVDS, and DSI outputs.

**Taking an Industrial HMI Gateway as an example:**

*   The STM32MP25's integrated H.264 Full HD video processor, 3D GPU graphics processor, ISP image signal processor, and display controller assist the Cortex-A35 CPU in achieving high-performance HMI graphical user interfaces.
*   The Cortex-M33 MCU handles industrial hard real-time applications.
*   The STM32MP25's integrated multi-port Gigabit Ethernet with TSN (Time-Sensitive Networking) switch functionality enables Ethernet ring topology.
*   The PCIe high-speed interface facilitates users connecting peripherals such as FPGAs for high-speed communication.
*   Various fieldbus communications can be achieved via bus interfaces such as CANFD, UART/USART.
*   The Cortex-A35 and Cortex-M33 have independent power domains and can be started independently. This ensures that if a Linux or Android-based application on the A35 core needs to restart for any reason in an industrial scenario, it will not interrupt real-time applications on the M33.
*   The Cortex-A35 offers excellent power efficiency. Combined with the accompanying STPMIC25 power management IC, it simplifies design and optimizes power consumption. When designing products, users do not need to configure additional cooling devices for the MP25, saving material costs and PCB space, while also ensuring product reliability, stability, and a longer lifecycle.
