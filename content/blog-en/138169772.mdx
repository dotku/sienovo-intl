The RK3588 is Rockchip's first high-end 8nm flagship chip, launched at the end of 2021. The RK3588S is a cost-optimized variant tailored for consumer markets, based on the RK3588, with some peripheral interfaces reduced while retaining key components such as the CPU, GPU, and NPU. Its primary applications include high-end ARM tablets, ARM laptops, conference panels, ARM servers, intelligent robots, AI video encoding/decoding, IP cameras, NVR/XVR systems, industrial control systems, smart cockpits, and more.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138169772/01_cefab087085ba47466ea30f39b486725.png)

The technical specifications of the RK3588S are as follows:

| Parameter | Specification | Description |
|----------|---------------|-------------|
| CPU | 8-core, 4×A76 + 4×A55 | Up to 3.0GHz |
| GPU | Mali-G610 MP4, supporting OpenGL ES 1.1, 2.0, and 3.2, OpenCL up to 2.2, and Vulkan 1.2 | |
| VPU | Encode: Supports real-time 8Kp30 encoding for H.265/H.264<br>Decode: 8Kp60 H.265, VP9, AVS2;<br>8Kp30 H.264 AVC/MVC;<br>4Kp60 AV1;<br>1080p60 MPEG-2/-1, VC-1, VP8 | |
| NPU | 6TOPS, NPU3.0 | |
| Memory | LPDDR4/LPDDR4x/LPDDR5, up to 32GB | |
| Storage | eMMC 5.1 | |
| Networking | 1000Mbps | |

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138169772/02_6a5c54bc2fd107286c5de6cd77bad4f5.png)

Functional block diagram of RK3588S

From the above specifications, it is evident that the RK3588S is highly suitable for developing ARM array servers, especially those used in cloud phones and cloud gaming platforms.

1) **Powerful CPU**: An 8-core high-performance ARM core with a maximum clock speed of up to 3.0GHz.

2) **High-performance VPU**: Exceptional video encoding capabilities that enhance visual quality for cloud phone and cloud gaming clients, while effectively reducing bandwidth requirements.

3) **High-capacity, high-speed LPDDR5 memory**: Fast read/write speeds with support for up to 32GB, enabling more concurrent Android instances and reducing cloud infrastructure costs.

4) **1000Mbps Ethernet interface**.
