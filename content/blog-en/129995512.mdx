The 2-6 axis dispensing control system consists of a TV5600 handheld terminal and an offline motion control card (OMC8849/OMC1600) based on the OMCV13 system. The two components are connected via Modbus serial communication. The TV5600 handles the human-machine interface (HMI) tasks, while the offline motion control card performs motion control and I/O operations, forming a split-type, full 3D, high-precision dedicated motion control system.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/129995512/01_63310c030dee2eb86d020bea15b301c2.png)

**Hardware Features:**

*   Controllable Axes: 2-6 axes
*   Maximum Pulse Frequency: 2MHz
*   Pulse Output: 5V differential output, Output mode: Pulse+Direction or Pulse+Pulse
*   Handheld Terminal Screen Resolution: 480x272 pixels, Color
*   128MB storage for handheld terminal, 128MB for offline card
*   Digital Input: All opto-isolated, Input voltage: 12-24V
*   Digital Output: All opto-isolated, NPN open collector 5-24VDC, rated current 0.5A, single channel maximum current 1A.

**Software Features:**

*   2-6 axis interpolation, supporting spatial lines, spatial arcs, and spline curves
*   Utilizes look-ahead velocity algorithm, automatic corner speed for arcs, and automatic corner filleting
*   Supports standard Modbus ASCII, Modbus RTU, and Modbus TCP communication protocols
*   Supports rich processing commands such as rapid traverse, linear interpolation, spline curves, file calls, and open user-defined commands (customers can program the specific actions of the commands)
*   Supports importing PLT, TCF, and G-code files.
*   Up to 1000 processing files can be stored, with each file storing 10,000 processing points.
*   Supports Lua scripting functionality specifically designed to address non-standard processes.
