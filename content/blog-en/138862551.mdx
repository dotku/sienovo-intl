This is a standard PMC specification, high-performance ARINC664 network interface card. The product supports application requirements such as ARINC664 network simulation, testing, and data analysis; it supports 2 ARINC664 ports in RJ45 plug form, with each port having up to 64K cache. The two ports can be used as two independent ports or as redundant ports. The product integrates an IRIG-B system clock, providing multi-node clock synchronization, which facilitates its use in multi-node simulation systems.

For transmission, it supports configuring up to 128 transmit VLs and 1024 transmit ports; for reception, it supports configuring up to 512 receive VLs and 4096 receive ports. It not only supports Queuing, Sampling, and SAP ports, but also SNMP and 615A loading protocols, among others.

As a standard PMC specification card, it can be conveniently applied in various hosts with PMC slots, such as CPCI/VME/VPX embedded systems. The product supports high-speed forwarding ports, reducing transmission delay and ensuring real-time communication during simulation and testing. Bundled with FlightPack protocol analysis and simulation software, it enables simulation, testing, and data analysis.
