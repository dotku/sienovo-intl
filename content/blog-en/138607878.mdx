FPGA solution: HDMI IN receives the original HDMI signal and outputs it for display on an LCD screen.

Customer Application: Extended Display

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138607878/01_681ac209d293c06487317b249ef2dacb.png)

Key Features:

1. Supports display of any resolution below 2K
2. Supports OSD overlay with multiple layers
3. Supports MIPI/EDP/LVDS/RGB screens
4. Supports scaling to match screen resolution
5. Zero latency, input-to-output
6. Supports independent audio signal output
7. Customizable features, expandable applications
