## IRIG-B Time Code Card

● 1-channel IRIG-B Time Synchronization Interface (RS485/Fiber Optic)
● 1-channel IEEE1588 V2 Time Synchronization Interface (RJ45/Fiber Optic)
● PTP Mode Time Synchronization Accuracy: ±100nS
● ETE Mode Time Synchronization Accuracy: ±250nS
● IRIG-B Time Synchronization Accuracy: ±250nS
● Self-holding Time Accuracy up to ±50mS/24 hours

## PCI-E Bus Interface Card

● Expands 6-channel PCI Express X1 Interfaces
● Cable Connectors with Anti-loosening Structure
● Dedicated High-speed Communication Cables (SFF 8088 Specification)
● Each Communication Port Contains 2-channel PCI Express X1

## 100Mbps Data Acquisition Card

● Data Acquisition Card
● Zero Packet Loss Data Acquisition Technology
● 4-channel Data Acquisition Ports (RJ45/SC/ST)
● Acquired Data with Timestamp, Timestamp Resolution 8nS
● Acquired Data Timestamp Accuracy up to ±100nS
● PowerPC, Main Frequency 1.0GHz, 2GB RAM
● Maximum Data Acquisition Throughput 400Mbps (Packet Size 64Byte)

## Gigabit Data Acquisition Card

● Data Acquisition Card
● Zero Packet Loss Data Acquisition Technology
● 2-channel Data Acquisition Ports (RJ45/LC)
● Acquired Data with Timestamp, Timestamp Resolution 8nS
● Acquired Data Timestamp Accuracy up to ±100nS
● PowerPC, Main Frequency 1.0GHz, 2GB RAM
● Maximum Data Acquisition Throughput 400Mbps (Packet Size 64Byte)
