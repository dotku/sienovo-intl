一、概述

The XMSW9118 fully domestically produced switch board provides a maximum of 48 10/100/1000BASE-T Ethernet interfaces and 8 SFP+ 10 Gigabit optical ports. It can achieve full-port wire-speed forwarding of data packets, supporting unicast, multicast, and broadcast packet types. It provides network port indicators, power indicators, and system operation indicators. The 48 10/100/1000BASE-T Ethernet interfaces use VPX connectors, and the 8 SFP+ 10 Gigabit optical ports use standard 10 Gigabit optical interfaces.

二、原理框图

The block diagram of the switch board is shown below, mainly composed of the CPU system, CTC7132 switching system, clock and power system, and other parts. It provides 48 10/100/1000BASE-T Ethernet interfaces and 8 SFP+ 10 Gigabit optical ports.

The CPU system uses Loongson 2K1000I as the processor, achieving control and management of the entire switch module through interfaces such as PCIe, management Ethernet port, debug serial port, and I2C.

The switching subsystem is mainly composed of the CTC7132 switch chip and Gigabit 8-port PHY chips, primarily realizing Ethernet data switching and forwarding, making it the core functional unit of the switch board. The system uses the CTC7132's MDC/MDIO interface to manage six 8-port Gigabit PHYs, enabling their configuration, management, and status monitoring.

三、功能性能指标

1、基本指标：

a) Main Processor: Loongson 2K1000;

b) Switch Chip: Centec CTC7132;

c) Operating System: Linux OS;

d) Provides 48 10/100/1000Base-T Ethernet ports + 8 10 Gigabit optical interfaces; the 10 Gigabit optical ports can be configured for use as Gigabit optical ports;

e) Provides 1 RS232 management serial port, with a default baud rate of 115200bps (configurable), brought out simultaneously via RJ45 and XH-3A connectors;

f) Provides 1 10/100/1000Base-T management Ethernet port, brought out simultaneously via RJ45 and XH-8A connectors; (reserved resistors, either path can be selected, default uses the path brought out by the XH-8A connector)

g) Supports MDI-X auto-adaptation;

h) Achieves full-port wire-speed forwarding of data packets, supporting unicast, multicast, and broadcast packet types;

i) Features board-level self-test functionality and Intelligent Platform Management Control (IPMC), capable of monitoring motherboard power voltage, temperature, and other information to report health status. IPMC has a reserved communication interface with Loongson 2K1000 (Party A is responsible for IPMC hardware design and software debugging, Party B is responsible for IPMC circuit layout and routing);

j) Provides network port indicators, power indicators, system operation indicators, and temperature alarm indicators;

k) Supports program upgrades via FTP or TFTP;

l) Interfaces use VPX standard connectors.

3、性能指标：

a) Port Performance

All data network ports on the switch board support 100% full wire-speed forwarding, complying with RFC2544 specifications, with a 64-byte packet delay of no more than 10μs and 0% packet loss rate.

b) Switching Capacity

The switching capacity of the switch board is no less than 256Gbps (bidirectional).

c) RAM Capacity

The Random Access Memory (RAM) capacity of the switch board is ≥2GB.

d) FLASH Capacity

The program storage (Flash) capacity of the switch board is ≥512MB.

e) MAC Address Table

The MAC address table of the switch board should be ≥64k.

f) VLAN Quantity

The number of VLANs on the switch board should be ≥4K.

g) BFD

BFD switchover time: ≤50ms.

h) Link Aggregation Convergence

Link aggregation convergence time: ≤50ms.

i) Other Performance Requirements

1) IPv4 Routes ≥16K;

2) IPv4 Host Table ≥10K;

3) ECMP ≥240;

4) Layer 3 Multicast Table ≥1K;

5) Layer 2 Multicast Table ≥2K.
