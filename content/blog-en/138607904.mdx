An FPGA-based solution that receives image data from a camera sensor and outputs it to the backend via HDMI.

Customer Application: Wireless RC Model Camera

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138607904/01_10cb11ab381f03be8044106cf77874ce.png)

Key Features:

1. Supports any resolution format up to 2K.
2. Supports various sensor models.
3. Supports adaptive camera configuration and aligns output timing.
4. Customizable features for extended applications.
