Module Overview

The FMC-XM148 is a JESD204B interface FMC mezzanine module based on the VITA57.4 standard. This module provides 8-channel, 14-bit, 500MSPS/1GSPS/1.25GSPS ADC acquisition capabilities. The ADC device on this board uses ADI's AD9680 chip, with a full-power -3dB analog input bandwidth of up to 2GHz. The ADC's host interface to the FPGA is interconnected via 16 high-speed serial GTX transceivers. This board is primarily designed for applications such as communications and wireless infrastructure, radar, wideband communications, millimeter-wave communications, and automated test equipment.
