4-Channel PoE NVR (200m Power Supply Distance)

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/143588880/01_145ce6c4d6bf429fa95a0c2fd45638f4.png)

I. Service Parameters:

1.  **Video input:** Supports 4-channel 5MP / 4-channel 1080P HD network video input, PoE network cable power supply, with an effective power supply distance of 200m.
2.  **Step on the Internet:** Eliminates complex router settings. Through Xiongmai Cloud Service, various network traversals can be easily achieved, enabling remote monitoring in one step.
3.  **Compression algorithm ideal:** The compression algorithm is upgraded to H.265. Advanced spatio-temporal filtering technology reduces the bitrate by over 30% compared to previous versions.
4.  **Playback quality:** Supports full real-time encoding recording and full real-time multi-channel synchronous decoding playback.
5.  **Main and Sub Stream:** Features simultaneous encoding of main and sub-streams. The main stream is used for local storage, ensuring high image quality, while the sub-stream is used for real-time network transmission, easily resolving bandwidth bottleneck issues.
6.  **Mobile monitor:** Supports mobile monitoring on various phone models (iPhone, Windows Mobile, BlackBerry, Symbian, Android).
7.  **Display:** Simultaneous HD VGA and HDMI output, with VGA and HDMI supporting 1080P HD output. True HD resolution output leads the technological trend.
8.  **Operation interface:** Windows-like operating style with powerful right-click menu functionality makes it exceptionally easy to get started.
9.  **Network:** Powerful network services (supports DHCP, PPPOE, FTP, DNS, DDNS, NTP, UPNP, EMAIL, IP permissions, IP search, alarm center, etc.), complete suite (WEB, client, SDK), easily achieving interoperability.
10. **Browsers:** Supports multiple web browsers (IE, CHROME, FIREFOX, SAFARI).
11. **Remote Access:** Built-in domain name service function (ARSP) enables one-click remote monitoring, making surveillance easy and enjoyable. It also provides domain name services such as 3322 (Xiwang), dyndns, Oray (Oray), and myq-see.
12. **Differentiation:** Customizable boot screen, over 20 languages, over 10 GUI interfaces, 10 types of remote controls, and dozens of button panels for combination selection, easily achieving product differentiation.
13. **Others:** Comprehensive protection circuits and a unique triple watchdog function ensure the device does not crash. Features unique black box technology for process monitoring of the entire device, greatly reducing maintenance costs.
