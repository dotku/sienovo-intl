An IoT gateway developed based on a core board, it can be used as a protocol converter,

industrial gateway, serial server, etc.

The IoT gateway integrates functions such as data acquisition, data analysis, data storage, multi-protocol conversion, and IoT platform access. It supports cellular, Wi-Fi, and Ethernet communication, meeting data acquisition and forwarding application needs in various IoT scenarios.

The IoT gateway's peripheral interfaces include dual Ethernet ports, input/output IOs, RS485, RS232, CAN bus, USB, SD card, 4G/5G communication, audio input/output, and Wi-Fi interfaces. The entire board adopts an industrial-grade design, capable of operating in various harsh environments.
