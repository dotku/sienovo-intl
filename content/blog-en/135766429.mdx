667 MHz dual-core CPU, 512 MB DRAM, 1 GB storage, Zynq-7020 FPGA, wider operating temperature range, 8-slot CompactRIO controller
The cRIO-9068 is a rugged, fanless embedded controller for advanced control and monitoring applications. This software-designed controller features an FPGA, a real-time processor running NI Linux Real-Time OS, and embedded user interface capabilities. The cRIO-9068 has an SDHC slot and various connectivity ports, including two tri-speed RJ-45 Gigabit Ethernet ports, one USB host port, and three RS-232 serial ports. Additionally, the cRIO-9068 supports an extended operating temperature range.
