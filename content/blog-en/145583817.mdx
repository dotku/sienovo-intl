**Application of 360-Degree Surround View in Unmanned Substations**

This system uses software algorithms to synthesize images from four outdoor fisheye lenses into a seamless panoramic image, which is then overlaid with a virtual 3D substation model. This composite image is transmitted in real-time via video signal to the central control display. The greatest innovation of this technology is that users can create custom viewpoints, i.e., custom virtual cameras. The system can freely and dynamically switch perspectives, intelligently emphasizing images of different areas around the substation during both static and dynamic processes. Simultaneously, this system also functions as a remote monitoring system, providing users with 24/7 monitoring records, comprehensive and blind-spot-free monitoring assistance from all angles, and other features. It is a high-tech smart hardware product integrated with AI.

Image Stitching Calibration

Enter "Main Menu" - "Stitching Calibration" - some versions require a password, the default password is "0000". Enter "Image Calibration" - Calibration method selection: Manual calibration / Automatic calibration. Manual calibration: Low site requirements, high success rate, requires manual point plotting. Automatic calibration: Requires a designated stitching cloth, demands a spacious site, automatically scans calibration points, short calibration time, generally used for large-scale production line operations.

Calibration Size Settings:

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/145583817/03_ad749e4e0480474dbd1e270769742449.png)

Select calibration size based on site size and vehicle size. Generally, there are three methods for ground calibration points:

1. Custom calibration cloth (fixed size, requires wind protection, typically for outdoor sites).
2. Calibration tape (multiple adjustable sizes, adheres to smoother surfaces, single-use).
3. Other calibration objects (tape measure, leather tape, custom wooden boards, ground paint, etc.).

Note: Calibration sizes can be proportionally scaled down or up. For example, when calibrating a toy house model, if the "Block Side Length" menu selects 1M, the actual size can be reduced to 1/5, which is 20cm. All dimensions use the same scale, and the final effect will be automatically scaled.
