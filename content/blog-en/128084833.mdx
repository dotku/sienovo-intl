> Built-in ultra-low latency 4K30 ISP IP, with ISP latency of 0.7 ms

> Built-in GigE Vision IP supporting GigE Vision 2.0 and GenICam V2.4.0 standards, with support for user-defined XML description files

> Built-in industry-standard U3 Vision IP for industrial machine vision

> FPGA-based, supporting Bayer, YCbCr, RGB, and other formats, meeting high frame rate/high resolution image acquisition demands

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/128084833/01_cd8b92dc35d0a0ad0b12b844beeb82aa.png)

*   Based on Xilinx Zynq UltraScale+ MPSoC ARM & FPGA heterogeneous processor
*   Utilizes SONY IMX334 4K Sensor and supports customization for various other sensors
*   Sub-millisecond ultra-low latency & 4K-level high-speed video processing
*   Supports diverse output interfaces: GigE/HDMI/USB3.0
*   Supports GenICam standard for machine vision applications
*   Provides complete hardware and software integration solutions and customizable image processing IP cores
*   Targeted at industrial, IoT, medical, machine vision, and other applications
