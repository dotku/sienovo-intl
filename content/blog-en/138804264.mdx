The digital class signage solution, featuring the localized OpenHarmony OS smart terminal mainboard AIoT-3568A and a facial recognition algorithm, supports multiple functions such as campus information publishing, facial recognition attendance, examination room management, and interactive inquiries. This helps schools achieve informatization, networking, and digitalization at the hardware level, building a smart education environment across the three dimensions of "school, teachers, and students."

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138804264/01_77af3187df2da49f06275495e3c03e69.jpeg)

**Solution Advantages**

**01/High-Performance Core Mainboard**

This solution utilizes the high-performance core mainboard AIoT-3568A, based on the RK3568 quad-core processor. It boasts powerful core processing capabilities and excellent power consumption control technology, offering 1 TOPS of computing power. It integrates various display interfaces, Ethernet, Wi-Fi, BT, and 4G, and features rich peripheral expansion interfaces. The terminal device supports stable operation in environments ranging from -10℃ to 60℃, making it suitable for semi-outdoor use.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138804264/02_bd406bfc1780e2983852201c702aa7fd.jpeg)

**Core Configuration:**

CPU: RK3568 Quad-core processor, main frequency up to 2.0GHz

NPU: Supports 1 TOPS computing power

Operating System: Android 11.0 (supports Linux/OpenHarmony)

RAM: Standard 2GB (4GB/8GB optional)

Storage: Standard 16GB (32GB/64GB/128GB optional)

**Display Interfaces:**

HDMI Output: 1 channel, standard Type-A female connector, supports up to 4Kx2K@60Hz resolution

LVDS Output: 1 channel, supports single/dual 6-8bit, can directly drive 50/60Hz LCD screens, maximum LVDS screen resolution supported is 1920*1080

eDP Output: 1 channel, can directly drive eDP interface LCD screens of various resolutions

**Peripheral Interfaces:**

USB: 1x USB3.0, 5x USB2.0

Serial Ports: 2x RS232, 2x TTL, 1x RS485

I2C Interface: 1 channel, can connect I2C interface TP or peripherals

CAN: 1 channel

Storage Card: TF card

**Audio Configuration**

Audio Input/Output: Speaker output (supports left and right channel output, default 6W, configurable to 10W, requires resistor modification), MIC IN*1

Headphone Output: Supports one 3-segment/4-segment headphone jack

**Network Communication:**

Wireless Network: Built-in WiFi; Bluetooth 4.2

Wired Network: Supports 10/100/1000M adaptive Ethernet

Mobile Network:

1. Supports 4G internet via PCI-E interface

2. Supports 5G internet via M.2 5G interface (optional)

**02/Localized Operating System**

All information and data within the digital class signage are processed in the cloud. System-level security capabilities, derived from OpenHarmony, ensure information security.

**03/Facial Recognition Algorithm**

The facial recognition algorithm included with the solution provides device-side localized functions such as face detection and tracking, face search and comparison, liveness detection, and person-ID verification.

①Face Detection and Tracking: Detects face positions in static images or video frames and continuously tracks detected faces. Supports various video capture devices, including visible light and near-infrared.

②Face Search and Comparison: Supports building a local face database and provides localized 1:1 comparison and 1:N search functions.

③Liveness Detection: Determines if the target object is live based on video streams, effectively preventing liveness attacks using videos, photos, etc. Supports video capture devices such as monocular (RGB) and binocular (RGB+near-infrared).

④Person-ID Verification (supported by ID verification version): Supports comparing images captured by the camera with ID photos to accurately determine if it is the same person.

**04/Supports Deep Customization**

Hardware and system can be deeply customized according to end-customer requirements, better aligning with actual functional needs.

**Application Cases**

This solution is primarily applied in schools, training institutions, and similar environments. Currently, it has been successfully implemented in schools across various locations, including Hefei, Bengbu, and Nanning.
