****Software Interface****

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/139566672/03_fbba70abd0468387b87af2376725e085.png)

Automatically discovers instruments, enabling simple control and management. Rich system APIs, including connection management APIs and test result judgment APIs, improve development efficiency. Graphical drag-and-drop operations and auto-completion reduce development complexity. Test cases are saved using CVS and Python, facilitating editing and version management.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/139566672/04_a34a77dfe19aa678aeaeb22933c863d0.png)

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/139566672/05_68db3f0b89609a90f60522bf727dae08.png)

Supports distributed deployment, and offers both sequential and parallel execution modes, flexibly adapting to various production test scenarios. Test data results can be stored in a database for visual analysis or exported and saved locally using custom templates.
