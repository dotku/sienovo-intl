**Solution Overview**

5G tele-operated driving is a crucial application of connected autonomous driving, providing redundant safety measures for vehicle operation, overcoming challenges such as difficult operations in hazardous environments, and effectively improving operational efficiency.

Sienovo's 5G Remote Driving Cockpit Latency Verification Solution efficiently collects vehicle-side sensor data and utilizes 5G communication technology to push data in real-time to the remote cockpit. This solution precisely verifies the system latency from sensor data streams to cockpit display, meeting the latency verification requirements of 5G remote driving cockpits in practical applications, ensuring immediate response for remote driving operations, and enhancing operational efficiency and safety.

**Solution Advantages**

■ Full-Link Time Synchronization: The acquisition system, tele-operated vehicle, and cockpit all receive time synchronization via PTP service, with the Ailiguang CS-300 Synchronization Time Server providing the time synchronization function.

■ Precise Timestamps: The Ailiguang DS-300 Data Acquisition Rig installed on the tele-operated vehicle provides camera data acquisition and millisecond-level precise timestamps.

■ Data Link Latency: To optimize the latency of 5G network remote control, Ailiguang provides hardware video encoding based on edge computing, optimizing the data copying and transmission link.
