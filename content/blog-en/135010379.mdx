The XINMAX XM-3568-01 motherboard features Rockchip's RK3568 quad-core Cortex-A55 processor, with a maximum frequency of up to 2.0GHz, delivering significantly enhanced performance. It supports up to 8GB of memory with frequencies reaching 1600MHz and offers full-chain ECC support for improved data security and reliability. The board is equipped with dual Gigabit adaptive RJ45 Ethernet ports and expands four additional Ethernet controllers via I211 chips, making it ideal for applications such as Gigabit routing, soft routers, and firewalls.

Supports HarmonyOS, Android 11, and Linux.

Support for Domestic Operating Systems

*   ### Galaxy Kylin Embedded Operating System V10
    

Galaxy Kylin Embedded OS V10 is a secure, real-time embedded operating system designed for IoT and industrial internet scenarios. It features information security, multi-domain isolation, cloud-edge-end collaboration, and support for diverse computing architectures, meeting the embedded requirements for miniaturization, reliability, security, real-time performance, and connectivity. Based on Linux, Galaxy Kylin Embedded OS V10 adopts a "domain-based virtualization + multi-domain isolation" architecture. By enabling hybrid deployment of real-time and non-real-time operating systems, it combines the rich ecosystem of Linux with the hard real-time capabilities of RTOS, providing broad application support across IoT and industrial internet domains.

Galaxy Kylin Embedded OS V10 supports mainstream domestic and international CPU platforms from a unified codebase, and is widely applicable in industrial manufacturing, public transportation, energy and power, financial technology, automotive electronics, digital communications, healthcare, and smart city fields.

## Product Features

#### Product characteristics

*   **Multi-Domain Isolation**
    

An embedded hypervisor for domain separation implemented in the Rust programming language, supporting device management across domains and enabling hybrid deployment and communication between real-time and non-real-time operating systems.

*   **Intrinsic Security**
    

Built on trusted hardware and industrial security requirements, it establishes an integrated intrinsic security framework based on independently developed software and hardware.

*   **Strong Real-Time Performance**
    

Executes critical real-time tasks within the real-time domain; offers microsecond-level interrupt latency and task switching delay; ensures reliable inter-domain communication and coordinated multi-domain operations.

*   **AI Intelligence**
    

Supports lightweight AI frameworks, enabling edge intelligence computing within functional domains.

*   **Industrial Stability**
    

Industrial-grade system stability with support for long-term unattended operation.

*   **Ubiquitous Connectivity**
    

Integrates mainstream industrial communication and IoT protocols to enable collaborative cloud-edge-end connectivity.

*   **Customizable On Demand**
    

Flexible OS size customization from megabytes to gigabytes based on specific application scenarios.

*   **Rich Ecosystem**
    

Compatible with the Kylin ecosystem, inheriting over 1.5 million software and hardware integrations from Galaxy Kylin general-purpose products, supporting a wide range of industry applications.
