## I. Vision Controller

The entire system primarily consists of two parts: hardware and software.

Hardware components: Vision Controller + Camera + Lens + Light Source + Light Source Controller

Software components: ADTvision module

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/129994046/01_3a8af4f925c1d5b0a9cd6e8340aaf475.png)

**Features**

1. Supports cameras with various interfaces such as Gigabit Ethernet and USB.

2. Supports area scan CCD cameras.

3. Provides 1 opto-isolated input, 1 opto-isolated output, and 2 GPIOs.

4. Supports online or offline inspection.

5. Supports various common communication interfaces such as RS232 and Ethernet.

6. Complies with OPC standard.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/129994046/02_3d4f11266d4f1f458e4d0679fba1e725.png)

## II. AVS4.0 Smart Vision Software

The AVS Smart Vision Recognition System is a new generation, easy-to-learn, easy-to-use, feature-rich, and powerful general-purpose vision software platform.

The AVS4.0 Smart Vision System is an intelligent general-purpose machine vision software developed to adapt to rapid market development and various application requirements. It covers image processing functions such as positioning, measurement, identification, OCR recognition, code reading, "fly-by" capture, and logical operations.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/129994046/03_b2d4ef65f52d4a9f05b6da67cf3309dd.png)

The AVS4.0 Smart Vision System features hundreds of practical tools, covering ten major categories of toolboxes, including image processing, pre-processing, positioning, blob analysis, measurement, identification, inspection, communication, logic control, and system tools.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/129994046/04_4b1b92469a7ed994c581a3123ae310f3.png)

The AVS4.0 Smart Vision System is an easy-to-learn, easy-to-use, feature-rich, and powerful general-purpose vision software platform. It encapsulates complex vision algorithms into modular list tools and uses a graphical drag-and-drop interface, allowing users to easily build complex vision application projects with just a few simple steps, without needing to write any code.

**Target Users:** Professional and non-professional technical personnel across various industries, with no prior machine vision expertise required.

**Applicable Industries:** 3C electronics, packaging industry, medical pharmaceuticals, automotive manufacturing, daily consumer goods, and more.

Xinmai provides integrated custom solutions for vision software and hardware.
