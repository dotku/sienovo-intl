1. eMMC Formatting

Reference: eMMC mounting, partitioning, and formatting [https://blog.csdn.net/u011975319/article/details/51803126](https://blog.csdn.net/u011975319/article/details/51803126)

1. Use a multimeter to check for power short circuits on the board.

2. Check if the input terminal power voltage and signal sequence are correct.

3. After powering on, use a multimeter or oscilloscope to test if the voltage at various power test points on the board is correct. ------ Q1 is not conducting, so D1 (LED) is not lit. R9 resistance needs to be adjusted.

4. Remove R330, R333, R328, R334. In the Windows system's sb_loader directory (cmd terminal), enter the command `sb_loader -f u-boot(2G).imx`. Use the sb_loader tool to flash the `u-boot(2G).imx` file into the board.

5. Internal eFuse ------ For detailed operation methods, refer to the chapter 'IMX6 Internal eFuse Blowing'.

6. SPI Flash Flashing File

```
> fatload mmc 0 10800000 u-boot.imx
or ext4load mmc 2 10800000 u-boot.imx
> sf probe
> sf erase 0 0xc0000
> sf write 10800000 0x400 $filesize
> reset
```
7. Insert the SDCARD boot card and power on again.
