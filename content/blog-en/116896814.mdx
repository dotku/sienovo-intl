**1 Applicability Description**

This example primarily demonstrates connecting to the DSP core using an emulator via the JTAG interface. The example is based on the Ximai XM138-IDK-V3 AM5728 development board.

## **2 Emulator Connection to DSP in Debug Mode**

1.  ****Experiment Steps********:****

*   Install CCS6.1.3 according to the instructions in the software installation manual. Pay attention to updating the relevant emulator drivers as per the document.
*   Select Debug boot mode for the development board, set the DIP switches to 11111, and connect the emulator to the JTAG interface.

If using an XDS200 or XDS560V2 emulator, ensure to update the emulator firmware and drivers.

*   Open CCS, configure the file, connect the ARM core, then connect the DSP core.

In the CCS interface, click View, then click Target Configurations.

## **3 Emulator Connection to DSP under Linux System**

1.  ****Experiment Steps****

*   Install CCS6.1.3 according to the instructions in the software installation manual. Pay attention to updating the relevant emulator drivers as per the document.
*   Select SD card boot mode for the development board, set the DIP switches to 01000, and connect the emulator to the JTAG interface.

If using an XDS200 or XDS560V2 emulator, ensure to update the emulator firmware and drivers.

*   Open CCS, configure the file.
*   Open the serial port window.
*   Connect to DSP1 in CCS.

In the CCS interface, click View, then click Target Configurations.
