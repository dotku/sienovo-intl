The ARINC429 IP CORE implements the ARINC429 interface protocol using a single FPGA. The product complies with the ARINC429/HB6096 bus protocol specifications, and each channel can be independently configured for reception or transmission. As a standard IP Core, it supports embedded system interfaces such as PMC, PCI, CPCI/VME/VPX, PCIe, as well as user-customized Local bus interfaces.
