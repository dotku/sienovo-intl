Key Features

◆ Based on NXP's LS1046A processor, an ARM Cortex-A72 64-bit quad-core, with an entirely fanless board design
◆ Supports 4 to 6 2.5G M12 Ethernet interfaces; features onboard memory, NAND Flash or Nor Flash, IRIG-B, etc.
◆ Supports operation in wide temperature (-40℃ ~ 80℃), high humidity, high heat, and other harsh environments

CPU

NXP LS1046A, Arm Cortex A72 4-core 1.8GHz

Memory

Onboard 4GB/8GB memory, optional

LAN

4 M12 Ethernet ports, 2 x 2.5G (POE), 4 x 1000M (optional)

I/O Interfaces

2 USB 3.0 ports, 1 RS232 debug port

Customizable run, power, and status indicators

Storage

Onboard NAND FLASH

1 SATA interface

Operating Environment

Operating Temperature: -40℃～ 80℃

Operating Humidity: 5~95%, non-condensing
