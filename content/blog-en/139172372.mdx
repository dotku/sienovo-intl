SAS Expander is a 12Gb/s Serial Attached SCSI (SAS) expansion chip supporting 48/36/24 ports, delivering high-performance, scalable, and cost-effective storage solutions.

The product comes with a feature-rich software development kit (SDK), including streamlined and easy-to-use standard API interfaces that meet diverse development requirements across various application scenarios such as storage subsystems and Storage Area Networks (SANs). This significantly shortens time-to-market, reduces deployment risks and total cost of ownership (TCO), and enhances product competitiveness.

• SAS/SATA dual storage interface  
• 12Gb/s SAS  
• 6Gb/s SATA  
• Supports EDFB bandwidth aggregation  
• Supports SAS wide ports  
• Supports partition management  
• Phy-based  
• T10-mode  
• Supports SXP internal virtual ports  
• Supports security authentication  
• Supports multiple boot modes  
• Supports port mirroring  
• Supports firmware upgrade  
• Supports SPIN UP  

Applications:  

• Scalable storage servers  
• Storage arrays  
• SES chassis management  
• Storage Area Networks (SANs)
