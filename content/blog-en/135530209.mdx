A 24-bit temperature measurement card with a USB 2.0 bus, supporting 1-channel RTD mode and 6-channel thermocouple mode, with no external conditioning required. It also supports multi-range voltage measurement, 16-channel bidirectional I/O, 2-channel counter/frequency measurement, and 3-channel PWM output.

Features Overview

The USB-XM2403 is a USB interface temperature data acquisition card. The USB-XM2403 features 7 analog inputs (configurable for temperature measurement or small signal measurement), 16 DIO channels, two counter/frequency measurement channels (DIO pins multiplexed), and 3 PWM outputs. The ADC chip supports programmable gain (×1, ×2, ×4, ×8, ×16, ×32, ×64, ×128), and each channel's amplification factor can be set individually via software. The logic control of the USB-XM2403 is implemented using a Field-Programmable Gate Array (FPGA) to enhance reliability. Additionally, digital ground (GND) and analog ground (AGND) are separated with single-point grounding to eliminate ground loop interference.

The USB master control chip of the USB-XM2403 card uses CYPRESS's CY68013A. This board is a high-speed, high-precision data acquisition board with a USB 2.0 bus, supporting hot-swapping and plug-and-play functionality.

Product Applications

The USB-XM2403 acquisition card can be directly connected to a computer's USB port to form data acquisition, temperature measurement, waveform analysis, and processing systems in various fields such as laboratories and product quality inspection centers. It can also be used to build industrial production process monitoring systems. Its main application scenarios include:

Small Signal Measurement

Pulse Counting

I/O Control
