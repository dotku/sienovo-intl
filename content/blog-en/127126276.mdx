**Solution Features**

① Utilizes high-performance Loongson 3A4000 processor, operating at 1.8GHz-2.0GHz, compatible with Loongson 3A5000 processor, operating at 2.3GHz-2.5GHz

② Equipped with Loongson's self-developed 7A1000 bridge chip, offering rich interfaces to meet I/O application requirements in desktop and server fields

③ Integrated FPGA-V7 coprocessor, supporting high-speed buses like SRIO and LVDS, enabling high-speed data transmission and real-time processing

④ Onboard solid-state chip storage, with fast read/write speeds

⑤ Achieves 100% indigenous production

⑥ Can be widely applied in national defense and military, cybersecurity, railway & rail transit, nuclear power & electricity, industrial measurement and control, medical, and other fields

⑦ Board standard: VPX-6U

⑧ Highly customizable platform

### **Performance Parameters**

CPU
Loongson LS3A4000, main frequency 1.8GHz–2.0GHz @4 cores, 28nm
Compatible with Loongson LS3A5000, main frequency 2.3GHz—2.5GHz @4 cores

Bridge Chip
Loongson 7A1000, providing Northbridge and Southbridge functions for Loongson processors

FPGA
Fudan Micro JFM7VX690T, compatible with Xilinx XC7VX690T

Memory
16GB DDR4, ECC

FPGA Memory
DDR3, 256MB

BootFlash
NorFlash, SPI interface, 16MB

DataFlash
Onboard solid-state chip, 128GB

Ethernet
6-channel 1000/100/10Mbps adaptive network ports

UART
3-channel UART serial ports

USB
6 x USB2.0, 3 channels routed to VPX connector, 3 channels routed to front panel

SATA
2-channel SATA interfaces

IIC Interface
3 channels, for IIC communication

PCIe
3-channel PCIe, including 1 x PCIe x4, 2 x PCIe x8, can be divided into up to 8 independent channels

Display
1 x LVDS interface; 1 x VGA interface, shared by front panel and VPX rear panel

SRIO
2-channel SRIO x8 interfaces

Other Interfaces
1 x SPI, multiple GPIOs

System Power
DC 12V, powered via VPX connector

Operating System
Supports domestic systems such as Loongnix, Kylin

Bootloader
UEFI or PMON

Operating Ambient Temperature
-20℃ – 55℃ / 0℃ – 55℃

Storage Ambient Temperature
-55℃ -- 85℃

Operating Ambient Humidity
95% (non-condensing)

Domestic Content Ratio:

RBV3400N
100% domestically produced and independently controllable components on the entire board

RBV3400W
Entire board contains imported components

RBV3500N
100% domestically produced and independently controllable components on the entire board

RBV3500W
Entire board contains imported components
