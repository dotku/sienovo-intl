Data Acquisition Card Technical Specifications  
Acquisition Mode: Synchronous acquisition, 24-bit Delta-Sigma ADC; hardware anti-aliasing filter  
Sampling Rate: 1 kS/s – 52 kS/s  
Accuracy: ±20 ppm (approximately 40 ppm, depending on final front-end selection)  
Input Channels: 4  
Software-selectable IEPE signal conditioning (0~5 mA adjustable); software-configurable gain settings: 1/2/4/8/16, dynamic range not less than 103 dB  

Software-configurable AC/DC coupling  
Lightweight and portable, dimensions no larger than 170 mm × 120 mm × 35 mm, weight no more than 500 g; convenient power supply options supporting both +5V DC input and USB power  
Low power consumption: no more than 3.5 W when externally powered (normal operation at 5 W)  
BNC connectors for easy connection to sensor cables  
Robust and durable, operating temperature range from -40°C to +70°C  
Data transmission to host computer via Ethernet  
Basic software interfaces typically provided, supporting customer secondary development at application layer  
Supports long-distance transmission  
Includes test panel with basic measurement functions  
Provides x86/x64 SDK for user-developed custom software  
Custom programs available, including source code examples for vibration spectrum and audio quality measurement developed in LabVIEW  
On-site technical consultation and technical support provided
