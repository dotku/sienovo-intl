Board Introduction:

This board is a Network Interface Board (NETI). It communicates internally with the main control board and various I/O boards, and externally supports RS485, CAN, or MVB (optional) network communication.

Specifications:

Power Supply: DC5V, DC3.3V

FPGA: XC6SLX25-2FG484I

MCU: MK60DN512VLL10 x 2

CPLD: EPM570T100I5N

Storage: PROM 16MB (FPGA)

Communication Interfaces (Panel Side): CAN x 2 (D-SUB9); RS485 x 2 (D-SUB9); Optional MVB x 2 (D-SUB9)

Communication Interfaces (Backplane Side): CAN x 1; I2C x 4

Dimensions: 220mm\*100mm\*1.6mm

Weight: 0.160kg

Operating Temperature: -25℃~70℃
