To cover various medical institution testing scenarios, fluorescence immunoassay analyzers can be categorized into handheld, single-channel, multi-channel, and fully automatic models. Among these, multiple core boards and mainboards based on the i.MX6 series processor can respectively meet users' performance and cost requirements. Liaowang Electronics has specifically achieved software platform compatibility, utilizing the same Linux kernel version, compilation tools, and QT development environment. This allows user applications to be freely switched between platforms, maintaining software and application consistency while also meeting the needs of both high-end and low-end hardware platforms.

The fluorescence immunoassay analyzer project consists of a series of boards and peripherals such as the main control board, sub-board, turntable, slave board, and thermal printer. The solution covers a range of components including core boards, M3 microcontrollers, etc., with the system optionally running Linux/Android 5. We provide services such as system customization and EMC rectification to minimize customer development cycles and difficulty, thereby accelerating time-to-market.
