The 2-Port 10G SFP+ NIC is a high-speed network interface card that utilizes a PCI Express 3.0 x8 interface, supporting dual-port 10 Gigabit Ethernet. It offers advantages such as high performance, high reliability, and low power consumption, making it an ideal choice for data centers, cloud computing, virtualization, and other fields.

It supports various network protocols, such as TCP/IP, UDP/IP, VLAN, etc., capable of meeting the demands of different application scenarios. In addition, it also supports technologies like SR-IOV and DCB, which can improve the efficiency and performance of network virtualization.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/139172287/01_5da56e05cb976a100ae82789930c8332.png)

Dual-Port 10 Gigabit Ethernet Controller
• PCIe 3.0 x8 Host Interface
(2.5GT/s, 5GT/s, 8GT/s)
• Compatible with 10Gb/s, 1Gb/s Ethernet
(SFP+/SFP)
• Supports TCP/UDP Transmit/Receive Segmentation/Aggregation Offload
• Supports Receive Side Scaling
• Supports NIC Virtualization
• Supports Data Center Bridging (DCB)
• Supports Flow Classification
• Supports Jumbo Frames up to 15.5KB
• Supports Sideband Management

Application Scenarios:
Servers/Workstations
• Virtualization
• Home PCs

Interface Overview
• Supports SFP-packaged fixed-rate 1Gbps optical modules
• Supports SFP+-packaged fixed-rate 10Gbps optical modules
• Supports SFP+-packaged multi-rate optical modules (up to 10Gbps)

Offload Capabilities
• Supports TCP/UDP Transmit Segmentation Offload (TSO)
• Supports TCP Receive Side Coalescing (RSC)
• Supports IPv4/6 TCP/UDP Transmit/Receive Checksum Offload

Security Features
• Integrated LinkSec security engines: AES-GCM 128-bit; Encryption + Authentication; One SC x 2 SA per port. Replay Protection with Zero Window
• Supports Firmware SM2/SM3 Secure Boot

Queue Management
• Supports both Non-Virtualization and Virtualization operating modes
• Supports Receive Side Scaling (RSS)
• Supports Data Center Bridging (DCB)
• Supports Flow Classification
• Supports SR-IOV Virtualization

Management Interface
• Supports PLDM over MCTP over SMBus Sideband Management
