This Software Defined Radio (SDR) processing platform, based on Xilinx ZYNQ SOC, utilizes a high-performance Xilinx ZYNQ series SOC (XC7Z020) to achieve 2-channel AD9361 wireless RF signal transmission and reception. It is widely applicable to applications such as SDR, mobile base stations, Wi-Fi, wireless local area networks, and specialized or general-purpose wireless devices. The ZYNQ7000 is an extensible processing platform launched by Xilinx. The ZYNQ SOC integrates a dual-core ARM Cortex-A9 processor, perfectly coupling the programmability of the FPGA with the video and multimedia interfaces of the ARM, making it suitable for low-power, low-cost signal processing products. Data interaction between the PS (Processing System) and PL (Programmable Logic) of this SOC is achieved via high-speed HP (High Performance) interfaces. The PS side can run operating systems like Linux and features an embedded Gigabit Ethernet protocol stack, making it easy to implement Ethernet TCP/IP protocol transmission. The PL side contains abundant programmable logic resources for front-end signal processing and other tasks.

#### Technical Specifications

*   Onboard SOC Real-time Processor: XC7Z020-2CLG484I;
*   Hardware Features:
    1.  Onboard 2x AD9361 RF Transceiver Chips, compatible with AD80305;
    2.  Features 1x Gigabit Ethernet Port;
    3.  Features 1x USB Serial Port, 1x USB OTG, 1x SD Card;
*   RF Specifications:
    1.  RF Frequency: 60MHz~6GHz;
    2.  ADC/DAC Sampling Bit Width: 12-bit;
    3.  2x TX, 2x RX, supporting full-duplex or half-duplex operation;
    4.  RF Matching Impedance: 50 Ohms;
    5.  Instantaneous Bandwidth: 56MHz;
    6.  Shielded Protection Design
*   Dynamic Storage Performance:
    1.  Memory Bandwidth: 32-bit, DDR3 SDRAM, 500MHz operating clock;
    2.  Memory Capacity: Supports up to 4GByte DDR3 SDRAM;
*   Other Interface Features:
    1.  Features JTAG On-line Debugging Interface;
    2.  Features 1x RS232 Serial Port, connected to the PS side;
    3.  Features 1x SPI Flash for ZYNQ loading;
*   Physical and Electrical Characteristics
    1.  Board Dimensions: 100 x 160mm
    2.  Board Power Supply: 1.6A max@+12V (±5%)
    3.  Cooling Method: Metal Cold Plate Cooling
*   Environmental Characteristics
    1.  Operating Temperature: -40°~﹢85°C;
    2.  Storage Temperature: -55°~﹢125°C;
    3.  Operating Humidity: 5%~95%, non-condensing
