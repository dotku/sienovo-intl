Unix/Linux/BSD systems all have three special files:

1) Standard Input (STDIN), located at /dev/stdin, typically refers to keyboard input, and is represented by file descriptor 0 in the shell.

2) Standard Output (STDOUT), located at /dev/stdout, usually refers to the terminal (terminal), i.e., the display, and is represented by file descriptor 1 in the shell.

3) Standard Error (STDERR), located at /dev/stderr, also refers to the terminal, but is specifically used for error messages. It is represented by file descriptor 2 in the shell.
