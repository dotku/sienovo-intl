### Detailed Introduction to the Phytium FT2000+ 64-Core ATX Motherboard

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138380336/01_d750589cc1b83cfae9dc9fb687d3cfde.jpeg)

**Item**

**Main Technical Specifications**

CPU

Onboard Phytium FT-2000+, 64-core

Memory

Supports 8\*DDR4 R-DIMM slots, up to 1TB memory capacity

RAID Functionality

Onboard SATA 3.0 controller, supports RAID 0, 1, and 10

VGA Display

BMC-integrated display controller

Network Controller

Onboard 4 Gigabit Ethernet ports

External and Onboard Interfaces

4 rear USB 3.0 ports

4 rear Gigabit Ethernet ports

1 rear BMC management network port

1 rear COM port

1 rear VGA port

2 onboard PCIe 3.0 x16 slots

4 onboard PCIe 3.0 x8 slots

1 onboard Mini-PCIe slot

2 onboard Mini SAS 3.0 connectors

1 onboard M.2 2280 slot

4 onboard SATA 3.0 interfaces

1 onboard USB 3.0 pin header

BMC Management Features

Onboard BMC management module, 1 BMC management network port, supports KVM over IP, remote power on/off, system status monitoring, and more

Firmware

Domestically developed, secure, and controllable BIOS

Operating System

Supports domestic server operating systems such as Kylin OS and UOS

Physical Dimensions

330mm x 305mm

Temperature

Operating: 0°C to 40°C; Non-operating: -20°C to 70°C

Humidity

Operating: 10% to 90%; Non-operating: 5% to 95% (non-condensing)
