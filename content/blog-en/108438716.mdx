### Selected Application Cases

![Industrial Automation](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/108438716/04_9ad6e2e9fdfa89c00e2f8f53d489f8d1.png)

Project Name: Industrial Robot

Project Description:  
Industrial robots are multi-jointed mechanical arms or multi-degree-of-freedom machines designed for industrial applications. They can automatically perform tasks by relying on their own power and control systems to achieve various functions. These robots can operate under human guidance or follow pre-programmed instructions. Modern industrial robots can also act based on principles defined by artificial intelligence technologies. This project is built on the AM5728 platform using the Linux operating system, requiring audio interfaces, Ethernet ports, serial ports, and an SD card. It must include a DSP, ensure system stability, support multiple serial ports, and have long-term component availability.

![Industrial Automation](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/108438716/05_002cf3b992b81dece1cb975b4e587c26.png)

Project Name: Robotic Arm

Project Description:  
A robotic arm is an automated control device that mimics the functions of a human arm and performs various tasks. This robotic system features multiple joints allowing movement in planar or three-dimensional space, or linear displacement. Structurally, it consists of a mechanical body, controller, servo mechanisms, and sensors, with specific actions defined by programs according to operational requirements. This project is based on the AM5728 platform running Linux, requiring serial ports, GPMC, SD card, GPIO, and Ethernet. It demands multiple serial interfaces, high stability, low cost, long-term supply availability, and an industrial-grade core board.

Project Name: Robotics Industry Control System

Product: XM5728-SOM-V2.3 Core Board Based on AM5728

Project Description:  
The XM5728-SOM-V2.3 is a Cortex-A15 solution based on the SMARC interface, offering low cost, high performance, Linux support, and a short development cycle.

![Industrial Automation](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/108438716/07_188120cc5b3dd86ef48c4f677c5bfad1.png)

Project Name: Power Monitoring System Equipment

Product: XM5728-SOM-V2.3 Core Board Based on AM5728

Project Description:  
The XM5728-SOM-V2.3 is built around the AM5728 DSP+ARM processor, featuring low power consumption, a hardware floating-point unit, rich peripheral interfaces, dual Ethernet support, stable performance, and low cost, making it the preferred choice for industrial control applications.
