The production line integrated tester offers excellent scalability and rich interfaces. Customers can use it with different test fixtures according to the testing requirements of various PCBA products, enabling rapid production line switching by changing test fixtures. The product features built-in relay switch management, switching, and connection to the corresponding test points on the DUT. This product integrates a main control board module, multimeter module, audio board module, signal source module, oscilloscope module, and power board module, providing various comprehensive functional tests for different products. This product comes with an automated test platform (Archon). Archon is a software platform for developing, running, and managing test plans and test cases, with MES integration support. This product can be widely applied in R&D debugging, NPI, and production testing of electronic product PCBAs. It can replace product debugging jigs, saving labor costs on the production line, shortening test time, increasing production capacity, and providing an economical test solution for customers.
