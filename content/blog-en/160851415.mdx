The XM250M3G digital interference source is a single-board miniaturized digital interference source. This board can generate arbitrary IF frequency output from 500MHz to 3000MHz, and the interference bandwidth, ranging from 50MHz to 250MHz, can be arbitrarily set. The primary interference signal type is Gaussian white noise. This digital board is mainly used in jamming scenarios within electronic countermeasures (ECM).

II. Technical Specifications:

1. Output Frequency Range: 500MHz-3000MHz;

2. Instantaneous Interference Bandwidth: 50MHz/100MHz/150MHz/200MHz/250MHz (settable);

3. Interference Type: Gaussian White Noise / Comb Spectrum;

4. Output Power: ≥-15dBm;

5. Input Signal Range: -25dBm~0dBm;

6. Power Supply: +12V/0.5A;

7. Control Interface: RS422;

8. Dimensions: 68mm\*42mm\*8mm;
