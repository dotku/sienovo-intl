## **XM7016 - Ethernet Data Acquisition Card**

The XM7016 is an Ethernet-based high-speed data acquisition card, featuring 16 true differential input channels, 24-bit resolution, a maximum sampling rate of 256 ksps per channel, a total synchronous sampling rate of 4.096 Msps across 16 channels, precision pre-gain amplification, and integrated IEPE/ICP hardware support. This product utilizes multiple high-precision 24-bit ADC units and a pre-differential amplification module developed by our company over many years. This design provides advantages such as high speed, high resolution, high precision, ultra-low noise, high common-mode rejection ratio, wide measurement range, and low temperature drift, making it suitable for various applications requiring precise high-speed acquisition. The acquisition card supports Ethernet/USB interface communication and offers highly customized services. All components of the XM7016 acquisition card are industrial grade, featuring full metal shielding, making it suitable for applications in environments with strong industrial interference, and possessing advantages such as moisture-proof, shock-proof, and anti-interference capabilities.

IEPE Mode Description

This acquisition card integrates IEPE/ICP hardware support.

The card internally integrates a 24V power supply unit, constant current drive, and receiving unit. Each channel can be individually switched via software settings between the common "Analog Input Mode" or "IEPE Mode".

In IEPE mode, the output is 24V 4mA (compatible with 2mA), and the ADC input is AC-coupled.

All 16 channels can be independently switched and controlled.

Differential input connection can further reduce transmission interference.

Connect IEPE sensor to input port.

When connecting an IEPE sensor, AGND serves as the negative input terminal and AIN+ as the positive input terminal.

\*The corresponding input must be set to IEPE MODE in the software.

PC Software
