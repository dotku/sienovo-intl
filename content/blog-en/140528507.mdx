A 3U 19-inch rackmount system, featuring a high-performance, low-power i5 processor, capable of operating in environments with a temperature range (TX: -25 to +55º C), and meeting the requirements of European railway application standard EN50155 for electromagnetic compatibility, shock, and vibration testing. It is a product designed for railway and metro Passenger Information Systems (PIS).

*   *   3U 19-inch Rackmount System
    *   High-performance, low-power i5 processor
    *   Large-capacity 4GB DDR3 high-speed memory
    *   2 x M12-D 10/100Mb Ethernet ports, upgradable to 1000Mb (M12-A)
    *   PCI, PCI-E, MiniPCIe expansion slots, supporting various applications such as video encoding cards, 3G, and WiFi
    *   4 x DB9 isolated serial ports (2 x RS232, 2 x RS485)
    *   8-channel isolated I/O, remote switch signal input
    *   DC 60V～160V input, meeting railway vehicle power supply requirements
    *   Fully compliant with EN50155 standard

System Configuration

Processor:

   Onboard Intel i5 2515E processor

Chipset:

   Intel HM65

Memory:

   Onboard DDRIII 1333MHz 4GB memory
   1 x SODIMM slot (supports up to 8GB)

Storage:

   1 x SATA CFast slot
   1 x SATA SSD slot
   4 x 2.5-inch SATA hard drive interfaces

Display:

   2 x VGA ports
   2 x DVI-D ports

I/O Interfaces

Network:

   2 x M12-D 10/100Mb Ethernet ports, upgradable to 1000Mb (M12-A)
   1 x RJ45 10/100/1000Mb Ethernet port

Serial Ports:

   4 x DB9 isolated serial ports (2 x RS232, 2 x RS485)
   1 x DB9 non-isolated serial port (RS232)

USB:

   4 x USB2.0 Type-A ports

GPIO:

   8-channel isolated GPIO (terminal block)

Audio:

   Line In, Line Out (terminal block)

Expansion Bus:

   1 x 32-bit PCI, 1 x PCI-E*1, 3 x MiniPCI-E

Operating Environment:

   -25℃～+55℃; 5%～90% RH at 40℃ (non-condensing)

Storage Environment:

   -40℃～+70℃; 5%～90% RH at 40℃ (non-condensing)

Power Supply:

   DC 60-160V input

Dimensions (W×H×D):

   482.4mm × 132.5mm × 300mm
