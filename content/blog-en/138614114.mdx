FPGA+MCU+SDRAM Solution for Pattern Making and Fabric Cutting in Garment Factory Production and Design Processes

Client Application: Garment Factory Manufacturing

Key Features:

1. Supports stepper motors and DC motors
2. Supports simultaneous 3-axis motor coordinated operation
3. Supports Ethernet/USB connectivity control
4. Supports LCD screen status display
5. Supports multiple HP11/HP45 print heads
6. Supports electromagnetic cutters and stepper motor cutters
7. Supports PC host software for configuration and print data
8. Supports material jam and collision detection and protection
9. Supports protocol data decompression
