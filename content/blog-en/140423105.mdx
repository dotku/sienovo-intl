*   Standard FMC mezzanine card, compliant with VITA57 specification
*   Supports 1-channel CameraLink Full mode or 2-channel CameraLink Base mode input
*   Supports 80-bit Deca mode (Full Plus mode), with a maximum bandwidth of 850 MByte/s
*   CameraLink receiver chip supports up to 85MHz clock frequency
*   Features 1-channel RS232/RS422/RS485 interface
*   Features 1-channel DVI display output interface, supports 1080P resolution

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/140423105/01_214d611c60ac49c892287cd1fa41c9ed.png)

This mezzanine card module is based on the FMC interface standard, featuring 1-channel CameraLink Full mode (or 2-channel CameraLink Base mode) acquisition and 1-channel HDMI (DVI) video output. The module has two CameraLink ports (SDR, 26PIN) that can be hardware configured for 1-channel Full input or 2-channel Base mode input. The CameraLink interface supports 80-bit Deca mode (i.e., Full Plus mode). This module supports 1-channel DVI video output (HDMI connector), with a maximum resolution of 1080P. This module is specifically designed as an FMC mezzanine card. By pairing it with different FMC carrier cards, an image analysis and verification platform based on CameraLink video interfaces can be quickly established. It can be widely applied in teaching, scientific research, and the development of high-end video and image products.

#### Technical Specifications

*   Image Interface Performance:
    1.  Supports 2-channel CameraLink Base digital camera input;
    2.  Supports 1-channel CameraLink Full mode input;
    3.  Supports 80-bit Deca mode, with a maximum transmission bandwidth of 850 MByte/s;
    4.  2 SDR26 connectors, model: 12226-5150-00FR;
    5.  Supports 1-channel DVI digital display output, with a maximum resolution of 1920x1080;
    6.  1 HDMI interface, model: 10029449-001RLF;
*   FMC Interface Specifications:
    1.  Standard FMC mezzanine card, compliant with VITA57.1 specification;
    2.  Board dimensions: 84.1 x 69 mm;
    3.  FMC connector model: ASP-134488-01;
    4.  Board powered by +3.3V;
    5.  3 groups of X/Y/Z signal lines with equal length design;
    6.  Supports IIC interface;
*   CameraLink Channel:
    1.  Decoder chip model: DS90CR288A;
    2.  Decoder chip maximum operating clock: 85MHz;
*   Other Interfaces:
    1.  1 RS422 interface;
    2.  8 GPIO inputs/outputs;
    3.  8 LED indicators;
*   Physical and Electrical Characteristics:
    1.  Board dimensions: 84.1 x 69mm;
    2.  Board power supply: 0.5A max @ +3.3V (±5%);
    3.  Cooling method: Natural air cooling;
*   Environmental Characteristics:
    1.  Operating temperature: -40°C to +85°C;
    2.  Storage temperature: -55°C to +125°C;
    3.  Operating humidity: 5%~95%, non-condensing;
