Domestic industrial processor, Loongson 2K1000 motherboard, 100% domestically produced solution

A fully domestic component solution featuring a homegrown processor.

The Xinmai 2K1000 development board adopts the Loongson 2K1000 processor, integrating two GS264 processor cores, operating at a main frequency of 1GHz, equipped with a 64-bit DDR3 controller, and various system I/O interfaces. Verified through extensive project applications, it is widely used in intelligent transportation, automated financial equipment, energy systems, industrial control, CNC machines, charging stations, and other industries.

**Loongson 2K1000 Processor Main Features**

- Integrated dual-issue superscalar GS264 dual 64-bit processor cores, 1GHz main frequency  
- On-chip shared 1MB L2 cache  
- Integrated GPU  
- On-chip display controller supporting dual DVO output  
- Integrated 64-bit DDR3 controller running at 533MHz  
- Integrated 2 x4 PCIe 2.0 interfaces;可 be split into 6 independent x1 interfaces  
- Integrated 1 SATA 2.0 interface  
- Integrated 4 USB 2.0 interfaces  
- Integrated 2 RGMII Gigabit Ethernet interfaces  
- Integrated HDA/I2S interface  
- Integrated RTC/HPET module  
- Integrated 12 UART controllers  
- Integrated 1 NAND controller  
- Integrated 2 CAN controllers  
- Integrated 1 SDIO controller  
- Integrated 2 I2C controllers  
- Integrated 1 LIO controller  
- Integrated 1 VPU decoder  
- Integrated 1 CAMERA interface controller  
- Integrated 1 temperature sensor  
- Integrated dynamic power management module  

**Processor Core**

GS264  
MIPS64 R2 architecture compatible  

**GPU**

- Supports OpenGL ES 2.0, OpenGL ES 1.1  
- Supports OpenVG  
- Futuremark certified  

**Display Controller**

- Dual DVO output  

**SATA Controller**

- 1 independent SATA port  
- Supports SATA 1.5Gbps and SATA2 3Gbps transmission  

**USB 2.0 Controller**

- 4 independent USB 2.0 HOST ports  
- Port 0 fixed in OTG mode  
- Compatible with USB 1.1 and USB 2.0  
- Built-in EHCI controller for high-speed transmission up to 480Mbps  
- Built-in OHCI controller for full-speed and low-speed transmission  
- Low-power management  

**UART**

- 1 full-function UART with flow control: TXD, RXD, CTS, RTS, DSR, DTR, DCD, RI  
- Up to 12 UART interfaces  

**PWM**

- 32-bit counter  
- Supports pulse generation and capture  
- 4-channel controller  

**ACPI Power Management**

- Dynamic voltage and frequency scaling for processor cores  
- Full-chip clock gating  
- PHY power-down capability  
- USB/GMAC wake-up support  
- Auto power-on upon power restoration  

**PCIe Interface**

- PCIe 2.0 compliant  
- Dual independent x4 interfaces  
- One x4 interface可 be configured as 4 x1 interfaces  
- One x4 interface可 be configured as 2 x1 interfaces  

**I2S Controller**

- Supports I2S input in master mode  
- Supports I2S output in master mode  
- Supports 8, 16, 18, 20, 24, 32-bit data width  
- Supports mono and stereo audio data  
- Supports sampling frequencies of (16, 22.05, 32, 44.1, 48) kHz  
- Supports DMA transfer mode  

**CAN Bus**

- Dual CAN interfaces  
- Multiplexed with GPIO  

**Encryption Module**

- Supports AES and DES algorithms  
- Supports RSA algorithm  

**SDIO Controller**

- 1 independent SDIO controller  
- Compliant with SD Memory 2.0 / MMC / SDIO 2.0 protocols  
- Supports SDIO boot  

**GPIO**

- 4 dedicated GPIO pins, 56 multiplexed GPIO pins  
- Remaining pins multiplexed with other interfaces, operating in respective interface voltage domains  

**VPU Decoder**

- Supports H.264  
- Supports H.263  
- Supports SVC  
- Supports MPEG-4  
- Supports MPEG-2  
- Supports MPEG-1  
- Supports Sorenson Spark  
- Supports JPEG  
- Supports RV8, RV9, RV10  
- Supports VP6.0, VP6.1, VP6.2
