A 4-channel QSFP28 100G Fibre Channel adapter based on a PCIe bus architecture. The card features one PCIe Gen3x16 host interface and a total of four QSFP28 100G fiber interfaces, enabling real-time data acquisition, real-time buffering, and high-speed PCIe transmission for four QSFP28 100G fiber channels. The card uses Xilinx's high-performance Virtex UltraScale+ series FPGA XCVU3P as the main processor, with two independent sets of 80-bit DDR4 SDRAM for large-capacity data buffering. The card also includes one RJ45 Gigabit Ethernet port and several I/O control signals. This standard full-height PCIe card can be widely applied in server-based Fibre Channel data acquisition, data transmission, and other scenarios.

#### Key Features

*   Standard PCI Express full-height card;
*   Compliant with PCI Express Gen3.0 specification, selectable x1, x4, x8, or x16 modes, with a bandwidth of 128Gbps;
*   Fiber transmission performance: Supports 4-channel QSFP28 100G fiber, 25Gbps/lane line rate;
*   Fiber transmission protocols: Supports Aurora and various other high-speed protocols;
*   Dynamic storage performance: 2 sets of 80-bit 4GByte DDR4 SDRAM, operating clock 1200MHz, operating data rate 2400Mbps;
*   The card supports one FMC expansion interface;
*   The card supports multiple discrete I/O interfaces for GPIO expansion;
*   Onboard 2x 256Mbyte QSPI NorFlash for FPGA loading;

#### Interface Characteristics

*   Front panel supports 3x QSFP28 Cages (supports 25Gbps/lane);
*   Rear panel supports 1x QSFP28 Cage (supports 25Gbps/lane);
*   Rear panel supports 1x RJ45 Gigabit Ethernet interface;

#### Software Support

*   Optional integrated Board Support Package (BSP):
    1.  FPGA DDR4 interface test program;
    2.  4-channel 100G fiber interface program, supports Aurora protocol;
    3.  PCIe interface test DEMO;
*   Provides drivers and Application Programming Interfaces (API):
    1.  Supports Windows 7 32-bit/64-bit operating systems;
    2.  Supports Linux operating systems;

#### Application Scenarios

*   Image acquisition systems
*   Analog data fiber acquisition and transmission systems
*   Radar system hardware-in-the-loop simulation
*   Network hardware acceleration
