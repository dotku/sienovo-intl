German company Sigloch Maschinenbau provides customized production lines for the entire process of binding brochures and books, including collators, adhesive binders, end sheet machines, staplers, backing machines, drying conveyors, and ribbon feeders.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/141283049/01_804ad149d037c1374a3abda039c70bc1.jpeg)

All equipment in the entire automated production line is controlled by the XC200 modular controller from Eaton Moeller (USA). This PLC is developed based on the German **CODESYS** industrial software platform. The Moeller XC200 controller can communicate with multiple XI/ON remote I/O devices. At the communication level, the CANopen bus handles communication between field modules, and smart drives are also connected via CANopen. Furthermore, communication between the PLC and operator units, as well as communication between machine units, is realized via Ethernet. Finally, for the operator interface, Moeller XVH300 and XV400 touch panels are used for human-machine interface (HMI) operations.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/141283049/02_5f2b5ecbe1df01ea57a25cb518c9c0cb.png)

**Controller Manufacturer: Eaton Moeller**

**XC200 Controller:**

*   512 KB program memory, 256 KB data memory
*   Eight digital inputs
*   Six digital outputs
*   Equipped with two 50 kHz high-speed counter inputs, two interrupt inputs, one incremental input
*   Supports local expansion of 15 XIOC modules
*   One MMC card slot
*   Integrated battery-backed real-time clock
*   Supports 32 KB of retained data
*   Supports programmable RS232 interface
*   Equipped with CANopen fieldbus interface
*   Equipped with Ethernet interface
*   Supports OPC server
*   Integrated Web server
