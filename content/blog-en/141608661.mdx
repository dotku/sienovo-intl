This ruggedized high-performance motion controller, built on an FPGA+DSP real-time control system architecture, supports multi-channel parallel synchronous real-time high-speed sampling. Data is directly passed point-by-point to the DSP for algorithm processing at the low level to ensure real-time performance. It can achieve high-speed point-by-point control output, ensuring the control system can excellently fulfill highly demanding real-time control tasks in scenarios involving multi-input, multi-output high-speed control.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/141608661/01_d2a84b25ba0b8db57e65a64762b9bc66.jpeg)

**Product Features**

**Accurate Measurement**

Extremely strong anti-interference capability ensures precise measurement of the controlled object's output signals.

**Superior Computing Power**

The core processor utilizes an 8-core high-performance DSP, possessing superior computational processing power.

**Fast Control Response**

High-speed processor hardware (DSP/FPGA) executes program instructions in real-time, reducing control response time to the microsecond level.

**Parallel Synchronous Operation**

Based on a strict phase synchronization mechanism, it ensures high-speed real-time parallel synchronous operation of multi-channel inputs and outputs.

**DMA Data Transfer**

Control status is transmitted to the upper-level machine in real-time via DMA, ensuring high-speed, long-term data transmission without crashes, data loss, and with stable reliability.

**Shared Low-Level Interface**

Provides programming interfaces required for control algorithm development, including timing interrupts, real-time access to input data, and real-time refreshing of output channel data.

**Key Specifications**

**Controller Core**

DSP: TMS320C6678, 8 cores, 1GHz clock speed

ZYNQ FPGA: XC7Z100-2FFG900I

**Communication Method**

Gigabit Ethernet communication

**Analog Signal Acquisition**

Number of Channels: 16 channels

Input Modes: AC, SIN-DC, IEPE

Maximum Sampling Rate: 35kHz

Voltage Ranges: ±5000mV, ±500mV, ±50mV, ±5mV

**Analog Signal Output**

Number of Channels: 16 channels

Voltage Output Range: ±10V

Maximum Single-Channel Conversion Speed: 35kHz

**Programming Interface**

Open API for low-level DSP development
