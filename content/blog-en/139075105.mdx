1. Product Introduction  
    1.1 Product Overview

This is a dual-CPU motherboard designed around the Intel Xeon Gold series processors, compliant with commercial EEB motherboard specifications, specifically engineered for high-performance, fully ruggedized servers. It integrates 64GB DDR4 memory with surface-mounted memory chips and provides abundant high-strength I/O interface connectors (6 SATA 3.0, USB 3.0, Gigabit Ethernet ports, VGA, and RS232 interfaces). Featuring high reliability, full ruggedization, high performance, and rich interfaces, this motherboard serves as the core platform for high-reliability, fully ruggedized hyper-converged computing servers.

Motherboard dimensions: 304.8\*295\*2.4mm.

2. Key Features and Performance  
    2.1 Processor

Supports ≥2 processors; uses Intel Gold 5120 CPUs, each with a base frequency ≥2.2GHz and ≥14 cores.

2.2 Memory

Each processor has 32GB DDR4 memory (with ECC) pre-soldered on board and supports expansion via 4 RDIMM slots. Dual-processor configuration provides 64GB DDR4 memory (with ECC) on board and supports up to 8 RDIMM slots. Maximum memory capacity supported by the motherboard is 320GB.

2.3 Storage

1) System disk interface: 1 M.2 NVMe interface for operating system storage;  
2) Disk type: SATA interface SSDs, each supporting ≥2TB capacity;  
3) Number of SATA disk interfaces: ≥6; supports RAID levels: RAID 0 (non-redundant striping), RAID 1 (mirroring), RAID 5 (distributed parity), RAID 10 (mirrored striping).

2.4 Gigabit Ethernet for Service Applications

Provides 5x 1000BASE-T Gigabit Ethernet ports, including one dedicated service port with BMC management capability via NCSI.

2.5 Display Interface

1) 2x VGA video interfaces (one ruggedized connector, one standard VGA; only one can be active at a time);  
2) Maximum VGA resolution ≥1920x1200 @ 60Hz.

2.6 USB 2.0 Interfaces

Provides 5x USB 2.0 ports (3 with ruggedized connectors, 2 standard).

2.7 Management Gigabit Ethernet Interface

Provides 1x 1000BASE-T Gigabit management Ethernet port.

2.8 RS232 Interface

Provides 1x 3-wire RS232 ruggedized connector interface.

2.9 Fan Connectors

Provides 4x fan connectors.

2.10 Front Panel Indicators and Power Control Signals

Provides 11 LED signals: Power 1, Power 2, Run, HDD (red and green), Warning, Network 1–5, and Power Switch; plus 1x power on/off signal.

2.11 Power Interfaces

1) Input power: DC 12V ±5%, DC 5V ±5% (standby);  
2) Fan power: 4x fan power connectors, DC 12V ±5%;  
3) SATA disk power: 6x SATA power connectors, DC 5V ±5%.

2.12 Graphics

Integrated graphics via AST2500 chip.

2.13 Software Support

Supports deployment of domestic operating systems such as Kylin OS, Oracle Database, HanGao Database, and domestic Dameng Database server editions.

2.14 KVM Functionality

The motherboard supports KVM connectivity, enabling human-machine interaction for server operation and management via KVM.

Supports KVM over IP—remote management-capable KVM switch with remote monitoring capabilities.

2.15 Health Diagnostics and Monitoring

1) Health diagnostics function:  
   - Reports status information via Simple Network Management Protocol (SNMP), capable of periodically sending self-check data to report server status including power state, availability, core parameters, fan status, etc.  
   - System maintenance functions support BMC system recovery and updates.

2) Web-based Management:  
   - Provides sensor monitoring, fan cooling control, power status monitoring, health log management, and server runtime status monitoring.

3) SNMP Remote Management:  
   - Offers real-time query interfaces for power state, temperature, voltage, fan status, power consumption, memory utilization, CPU usage, CPU temperature, used and remaining disk capacity, and RAID status.  
   - Also provides control interfaces for power on/off and fan speed adjustment.

4) SNMP Fault Reporting (higher-level motherboards can also retrieve the following information via "access"):  
   - Reports power state, temperature, voltage, fan status, power consumption, memory utilization, CPU usage, CPU temperature, used and remaining disk capacity, network connection status, RAID status, or fault information to a designated IP address.  
   - Provides O&M alarm message interface; when abnormal operation is detected, sends real-time alert messages in SNMP packet format to a specified address. Fault reports must include at least fault code, cause, impact, and recommended resolution.
