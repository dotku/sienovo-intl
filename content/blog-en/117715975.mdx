1. Core Board Introduction

➢ SOM-XM5728 is an industrial-grade core board developed by Xinmay Technology based on the TI AM5728 processor. It features an ENIG (Electroless Nickel Immersion Gold) lead-free design, a 10-layer PCB, and industrial-grade reliability, making it suitable for applications such as motion control systems, machine vision processing, image recognition, audio analysis, video analytics, and high-end measurement instruments.

Customers can choose different CPU, memory, and storage configurations depending on their specific application requirements. The core board is built around the TI AM5728 processor, integrating dual ARM Cortex-A15 cores and dual C66x™ Floating-Point VLIW DSPs running at 750MHz, along with PRU-ICSS modules supporting multiple industrial protocols. It supports industrial-grade DDR3 memory ranging from 1GB to 4GB and eMMC flash storage from 4GB to 16GB, enabling flexible and convenient expansion.

The SOM-XM5728 supports industrial communication protocols including EtherCAT, EtherNet/IP, and PROFIBUS, enabling seamless bus control and industrial networking. With dual C66x DSPs, it can implement more complex and high-precision motion control interpolation algorithms. Additionally, by connecting to an FPGA via the GPMC bus, the board can extend its capabilities in industrial control and image processing.

Equipped with 8x McASP interfaces and dual C66x DSPs, the SOM-XM5728 enables advanced audio processing algorithms for 16- or 32-channel applications, supporting use cases such as sound source localization and echo cancellation.

The board delivers powerful video encoding and decoding performance, supporting hardware acceleration for up to 1x 1080P60, 2x 720P60, or 4x 720P30 video streams, as well as H.265 software decoding. Leveraging the computational power of the dual C66x DSPs, it enables complex image recognition, video analysis and recognition, real-time data processing, and other high-end AI applications.

It features a high-performance GPU with a dual-core SGX544 3D accelerator and GC320 2D graphics engine, supporting OpenGL ES 2.0.

The SOM-XM5728 software SDK provides Linux 4.14.67 and DSP RTOS systems, supporting multi-core development with OpenCL, OpenMP, and SysLink IPC. It includes communication methods and sample cases for ARM-DSP multi-core interaction, significantly simplifying development for high-speed signal processing, audio/video data analysis, and similar tasks.
