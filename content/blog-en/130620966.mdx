I. Processor Introduction

The RK3588 is a low-power, high-performance processor suitable for Arm-based PCs, edge computing devices, personal mobile internet devices, and other digital multimedia applications. It adopts an 8nm process, integrating quad-core Cortex-A76 and quad-core Cortex-A55, each with NEON. The RK3588 supports 8K@60fps decoding, features a high-quality JPEG encoder/decoder, a professional image pre-processor, and a built-in GPU fully compatible with OpenGLES 1.1, 2.0, and 3.2. The RK3588 introduces a new generation, fully hardware-based, maximum 48-megapixel ISP, a built-in NPU supporting INT4/INT8/INT16/FP16 mixed precision computing, and supports Android 12 and Linux systems. II. Product Advantages

Powerful Performance
It is a brand new high-end AIoT application chip, adopting an 8nm LP process, equipped with an octa-core 64-bit CPU (quad-core Cortex-A76 and quad-core Cortex-A55), with a main frequency up to 2.4GHz, 8GB RAM, and 32GB storage.

Quad-core Architecture GPU
Integrates a Mali G610 MP4 quad-core GPU, supports OpenGLES 1.1, 2.0, 3.2, OpenCL 2.2, and Vulkan 1.2. A special 2D hardware engine with MMU will maximize display performance and provide very smooth operation.

Powerful Video Codec
8K encoding + 8K decoding, multiple video sources decoded simultaneously. Supports 8K@60fps H.265/H.264/VP9/AV1 video decoding and 8K@30fps H.265/H.264 video encoding.

Built-in Independent NPU
The RK3588 integrates Rockchip's fourth-generation AI computing engine with full independent intellectual property rights, improving MAC unit utilization and reducing bandwidth consumption by 30%. With 6 Tops of NPU computing power, it empowers various AI scenarios.

Rich Interfaces
It features a comprehensive set of interfaces including dual Gigabit Ethernet, 5G/4G module, HDMI input, HDMI output, PCIe 3.0, PCIe 2.0, SATA 3.0, USB 3.0, USB 2.0, and Type-C.

Supported Systems
Smoothly runs Android 12 system, with fully open-source code.

III. Features Introduction

Utilizing the Rockchip RK3588 processor, this is a new generation of high-end AIoT application chip, adopting an 8nm LP process. It features an octa-core 64-bit CPU (quad-core Cortex-A76 + quad-core Cortex-A55 architecture), integrates a Mali G610 MP4 quad-core GPU, and has a built-in AI accelerator NPU with computing power up to 6 Tops. It integrates independent 8K video hardware encoders and hardware decoders, providing many powerful embedded hardware engines. While offering stronger performance, power consumption is also well controlled. Its performance is outstanding across various industries.

Built-in NPU, computing power up to 6 Tops

Built-in NPU, supports INT4/INT8/INT16/FP16 mixed precision computing, with computing power up to 6 Tops. Supports deep learning frameworks, and network models based on a series of frameworks such as TensorFlow/MXNet/PyTorch/Caffe can be easily converted, empowering various AI scenarios.

8K Video Codec, Clearer Images

The RK3588 supports 8K@60fps H.265/H.264/VP9/AV1 video decoding and 8K@30fps H.265/H.264 video encoding.

Supports simultaneous decoding of up to 40 channels of 1080P HD video. Its video codec capabilities allow for 8K high-definition presentation, with finer image details.

The RK3588 can achieve 4-screen display, supporting true 8K or 4K@120Hz display capabilities, and also supports 1080P HD images.

Truly achieves ultra-high-definition, multi-resolution arbitrary combination multi-screen heterogeneous display functionality. Effectively enhances the scalability for industry customization.
