This 20G Ethernet real-time packet capture and storage card, based on PCIe bus architecture, is a half-height, half-length PCIe card, suitable for mainstream half-height server chassis or micro-workstations. The board utilizes an FPGA controller, enabling real-time transmission and reception of 20G Ethernet data. It applies algorithms for analysis, filtering, and encryption processing to Ethernet packets, stores the data in real-time, and can also perform real-time playback of recorded data files. It offers high stability, reliability, and real-time performance. This card can be widely used in data centers, servers, and other applications.

#### Key Features

*   Standard PCI Express half-height, half-length card, suitable for half-height chassis;
*   Compliant with PCI Express Gen2.0 specification, with optional x1, x4, or x8 modes, offering theoretical bandwidth up to 40Gbps;
*   Supports 2-channel 10 Gigabit network interfaces;
*   Onboard 2 GByte DDR3 SDRAM memory;
*   Supports packet lengths from 64B to 1518 Bytes;
*   Supports hardware playback;
*   Storage bandwidth 800 MByte/s;
*   Onboard 1 x 32 MByte SPI Flash, for storing small amounts of parameter information;
*   Onboard 1 x 128 MByte BPI Nor Flash, for FPGA loading;

#### Interface Characteristics

*   Front panel supports 2 SFP+ Cages (max 10Gbps/lane);
*   Front panel supports 2 LED indicators;

#### Software Support

*   Optional integrated Board Support Package (BSP):
    1.  2-channel 10G Ethernet interface program;
    2.  PCIe DMA data transfer interface;
*   Provides drivers and Application Programming Interfaces (API):
    1.  Supports Windows 7 32-bit/64-bit operating systems;
    2.  Supports Win Server 2008/2012;

#### Application Scenarios

*   Video and Image Acquisition and Processing
*   Data Acquisition and Processing
*   High-Performance Computing (HPC)
*   Network Hardware Acceleration
