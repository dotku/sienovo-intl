## ******EC07-********Industrial-Grade Edge AI BOX Based on NVIDIA Jetson Xavier NX Core Module******

**(Carmel ARMv8.2 + 8GB DDR4L + 16GB eMMC)**

This is an industrial-grade, high-performance Edge AI BOX built around the NVIDIA Jetson Xavier NX core module. By combining the Xavier NX core module with a self-developed carrier board and leveraging the core module's powerful expansion bus capabilities, this compact system meets stringent size requirements while delivering exceptional inference processing performance. It is a compact edge computing solution ideal for high-performance video and image recognition tasks—such as identifying people, objects, text, and other visual features. When paired with wired or wireless network connectivity and cloud servers, it enables a complete AIoT solution combining cloud-based learning with edge-side inference. This industrial computer is well-suited for AI and IoT applications and is widely used in security surveillance, smart communities, smart campuses, drones, unmanned retail, robotics, and intelligent healthcare.

**Industrial Stability:** Designed with industrial-grade reliability principles, the device features a full-aluminum enclosure that is compact and robust. The chassis uses passive heat dissipation for silent operation. The exterior is sandblasted and finished in industrial gray, seamlessly blending into industrial environments.

**Outstanding Performance:** Complies with safety standards: GB4943.1-2011, EMC standards: GB17625.1-2012, GB/T9254-2008; Wide temperature operation: -20°C to 50°C; Dual DC power input interfaces: 12V DC, equipped with both Phoenix terminal and adapter power inputs for flexible deployment in indoor, outdoor, and vehicular environments; Vibration resistance: 0.5g rms / 5–500Hz / random vibration; Mounting options: rubber feet or bracket mounting; Programmable green LED indicator; Carrier board compatible with both Xavier NX and Nano modules, allowing flexible selection. **Configurable by Demand:** Choose different core modules and accessories based on performance and functional requirements.
