### Tengrui D2000-8 MXM VPX

1. Overview

The XMVPX-108 is a low-power logic and graphics processing VPX blade based on the Phytium D2000/8 processor, featuring onboard 32GB DDR4 memory and equipped with the Phytium X100 chipset to provide general-purpose I/O interface functions. The GPU adopts an MXM small plug-in card form factor, paired with an 8GB graphics card. It provides network interfaces such as RapidIO, 40GBASE-KR4, and 1000BASE-X to meet various network interconnection requirements for system business interconnection and control plane interconnection. The Tengrui D2000 integrates 8 high-performance FTC663 cores independently developed by Phytium, compatible with the 64-bit ARMv8 instruction set, clocked at 2.3GHz, with a TDP power consumption of 35W. It also integrates rich I/O interfaces and supports Phytium's self-defined processor security architecture standard PSPA 1.0, meeting the demands for performance and trusted security in more complex application scenarios.

It can be widely applied in edge computing gateways, intrusion detection, VPNs, network monitoring, and other application fields.

3. Terminology

Central Processing Unit (CPU): As the computational and control core of a computer system, it is the ultimate execution unit for information processing and program execution. Since its inception, the CPU has achieved tremendous development in logical structure, operational efficiency, and functional extensibility.

DDR4: A type of computer memory specification. It belongs to the SDRAM family of memory products, offering higher operating performance and optimized voltage compared to DDR3 SDRAM. It is the successor to DDR2 SDRAM (Synchronous Dynamic Random Access Memory), offering up to eight times the performance, and is currently a popular memory product specification.

SSD: Commonly known as Solid State Drive. It is widely used in various fields such as military, automotive, industrial control, video surveillance, network monitoring, network terminals, power, medical, aviation, and navigation equipment.

BIOS: An acronym for "Basic Input Output System". It is the first software loaded when a personal computer starts up.
