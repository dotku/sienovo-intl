****Chapter 1: Product Introduction****

****1.1 Product Overview****

The XM-D2000GW is a high-performance network security motherboard based on the Phytium D2000 8-core processor and X100 chipset. The D2000 is Phytium's first 8-core desktop platform processor, supporting dual-channel DDR4-2666 memory. The chip integrates national cryptographic SM2/SM3/SM4/SM9 acceleration engines and supports single-precision and double-precision floating-point instructions, accelerating network data processing. It can be widely applied in fields such as edge computing gateways, intrusion detection, VPNs, network monitoring, and security auditing.

****1.2 Product Features****

  Onboard Phytium D-2000 8-core 2.3GHz high-performance processor
  Utilizes Phytium X100 chipset
  Supports dual-channel DDR4-2666 memory, up to 64GB
  Onboard 6 Gigabit electrical ports, supporting 3 sets of third-generation intelligent bypass (optional)
  Onboard 4 Gigabit optical ports
  The side of the motherboard supports 1 PCIe X16 expansion slot (default X8 signal), and the rear supports 1 PCIe X8 expansion slot
  Supports network expansion modules such as 4 Gigabit electrical ports and 4 Gigabit optical ports
  Supports operating systems such as Kylin OS and UOS

****1.4 Product Specifications****

****Dimensions****

300*195mm (Length*Width)

****Processor****

Phytium D-2000, 8 cores, main frequency 2.3GHz, TDP: 40W

****BIOS****

Phytium UEFI

****Chipset****

Phytium X100

****Memory****

Supports dual-channel DDR4-2666MHz, using U-DIMM slots, maximum support 64GB

****Display Interface****

Supports 1 VGA pin header

****Bus Expansion****

1 PCIeX16 slot on the right side, supporting PCIe3.0 (default X8 signal)
1 PCIeX8 slot at the rear, supporting PCIe3.0

****Network Interfaces****

Front panel supports 6 electrical and 4 optical ports:

LAN1-LAN4 default uses Wangxun WX1860A4 network card chip;
LAN5-LAN6 default uses Wangxun WX1860A2 network card chip;

SFP1-4 default uses Wangxun WX1860A4+YT8521 network card chip;
Supports 3 sets of electrical port bypass (optional)

****USB****

2 USB3.0 interfaces on the front panel

2 USB2.0 interfaces via onboard pin header, using 2X5 2.54mm pin header

****M.2****

Supports 1 M.2 SSD, supporting SATA/NVMe protocols (NVMe protocol by default)

Supports 1 M.2 KEY-B interface, supporting 4G/5G modules, with onboard SIM card slot (optional)
Supports 1 M.2 KEY-E interface for WIFI support (optional)

****mSATA****

1 standard mSATA interface

****SATA****

2 SATA interfaces, supporting SATA3.0

****Watchdog****

Supports 1-channel watchdog function (CPU integrated)

****GPIO****

Default 6-channel digital GPIO from CPU (optional 8-channel digital GPIO from microcontroller), using 2X5 2.54mm pin header

****Real-time Clock****

External 3V lithium battery

****Serial Interfaces****

Supports 2 RS232 serial ports: 1 RJ45 interface on the front panel; 1 via onboard pin header, using 2X5 2.54mm pin header

****Power Supply****

Uses 8P_DC 12V power supply

****Operating Temperature****

0℃-40℃

****Storage Temperature****

-10℃-70℃
