The AI BOX is a high-performance, low-power edge computing device based on the RK1126 platform, delivering 2 TOPS of computing power. The RK1126 supports 4K H.264/H.265 decoding and 1080p@60fps H.264/H.265 encoding, with HDMI video output supporting up to 4K@30fps. With a lightweight and flexible design, it is widely applicable in fields such as new retail, intelligent security, smart classrooms, and gaming consoles.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/127326942/01_c06a3d32855ff05e341043e82d648c99.png)

Algorithms can be customized and ported according to customer requirements:

Supported functions include fall detection, climbing recognition, fight detection, absence from post detection, sleeping on duty detection, flame detection, smoke detection, fire lane obstruction detection, virtual fence (electronic perimeter), garbage overflow detection, and people counting.

Main Specifications:

The RV1126 HCoreV1.1 core board features Rockchip's RV1126 main chip, 1GB DDR3 memory, and 8GB eMMC storage. It supports up to 4 channels of 1080p@30fps AI computing, integrating a 2.0 TOPS NPU. It supports integration of algorithms from frameworks such as ADAS, Caffe, TensorFlow, MXNet, and PyTorch. Equipped with 100M/1000M Ethernet ports, firmware can be upgraded via SD card. It provides two USB interfaces, configurable as HOST or DEVICE. Optional external Wi-Fi, external active GPS, or 4G/5G modules are supported. Serial debugging is supported, along with one CAN bus interface, expandable to multiple CAN channels via serial. A secondary development SDK is provided, and shielding cover-based heat dissipation is supported. Custom PCBA development is available.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/127326942/02_bf244cda0299983ac5093df63ed8f12a.png)

This 8-channel edge computing box is built on the Rockchip RV1126 chip platform, a lightweight, independently developed video intelligent analysis device capable of running multiple mainstream AI algorithms. It provides enterprises with a lightweight, multi-platform, easily portable, high-performance, energy-efficient edge computing solution that supports secondary development. It is widely used in behavior analysis across industries such as power, energy, gas stations, smart construction sites, smart communities, intelligent transportation, and intelligent security.

Key Features:  
● 2 TOPS computing power module for flexible deployment across various algorithm environments.  
● Supports mainstream deep learning frameworks including ADAS, Caffe, TensorFlow, MXNet, and PyTorch.  
● Supports network models for face detection, flame detection, smoking detection, object tracking, recognition, posture estimation, hard hat detection, and more.  
● Equipped with 100M/1000M Ethernet ports, expandable to 4G/5G/Wi-Fi wireless connectivity.  
● Low-power design with passive and optional active fan cooling for stable and reliable operation.  
● Fully embedded, highly integrated structure suitable for desktop placement or wall mounting, enabling easy installation and maintenance.  
● Rich peripheral interface options including MSATA, M.2, RS485, RS232, TF card, LINE IN, and LINE OUT.
