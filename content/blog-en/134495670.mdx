1. Compact Board Design

This motherboard features a compact form factor measuring only 125\*85mm, making it smaller and more space-efficient, perfectly suitable for various high-end intelligent self-service terminals;

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/134495670/01_fab340c0a8af5d8acccb1263e1ccc7b5.png)  

2. Octa-Core High-Performance Processor

Powered by the RK3588S octa-core 64-bit processor built on 8nm LP process technology, with a maximum clock speed of up to 2.4GHz. It runs Android 12.0 and integrates an ARM Mali-G610 MP4 quad-core GPU along with a built-in AI accelerator NPU delivering 6TOPS of computing power, offering exceptional performance;

3. Multiple Display Interfaces with Triple Independent Displays

Supports LVDS/eDP/MIPI output: LVDS displays support up to 1920\*1200 resolution, eDP up to 3840\*2160, and MIPI up to 1920\*1200. Triple independent display output is supported;

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/134495670/02_5bd8d06c5a783cb9096fac7951a6904f.png)  

4. Support for USB/MIPI Cameras

Supports both single and dual USB cameras as well as MIPI cameras. The MIPI interface supports 1x4-lane @ 2.5Gbps/lane, with a maximum resolution of up to 48MP (8064x6048);

5. Rich Peripheral Expansion Interfaces

The AIoT-3588SE offers abundant expansion interfaces, including 6 USB ports (2 USB 3.0, 4 USB 2.0), 3 serial ports (2 TTL, 1 RS232), 1 I2C interface (for connecting peripherals such as touch panels), 4 GPIO interfaces, and 1 ADC interface. It supports plug-and-play compatibility with mainstream industry peripherals, fully meeting functional expansion needs for terminal devices;

6. High-Speed Network Communication Capability

Supports 10/100/1000M adaptive Ethernet, built-in Wi-Fi, and Bluetooth 5.0, enabling faster network communication speeds;

7. Support for Deep System Customization

Ships with Android 12.0 by default, but also supports multiple operating systems including Linux and OpenHarmony, providing a secure and stable system environment for terminal product development. Deep system customization is supported, with API reference code provided for system calls, fully enabling upper-layer application and APP development;

8. Application Scenarios

The AIoT-3588SE is primarily used in industry-specific tablet PCs, dual-screen dual-touch intelligent terminals, robots, smart fitness mirrors, and similar applications.

The RK3588 core board plays a significant role in industrial robot control products, delivering high-performance computing and multimedia processing capabilities for industrial automation and robotic control. Below are application scenarios of the RK3588 core board in industrial robot control:

Motion Control:

The RK3588 core board can control industrial robot motion, including joint control, trajectory planning, and speed regulation. It supports multi-axis control, enabling robots to perform precise motion tasks.

Vision Systems:

Industrial robots are typically equipped with vision systems for object recognition and positioning. The high computing power of the RK3588 supports complex image processing and machine vision tasks such as object detection, visual navigation, and automatic calibration.

Sensor Data Processing:

Robots interact with various sensors (e.g., LiDAR, cameras, force sensors) to gather environmental data. The RK3588 core board can process and analyze sensor data for decision-making and control.

Path Planning and Collision Detection:

The RK3588 supports advanced path planning and collision detection algorithms, helping robots avoid obstacles, plan optimal paths, and execute complex tasks.

Remote Monitoring and Control:

Industrial robots often require remote monitoring and control, allowing operators to manage robot operations remotely. The RK3588 core board supports remote access and control functions.

Human-Machine Interface (HMI):

It can be used to create intuitive HMIs, enabling operators to easily configure and monitor robot operations, including touch screen interfaces and multimedia feedback.

Automation Tasks:

The RK3588 core board can perform various automated tasks such as assembly, welding, painting, pick-and-place operations, improving production efficiency and consistency.

Safety:

Safety is critical in industrial robot applications. The RK3588 supports safety control and emergency stop functions to ensure operator and equipment safety.

Network Connectivity:

The core board supports multiple communication interfaces, including Ethernet and fieldbuses (e.g., EtherCAT), enabling communication with other automation devices and systems.
