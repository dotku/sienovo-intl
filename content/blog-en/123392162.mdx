A precision control system for injection molding machines was designed using a DSP as the lower-level controller and an ARM as the upper-level controller. The lower-level controller handles the execution of control algorithms and I/O operations, while the upper-level controller manages the human-machine interface and mold data. This control system offers excellent real-time performance and high control accuracy. However, a limitation is that control algorithms were not implemented directly on the control system, making it impossible to verify compatibility with different control algorithms.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/123392162/03_d97b785cefc4f41411fc4ae66d33ccdd.png)

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/123392162/04_775b2d3823587c1fe3c0095649d112eb.png)

The injection molding machine controller is a control system based on Texas Instruments' TMS320F28335 chip, serving as the core component of the entire injection molding machine pump-controlled servo system. The control system can collect signals from corresponding sensors to monitor in real time pressure, position, temperature, flow, and other status information of various components. It then processes these signals using designed control algorithms to execute actions required by the injection molding process.
