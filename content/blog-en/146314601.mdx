Board Overview:

This board serves as the main control board, primarily responsible for logic control, data processing, board communication management, system safety protection switching, and data storage functions.

Performance Specifications:

Power Supply: DC5V; DC3.3V; DC12V

FPGA: XC7A100T-2FGG484I

Storage: EEPROM 256kb (DSP)

Flash 256kb (FPGA)

Communication Interfaces (Front Panel Side): RS485 x1; CAN x 1; SFP(2.125G)x1;

SFP(1.0625G)x 2; SFP(100M)x 1;

ETH (100M)x 1

Communication Interfaces (Backplane Side): LVDS x 90 lines; CPCIe x 1; Aurora x 2; RS485 x 1; I2C x 1; CAN x 1; IO HardWire x 32

Dimensions: 160mm\*233.5mm\*1.6mm

Weight: 0.175kg

Operating Temperature: -25℃~70℃
