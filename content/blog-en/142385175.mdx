Smart Face Recognition Motherboard

Face Access Control Application

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/142385175/02_9040baead5e6b8e3b566a1957ad3da78.png)

The Face Payment Terminal utilizes an advanced Qualcomm SDM450 platform CPU, featuring a balanced hardware configuration and stable performance. The terminal is equipped with dual-channel infrared liveness detection cameras and supports reverse QR code scanning and IC card scanning. The motherboard uses a Qualcomm Octa-core 64-bit ARM Cortex-A53 processor, supporting multi-network Smart LTE Cat 6 communication, with a built-in Qualcomm Adreno™ 506 GPU, and running on the Android operating system. Its powerful performance and rich multimedia features fully meet customer demands for high speed and multimedia capabilities in industrial and consumer applications.

Applied in close-range face recognition scenarios, such as ticket redemption and face recognition in scenic spots, kindergarten entry management, vending machines, lottery machines, coin exchange machines, face payment in new retail, and identity verification at train stations.

Powered by the Qualcomm SDM450 chip, running Android 7.1.2, it integrates full Netcom 4G communication and 2.4G/5.8G WiFi, along with a MIPI display interface. The board includes a hardware watchdog, supports remote power on/off and network-scheduled power on/off, and enables remote OTA upgrades. It also supports various communication interface expansions, ensuring smooth operation for diverse network terminals.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/142385175/03_cea42596d345dbfbd394c3e6db8743be.png)

It offers 4 USB ports, 4 serial ports, 6 IO ports, and other communication interface expansions, compatible with mainstream industry peripherals. It supports dual MIPI output and can be paired with a dual-lens USB camera for AI face recognition applications. It comes with an integrated 4G communication module and BT 4.2.

◆ Supports mainstream face recognition algorithms on the market, providing corresponding technical integration support.

◆ Optimized hardware and system for face recognition application scenarios, achieving precise face recognition.

◆ Supports USB dual-lens (infrared + visible light) cameras for more accurate face recognition.

◆ Supports dual MIPI display output.

◆ Supports Android system customization, providing system API interface code for perfect support of customer's upper-layer APP development.

◆ Supports mainstream USB devices such as printers, card readers, PIN pads, fingerprint scanners, cameras, ID card readers, QR code scanners, etc., and provides demo test programs.
