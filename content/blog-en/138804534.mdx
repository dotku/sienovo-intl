A series of delivery robots based on the high-end smart terminal motherboard IoT-3399E have been developed for delivery services in various offline stores. These delivery robots enable intelligent recognition, precise delivery, automatic obstacle avoidance, autonomous scheduling systems, voice announcements, and information interaction, significantly improving service efficiency and offering consumers a more convenient and intelligent service experience.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138804534/01_8427d030a1111f931cf4fbad4636626d.jpeg)

Core Motherboard Advantages

The core motherboard used in the delivery robot project is the Xinmiao IoT-3399E, which is based on the RK3399 hexa-core 64-bit processor. It features exceptional processing performance, high computing power, and strong expandability, offering significant advantages.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138804534/02_177e077d55dc9915a71565036624ed49.jpeg)

**More Powerful**

**① Stronger Core**

CPU: RK3399 hexa-core processor with a dual Cortex-A72 + quad Cortex-A53 big.LITTLE CPU architecture, up to 1.8GHz clock speed

GPU: Mali-T860 GPU, supporting OpenGL ES1.1/2.0/3.0/3.1, OpenCL, and AFBC (frame buffer compression)

**② Stronger System**

Android 7.1 system, compatible with Linux/OpenHarmony operating systems

**③ Enhanced Functionality**

Superior display capabilities: Supports LCD displays with LVDS/eDP/HDMI interfaces, cut screens, 4K ultra-HD display via eDP/HDMI, and dual independent displays with independent touch

Supports MiPi Camera, up to 13MP

Stronger expansion capability: 7 USB ports, 3 RS232 ports, 1 RS485 port, 1 TTL port, 1 DEBUG port, 1 IIC port, 4 GPIO ports, 1 ADC port, and 1 IR port

**Faster**

① Faster Networking:

Supports 10/100/1000M adaptive Ethernet

Supports 5G and 2.4G Wi-Fi, with optional BT4.0

Supports 4G internet via PCI-E interface

② Faster Data Transfer: Supports USB3.0 and TYPE-C interfaces

**More Reliable**

The IoT-3399E is manufactured using 4mil immersion gold process. The core board’s gold fingers are corrosion-resistant, secured with four mounting screws for stable and reliable operation, capable of running 7x24 hours without failure. Subjected to multiple rigorous performance tests, each motherboard undergoes 121 test items over 16 hours. With rich system interfaces, over 100 practical interfaces are pre-packaged for secondary development.
