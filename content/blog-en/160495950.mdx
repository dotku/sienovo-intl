An RK3588-based ultra-small drone AI module, measuring **45×45mm and weighing 24.5g**, with a **6TOPS NPU**, natively supporting **visible light / infrared video tracking**, with a latency of < 100ms, suitable for drone narrowband transmission and lightweight payload requirements.

I. Core Hardware (RK3588)
CPU: Octa-core (4×A76@2.4GHz + 4×A55@1.8GHz)
NPU: 6TOPS, supports tracking models such as YOLO/SSD/DeepSORT
Process / Power Consumption: 8nm LP, entire board \*\*≤6W\*\*, suitable for airborne low-power scenarios
Dimensions / Weight: 45×45×5mm, 24.5g (40g with casing)
II. Video Tracking Capabilities (Drone-Specific)
Input: 2×MIPI‑CSI (up to 4K@30fps), analog / digital interfaces, compatible with visible light / infrared cameras
Algorithm: Self-developed KCF/DSST + deep learning, supports multi-target detection, locking, and continuous tracking
Accuracy / Metrics: Tracking accuracy \*\*≤1 pixel\*\*, minimum target 3×3 pixels, strong resistance to occlusion / interference
Latency: End-to-end \*\*<100ms\*\*, suitable for high-speed drone dynamic tracking
Output: Target miss distance (pixels / angle), tracking box, H.265 narrowband encoding (0.5–2Mbps)
III. Interfaces and Integration (Drone-Friendly)
Video: MIPI×2, CVBS, HDMI; 1–8 channels 1080P@30–60fps processing
Control: RS422, UART, USB2.0/3.0, Fast Ethernet, supports flight control linkage
Power Supply: DC 5V±10%, ≤6W, compatible with drone battery systems
System: Linux/ Android, supports ROS2, facilitating secondary development and navigation integration
IV. Typical Application Scenarios
Target Tracking: Ground vehicles, personnel, vessel locking, automatic follow-me flight
Security Inspection: Photovoltaic / power grid / border defense, abnormal target recognition + continuous tracking
Counter-Tracking: Infrared small target detection, night drone capture
Live Transmission: Narrowband HD video, low-latency video transmission + tracking overlay
