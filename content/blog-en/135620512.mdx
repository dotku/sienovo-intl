51.2 kS/s/ch, 2-Channel C Series Universal Analog Input Module
The NI‑9218 is designed for versatile measurements. It provides built-in support for accelerometers, powered sensors, full-bridge and voltage measurements, as well as quarter-bridge, half-bridge, 60V, and current measurements, utilizing adapters for specific measurement types. Each channel can be individually selected, allowing users to perform different measurements on each channel. The NI‑9218 is an ideal choice for developing versatile test systems required for automotive, off-highway, and data logging systems.
