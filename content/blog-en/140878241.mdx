Note:

*   The A500-3000 supports wide temperature operation. If users expand with Micro SD cards and M.2 SSDs, models with corresponding temperature specifications must be selected.
*   eMMC and Micro SD cards are Flash-based storage media. M.2 SSDs are Flash-based storage media with a SATA interface.

    Currently, NAND Flash is widely used in the industry. NAND Flash stores data by using floating gates to store electrons. After electrons repeatedly pass through the floating gate, the floating gate's ability to store electrons weakens, eventually leading to breakdown and inability to store data. This characteristic is a common issue with NAND Flash. Therefore, when using NAND Flash, it is important to fully evaluate the write data volume of the application to prevent premature wear-out leading to device failure.

*   For detailed information on M.2 SSD application scenarios, please refer to the 《[M.2 SSD Technical White Paper](https://e.huawei.com/cn/material/datacenter/server/1adb2a1784e7433895a7896f457da14a "M.2 SSD技术白皮书")》.
*   For detailed information on SD card application scenarios, please refer to the 《[SD Card Technical White Paper](https://e.huawei.com/cn/material/datacenter/server/cb33058036ca4da89bccfe2d7594f9e4 "SD卡技术白皮书")》.
