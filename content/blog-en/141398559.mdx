### 51.2 kS/s/ch, 2-Channel C-Series Universal Analog Input Module

The NI-9218 is designed for versatile measurements. With measurement-specific adapters, it provides built-in support for accelerometers, powered sensors, full-bridge and voltage measurements, as well as quarter-bridge, half-bridge, 60V, and current measurements. Each channel can be configured individually, allowing users to perform different types of measurements across channels. The NI-9218 is an ideal choice for developing universal test systems required in automotive, off-highway, and data logging applications.

Product Details

Maximum Number of Differential Analog Input Channels: 2

Conformal Coating: No

Sealed: Yes

Maximum Sampling Rate: 51.2 kS/s/channel

Electrical Signals Measured: Voltage, Current

Supported Sensor Types: IEPE, Powered, Accelerometer, Bridge

Analog Input Isolation: 60 VDC channel-to-channel isolation

Supported Hardware Platforms: C-Series
