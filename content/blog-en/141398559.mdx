## Domestic Alternative to NI: Full Domestic Alternative to NI-9218, 2-Channel C-Series Universal Analog Input Module

However, for users who prefer domestic alternatives, there are options available that can provide similar functionality.

This module is designed to provide built-in support for various types of measurements, including accelerometers, powered sensors, full-bridge and voltage measurements, as well as quarter-bridge, half-bridge, 60V, and current measurements.

### Analog Input Isolation

### Supported Hardware Platforms

The module is compatible with C-series hardware platforms.
