FPGA+USB+CCD Industrial Camera

Customer Application:

For precision object measurement and machining accuracy

Key Features:

1. Supports SONY low-noise CCD
2. Supports low latency, 1-frame delay from acquisition to display
3. Supports USB2.0 UVC driver / custom driver
4. Supports microscopic magnifier + supplementary lighting
5. Supports PC host software for adjusting CCD parameters
6. Supports PC SDK development kit for integration into professional measurement software
7. Supports PC host software for configuring and printing data
8. Supports object motion detection
9. Supports object contour recognition
