Board Overview:

ŃĆÉFMC\_XM155ŃĆæ FMC\_XM155 is a VITA57.1 standard-based FMC daughter card module that implements 2-channel, 14-bit, 500MSPS/1GSPS/1.25GSPS DC-coupled ADC synchronous acquisition.

This module complies with the VITA57.1 specification and can be directly used with an FPGA carrier card. The board's ADC device uses Analog Devices' AD9680 chip, which features two analog input channels and two JESD204B output data lane pairs, capable of sampling broadband analog signals up to 2GHz. The ADC front-end employs a wideband, low-noise, low-power fully differential amplifier with a gain-bandwidth product of up to 8GHz, offering excellent linear performance and up to 12dB gain from DC to 2GHz. This board is primarily designed for applications such as radar, broadband communications, millimeter-wave communications, and automated test equipment.
\[FMC\_XM121\] FMC\_XM121 is an FMC standard-compliant daughter card module that implements 2-channel, 14-bit, 1GSPS ADC synchronous acquisition and 2-channel, 16-bit, 2.5GSPS DAC synchronous playback.
This module complies with the VITA57.1 standard and can be directly used with an FPGA carrier card. The board's ADC device uses Analog Devices' AD9680 chip, which features two analog input channels and two JESD204B output data lane pairs, capable of sampling broadband analog signals up to 2GHz. The DAC device uses TI's DAC38J82 chip, which features a JESD204B interface, allowing for flexible configuration and programming, and enabling synchronization of multiple devices. This board is primarily designed for applications such as radar, broadband communications, millimeter-wave communications, and automated test equipment.
\[FMC\_XM139\] FMC\_XM139 is a 4-channel, 500MSPS/1GSPS, 14-bit AD acquisition daughter card module.
The board's ADC device uses Analog Devices' AD9680 chip, with a full-power -3dB analog input bandwidth of up to 2GHz. The ADC interfaces with the FPGA host via 8 channels of high-speed serial GTX transceivers. This board is primarily designed for applications such as communications and wireless infrastructure, radar, broadband communications, millimeter-wave communications, and automated test equipment.
