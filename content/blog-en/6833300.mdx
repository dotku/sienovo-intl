**Q1**: How do I switch from GUI mode to command-line mode in Linux under a virtual machine?

**A**: Run `init 3`.

**_NOTE_**: In Windows virtual machine environments, Ctrl+Alt+F1~F7 shortcuts are usually disabled.

**However, the following method consistently works (special thanks to Lele):**

**GUI/Terminal mode → Command-line mode: Ctrl+Alt+Space+F1**  
**Command-line mode → GUI/Terminal mode: Ctrl+Alt+Space+F7**

**Q2**: How to switch from command-line mode to GUI mode in Linux?

**A**: You can use `startx`, `xdm`, `gdm`, or `kdm`. If you cannot enter the X Window system, the configuration might be incorrect. In such cases, use the `Xconfigurator` command to reconfigure it!

logout — Log out  
reboot — Reboot  
startx — Start graphical interface  
gdm — Launch GNOME desktop environment  
kdm — Launch KDE login interface

**Q3**: Why does a program appear to have frozen?

**A**: Switch to root using `su root` and refresh.

**Q4**: What should I do if a self-written program enters an infinite loop? How do I terminate the process?

**A**: Press `Ctrl+C`
