This is a fully domesticated 8-channel fiber optic dual-FMC interface data preprocessing platform based on a domestic 28nm 7-series FPGA 7VX690T. The board utilizes a high-performance 7-series FPGA as a real-time processor, implementing 4-channel 10G SFP+ fiber optic and 1-channel QSFP+ communication interfaces, and providing 1-channel x8 PCIe data transmission. It features two independent 64-bit DDR3 SDRAM high-capacity memory banks. The board has two FMC+ expansion interfaces, allowing for the insertion of different daughter cards to achieve various interface functionalities, enabling rapid construction of data acquisition and processing platforms based on FMC interfaces and PCIe bus. The board also supports one RJ45 Gigabit Ethernet port for data interaction with a host PC via Ethernet. This board is a standard full-height PCIe card, compatible with standard PCIe industrial PCs and workstations, making it suitable for resource-intensive data centers or workstation data acquisition and processing scenarios.

#### Key Features

*   Standard PCI Express full-height card;
*   Compliant with PCI Express Gen3.0 specification, optional x1, x4, x8 configurations, with a bandwidth of 64Gbps;
*   Fiber Optic Transmission Performance: Supports 4-channel SFP+ 10G fiber, 10Gbps/lane line rate;
*   Fiber Optic Transmission Protocols: Supports Aurora and various other high-speed protocols;
*   Dynamic Storage Performance: Two 64-bit 4GByte DDR3 SDRAM banks, operating at 800MHz clock frequency, with a data rate of 1600 MT/s;
*   The board supports two FMC+ expansion interfaces;
*   The board supports multiple discrete I/O interfaces for GPIO expansion;
*   Onboard 256MByte BPI NorFlash for FPGA configuration;

#### Interface Characteristics

*   Front panel supports 4 SFP+ cages (supporting 10Gbps/lane);
*   Rear panel supports 1 QSFP+ cage (supporting 10Gbps/lane);
*   Rear panel supports 1 RJ45 Gigabit Ethernet interface;

#### Software Support

*   Optional integrated Board Support Package (BSP):
    1.  FPGA DDR4 interface test program;
    2.  4-channel 100G fiber optic interface program, supporting Aurora protocol;
    3.  PCIe interface test DEMO;
*   Provides drivers and Application Programming Interfaces (API):
    1.  Supports Windows 7 32-bit/64-bit operating systems;
    2.  Supports Linux operating systems;
