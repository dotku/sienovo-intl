This 4-channel QSFP28 100G optical fiber channel adapter, based on a PCIe bus architecture, features one PCIe Gen3x16 host interface and a total of four QSFP28 100G optical fiber interfaces. It enables real-time data acquisition, real-time buffering, and high-speed PCIe transmission for four QSFP28 100G optical fiber channels. The board utilizes Xilinx's high-performance Virtex UltraScale+ series FPGA XCVU3P as its main processor and includes two independent 80-bit DDR4 SDRAM large-capacity data buffers. The board also has one RJ45 Gigabit Ethernet port and several I/O control signals. This board is a standard full-height PCIe form factor, widely applicable in server-based optical fiber channel data acquisition, data transmission, and similar scenarios.

#### Key Features

*   Standard PCI Express full-height card;
*   Compliant with PCI Express Gen3.0 specification, with optional x1, x4, x8, or x16 modes, offering 128Gbps bandwidth;
*   Optical Fiber Transmission Performance: Supports 4x QSFP28 100G optical fibers, with 25Gbps/lane line rate;
*   Optical Fiber Transmission Protocols: Supports Aurora and various other high-speed protocols;
*   Dynamic Storage Performance: 2 sets of 80-bit 4GByte DDR4 SDRAM, operating at 1200MHz clock frequency, with a 2400M data rate;
*   Board supports 1 FMC expansion interface
*   Board supports multiple discrete I/O interfaces for GPIO expansion;
*   Onboard 2x 256MByte QSPI NorFlash for FPGA loading;

#### Interface Features

*   Front panel supports 3x QSFP28 Cages (supporting 25Gbps/lane);
*   Rear panel supports 1x QSFP28 Cage (supporting 25Gbps/lane);
*   Rear panel supports 1x RJ45 Gigabit Ethernet interface;

#### Software Support

*   Optional integrated Board Support Package (BSP):
    1.  FPGA DDR4 interface test program;
    2.  4-channel 100G optical fiber interface program, supporting Aurora protocol;
    3.  PCIe interface test DEMO;
*   Provides drivers and Application Programming Interface (API):
    1.  Supports Windows 7 32-bit/64-bit operating systems;
    2.  Supports Linux operating systems;

#### Application Scenarios

*   Radar system hardware-in-the-loop simulation
