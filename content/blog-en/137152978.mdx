Title: Fully Domestic 24-Bit 8-Channel Ethernet Data Acquisition Card with Customization Support, LabVIEW-Compatible 100K Sampling

Content:  
The XM702 is a high-speed Ethernet-based data acquisition (DAQ) card featuring 8 true differential input channels, 24-bit resolution, a maximum single-channel sampling rate of 100 kSPS, and a total synchronous sampling rate of 800 kSPS across all eight channels. It integrates precision pre-amplification with built-in hardware support for IEPE/ICP sensors. This product employs multiple high-precision 24-bit ADC units combined with our company's proprietary front-end differential amplification module developed over years of experience, delivering high speed, high resolution, high accuracy, ultra-low noise, excellent common-mode rejection, wide measurement range, and low temperature drift. These characteristics make it ideal for precision, high-speed data acquisition in various demanding applications.  

The LAN communication uses the TCP/IP protocol with an embedded packet-loss prevention algorithm, ensuring long-term stable data transmission without data loss.

The DAQ card is available in multiple variants: the USB version XM702H and the pluggable LAN connector version XM702. All components on the XM702 DAQ card are industrial-grade, and the device features full metal shielding, making it suitable for use in environments with strong industrial interference. It also offers excellent resistance to moisture, vibration, and electromagnetic interference.

Product Features

\* 24-bit high-resolution ADC with LAN (Ethernet) data transmission  
\* Input range: 0 to ±10 V  
\* High sampling rate: 800 kSPS aggregate; up to 100 kSPS per channel  
\* Sampling rate: 4 configurable digital I/O ports  
\* 2 configurable PWM outputs  
\* 1 configurable counter input  
\* 1 configurable DAC output  
\* High-precision reference voltage: 2.5 V ±0.05%  
\* Configurable external trigger for quantitative data acquisition  
\* Configurable external clock for synchronized sampling  
\* Supports combined external trigger and external clock synchronized acquisition  
\* Built-in 256 MB memory to handle temporary network disconnections  
\* Supports offline data storage via SD card  
\* Software-switchable IEPE/ICP mode support  
\* High-stability LAN module for reliable and stable data transmission  
\* Input impedance: 1 GΩ  
\* ADC nonlinearity THD: >100 dB  
\* ADC input common-mode rejection ratio (CMRR): >140 dB  
\* Temperature drift: <2 μV/°C  
\* Metal-shielded enclosure for strong anti-interference performance  
\* Entire system built with low-noise, industrial-grade components, low thermal drift, and high noise immunity  
\* TCP/IP data transmission with CRC error checking to prevent data loss

Product Applications

\* Weak signal measurement and acquisition  
\* Low-noise data acquisition  
\* Ethernet-based networked measurement systems  
\* High-impedance differential signal measurement

DAQ Card Software Description

The XM702 Ethernet 24-bit data acquisition card is compatible with common operating systems including Windows 11/10/7 and Linux. It provides dynamic link libraries (DLLs) for popular programming languages such as Python, VB, VC, C#, C++Builder, Delphi, LabVIEW/LabWindows, and MATLAB. The API functions are simple and user-friendly, facilitating rapid development of custom applications. Example code is provided for Python, VB, VC, C#, C++Builder, Delphi, LabVIEW/LabWindows, and MATLAB.
