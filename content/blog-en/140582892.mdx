**Current Status of AFC System Development**

An AFC system is an information system that integrates technologies such as computers, network communication, and automatic control. It is based on centralized computer control of automatic ticketing systems, automatic gate systems, central management systems, and more, to achieve automated and networked processes for rail transit ticketing, fare calculation, statistics, clearing, and overall management. Moreover, current domestic AFC systems incorporate biometric technologies like facial recognition and big data analytics, which can also enable self-service functions such as contactless ticketing and validation. This creates a more efficient, smarter, and safer urban rail transit travel experience, significantly improving urban rail transit operational efficiency.

In practical applications, with the architectural optimization and overall performance improvement of current domestic AFC systems, customers in the industry have put forward the following requirements for their main control computers:

High Energy Efficiency: Requires selecting main control computer products with stronger data processing capabilities to meet the AFC system's demands for large-volume data transmission and processing.

Rich Expansion: Requires selecting main control computer products with richer functional interfaces and stronger compatibility to meet the AFC system's requirements for deploying more AI functions.

More Secure and Reliable: Requires selecting main control computer products with stronger security and greater durability to enable secure data transmission and long-term stable operation for the AFC system.

**Suitable for AFC Systems**

This industrial PC is built on the Phytium E2000Q processor, adopting a fanless, wide-temperature heat dissipation structure. It is deeply adapted to Galaxy Kylin Embedded Operating System V10 and features high security, high energy efficiency, low power consumption, rich interfaces, and rugged durability, fully meeting the requirements for AFC system applications.

Key technical features are as follows:

Supports Phytium E2000Q processor, integrating Phytium high-efficiency processor cores 2\*FTC664 and 2\*FTC310, compatible with 64-bit ARMV8 instruction set and supporting both ARM64 and ARM32 execution modes. It supports single-precision, double-precision floating-point arithmetic instructions, ASIMD processing instructions, and hardware virtualization, capable of adapting to different workloads;

Supports PSPA security standard 1.0, domain-isolation-based security mechanisms, and integrates ROM as a trusted boot root. It is adapted to Galaxy Kylin Embedded Operating System V10 and features a developed watchdog function, providing customers with a safer and more stable system operating environment;

Onboard 4GB DDR4 memory, with support for up to 8GB RAM, and supports 1\*TF and 1\*M.2 NVMe for further storage expansion;

Supports 2\*adaptive Gigabit RJ45 network ports, onboard Wi-Fi, and 1\*MINI-PCIE slot for 4G/5G network module expansion;

Supports 2\*USB3.0 and 2\*USB2.0 interfaces for interconnection of multiple smart devices;

Supports 5\*COM ports (default RS232, two optional RS485) for high-speed data transmission over long/short distances;

Supports 1\*DP display interface, 1\*Headphone, and 2\*5W AMP audio interfaces for audio and video output, along with 1\*POWER on/off button for user convenience;

Fanless design, supporting power shutdown, clock shutdown, DVFS, core disabling, and frequency reduction operations to achieve lower power consumption;

Supports DC 12V power supply, meeting wide-temperature operation requirements from 0℃ to 60℃, and possessing rugged durability features such as shock resistance, dustproof, and moisture-proof, allowing stable operation under various conditions, including complex electromagnetic environments;

Compact size (176x155x51.5mm), supports desktop installation, suitable for multiple scenarios.
