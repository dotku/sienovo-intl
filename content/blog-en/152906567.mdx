**Solution Introduction**

In Hardware-in-the-Loop (HIL) testing, data replay can directly input real sensor data into in-vehicle hardware to test the performance of perception modules, evaluate the hardware's real-time processing capabilities, and verify the stability of hardware-software co-operation.

The autonomous driving data replay test solution is a comprehensive testing solution specifically designed for the autonomous driving domain. It provides full-process support, from road scene data acquisition and sensor data replay to function activation verification, ensuring efficient execution of the testing process, reliable system operation, and helping to improve the R&D efficiency of intelligent driving systems.![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/152906567/01_894edf55b3924043ba2fba7e291679cf.png)

**Solution Advantages**

■ Multi-sensor data compatibility: Supports various types of sensors (e.g., LiDAR, cameras, millimeter-wave radar, etc.) and data replay in different formats.

■ High-precision synchronization: Ensures time synchronization accuracy of less than 1ms between multi-channel camera data, and between other sensor data and bus data.

■ High-performance real-time system: Equipped with a high-performance real-time host, it achieves precise data replay, ensuring data real-time performance, as well as consistency in data content, frame rate, and timing.

■ Stable and reliable: Features a power protection mechanism, maintaining stable operation even after sudden power loss or restart; high-protection design ensures stable operation in various complex working conditions.
