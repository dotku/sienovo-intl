**Product Applications**

The USB-XM2432 is a data acquisition card based on the USB 2.0 bus. It can be directly connected to a computer's USB port to form data acquisition, waveform analysis, and processing systems in various fields such as laboratories and product quality inspection centers. It can also be used to build industrial production process monitoring systems. Its main applications include:

Electronic product quality inspection

Medical testing

High-precision synchronous signal acquisition

I/O Control

**Performance Features**

A/D Converter: 24-bit Σ-Δ ADC: AD7768

Low intrinsic noise, high channel isolation;

Differential input range: ±10V;

A/D Trigger Modes: Software trigger, external trigger;

Clock Source: Internal clock, external clock (selectable as synchronous clock / trigger clock);

Multi-channel: 32 channels

Coupling Method: DC;

Sampling frequency is software-configurable (Range: 1KHz～256KHz / 1KHz～200KHz / 1KHz～160KHz modes selectable);

Onboard large-capacity memory (4M words), supports continuous acquisition, enabling real-time monitoring;

Board ID recognition (0～15), supports multi-card operation;

Provides drivers and dynamic link libraries for WIN7/WIN10/XP;

Dimensions (excluding enclosure): 92(W) × 140(L) (mm);

**Technical Specifications**

   Operating Voltage: 12V±0.25V(0.7A)

   Operating Temperature: 0℃～70℃

   Storage Temperature: -10℃～85℃
