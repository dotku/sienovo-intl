This high-performance data preprocessing FMC carrier board is based on a PCIe bus architecture. The board features 1 FMC+ (HPC) interface, 1 PCIe x8 host interface, 1 RJ45 Gigabit Ethernet port, and 2 QSFP+ 40G optical fiber interfaces. It utilizes Xilinx's high-performance Kintex UltraScale series FPGA as a real-time processor to enable FMC interface data acquisition, processing, and backplane interface interconnection. The board includes two independent 72-bit DDR4 SDRAM large-capacity caches. By integrating various FMC daughter cards, this board can quickly establish server-based data acquisition, real-time processing, and high-performance storage platforms. It can be widely applied in scenarios such as radar and IF signal acquisition, and video image acquisition.

#### Technical Specifications

*   Onboard FPGA Real-time Processor: XC7K325T-2FFG900I;
*   Host Interface Specifications:
    1.  Complies with PXIe standard, supports PCI Express 2.0 specification;
    2.  Supports PCIe Gen2 x8 @ 5Gbps/lane;
    3.  PCIe Bidirectional DMA Transfer Bandwidth: 3.2GByte/s;
*   Other Interface Specifications:
    1.  1 SFP Optical Interface, supporting 2.5G/3.125Gbps/lane rates;
    2.  1 RJ45 Adaptive Gigabit Ethernet Port, can function as an EtherCAT master;
    3.  1 RJ45 Fast Ethernet Port, can function as an EtherCAT slave;
    4.  2 RS485 Interfaces, with RJ11 external connectors;
    5.  1 SDI Video Input, 1 SDI Video Output (optional function);
    6.  1 DVI Display Output Interface;
*   Dynamic Storage Performance:
    1.  Storage Bandwidth: 64-bit, DDR3 SDRAM, 500MHz operating clock;
    2.  Storage Capacity: Supports up to 4GByte DDR3 SDRAM;
*   Other Interface Performance:
    1.  Onboard 8-channel Opto-isolated Digital IO Interfaces;
    2.  Onboard 4-channel SATA Hard Drive Interfaces, for non-volatile storage;
    3.  Onboard 8-bit DIP Switch, 4 LED indicators;
*   Physical and Electrical Characteristics
    1.  Board Dimensions: 106.65 x 167.65mm
    2.  Board Power Supply: 1.7A max @ +12V (±5%)
    3.  Cooling Method: Air cooling
*   Environmental Characteristics
    1.  Operating Temperature: -40°C ~ +85°C;
    2.  Storage Temperature: -55°C ~ +125°C;
    3.  Operating Humidity: 5%~95%, non-condensing
