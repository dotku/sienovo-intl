It is a BMC daughter card. This daughter card provides 1 Gigabit Ethernet interface connected to a Gigabit switch chip, outputs 1 Fast Ethernet debug network interface, and outputs 2 + 8 (optional) IPMB (I2C) interfaces with isolation processing (I2C BUFFER). It supports the IPMI 2.0 standard communication protocol, with configurable communication rates of 100Kbps/400Kbps. It can control the power on/off of other blades via IPMI commands, and features fault reporting, power-on self-test (POST), and operational status self-test functions. It provides software interfaces to output information such as current, voltage, main chip temperature (including CPU, 40G switch chip), and firmware version (if available) to user software. It has the capability to collect and manage monitoring information for all blades in the entire system, and can provide independent system-wide status monitoring.
It adopts a PMC (VITA 42) architecture for easy ruggedization. Interaction between the AST2400 and the switch blade CPU, sensors, etc., is achieved via two standard PMC connectors. The board size is 75mm × 100mm.

Form Factor PMC Architecture

Board Dimensions 75mm × 100mm

System Configuration

Memory Onboard 128MB DDR3 memory

UART 2-channel UART interfaces
