A 1553B bus adapter card specifically developed for GJB5186 testing, supporting 4Mbps and 1Mbps bus speeds. This product provides 2 dual-redundant 1553B channels, 1 dedicated test channel, and 2 cable test channels. In addition to features such as programmable coupling methods, programmable bus signal amplitude, common-mode voltage injection, and bus signal waveform acquisition, the new generation TM53x card also adds a high-speed DAC to enrich and improve GJB5186 electrical performance testing, as well as cable continuity and cable short-circuit test functions.

The PXI/PXIe utilizes a self-developed, state-of-the-art 1553B protocol processor, featuring advanced BC, RT, and BM capabilities. When used with FlightPack 5186 (FP-5186) test software, it enables real-time analysis and simulation of 1553B bus data, as well as test applications for specific GJB5186 test subsets. The PXI/PXIe can be used in PXI and PXIe chassis to form dedicated avionics bus test instruments and build various application test systems.

BC/RT/BM Functions

BC

· Supports subframe periodic mode and message periodic mode
· The following processes are programmable:
     - Content and timing of major frames and subframes
     - Message inter-gap time
     - No response and late response times
     - Message retry conditions and count
· Optimized message scheduling for transmitting vector words, supporting BC automatic scheduling based on vector words. BC automatically determines the corresponding bits of the received vector word and automatically sends event messages that meet the conditions.
· Non-periodic messages can be inserted into the running message list.
· Message automatic jumping can be achieved based on data words or status words.
· Supports multiple data buffers
· Supports automatic bus switching function
· Supports protocol layer error injection

RT

· Optimized flag bit operation for transmitting vector words, supporting RT automatic clearing of corresponding vector word bits. After the RT transmits data to the BC, it automatically detects the vector word and clears the corresponding bit of the vector word.
· Can simultaneously support 31 RT simulations
· Response time and status word are programmable
· Supports all 1553B mode word codes
· Supports RT monitoring function
· Supports multiple data buffers
· Supports protocol layer error injection

BM

· Can monitor all 1553B bus information
· Supports 1553B bus waveform acquisition
· Supports monitoring filters, such as RT address, subaddress
