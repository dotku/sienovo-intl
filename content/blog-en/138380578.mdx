A fully domestically produced BMC daughter card capable of implementing ChMC/IPMC functionalities. The daughter card complies with the IPMI 1.5/2.0 protocol specification and also supports custom OEM commands. The daughter card can acquire carrier board environmental information, board attribute information, and board status information, including data such as temperature, voltage, and current; FRU, system version, and CPU model; memory utilization, CPU utilization, and network status. The daughter card can also control board power on/off and reset. Furthermore, it can dynamically configure IPMB addresses and IP addresses based on chassis number and slot number, and can provide customized web management interfaces for customers.
