## Three Steps to Quickly Set Up an Android Development Environment

Setting up an Android development environment can be a straightforward process if you follow the right steps. In this guide, we will walk you through the essential steps to download and configure the Android Development Tools (ADT) bundle, which includes a ready-to-use SDK and Eclipse IDE. This setup eliminates the need to navigate through various Google sites, making it convenient for developers.

### Step 1: Download the ADT Bundle

To get started, you need to download the ADT bundle, which contains everything you need to begin Android development. You can find the ADT bundle at the following link:

- [Download ADT Bundle](http://blog.csdn.net/LANG791534167/article/details/45061509)

Look for the file named `adt-bundle-windows-x86_64-20140321.zip`. This package is approximately 510.0 MB and includes an all-in-one Android development package suitable for versions 4.2 and possibly 4.4. 

**Note:** The included Eclipse IDE is 64-bit, so ensure your system meets this requirement.

### Step 2: Extract and Set Up the Environment

Once you have downloaded the `adt-bundle-windows-x86_64-20140321.zip` file, extract it to a preferred location on your computer. The extraction process will create a directory containing the Eclipse IDE and the necessary Android SDK components.

After extracting the files, navigate to the Eclipse folder and launch the `eclipse.exe` file to start the IDE. The first time you run Eclipse, it will prompt you to select a workspace. Choose a location where you want to store your projects, and click "OK."

### Step 3: Additional Environment Setup

If you plan to work with PHP in your Android projects, you may want to add PHP support to Eclipse. This can be done by following these steps:

1. In Eclipse, click on `Help` in the menu bar.
2. Select `Install New Software...`.
3. In the "Work with" field, enter the following update site URL:  
   [http://download.eclipse.org/tools/pdt/updates/3.4](http://download.eclipse.org/tools/pdt/updates/3.4)
4. Select the PHP Development Tools (PDT) from the list of available software and follow the prompts to complete the installation.

By adding PHP support, you can enhance your development capabilities within the Eclipse IDE, allowing for a more versatile development environment.

### Conclusion

With these three simple steps, you can quickly set up your Android development environment using the ADT bundle. This setup provides you with all the necessary tools to start developing Android applications without the hassle of visiting multiple sites. Happy coding!
