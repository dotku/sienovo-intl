### 1.33 GHz Dual-Core CPU, 1 GB DRAM, 4 GB Storage, -20 °C to 55 °C, Artix-7 75T FPGA, 8-Slot CompactRIO Controller

The cRIO-9056 is a rugged, deployable controller designed for data acquisition and control applications. Equipped with an Intel Atom dual-core processor, an Artix-7 FPGA, and eight C-series module slots, this controller runs the NI Linux Real-Time operating system and provides I/O access via the NI-DAQmx driver or LabVIEW FPGA module. Based on TSN networking, the controller delivers precise synchronization timing and deterministic communication, making it ideal for distributed measurement and control systems. It features one Gigabit Ethernet port, USB 3.1 host port, USB 2.0 device port, trigger lines, and a μSD card slot for local data storage.
