16-Channel Vibration + 2-Channel Rotational Speed (24-bit) High-Precision Terminal Acquisition Board

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/144676652/01_2cbbc7e0422547368a89ff3fe3707ac4.png)

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/144676652/02_2b12afcd50934282b0ce9769d41ba22f.png)

Features a domestic AG16KF256 FPGA + GigaDevice GD32F450.

The hardware architecture, based on a domestic ARM processor, while having weaker computational capabilities than a T3 processor, offers advantages such as lower cost and easier code maintenance.

Built-in algorithms can perform feature value calculations and FFT computation functions.

The FPGA remains responsible for front-end signal acquisition, filtering, and sampling, while the ARM primarily handles back-end feature value calculation, FFT computation, and soft-core protocol interfacing and data upload.
