The ME10 SoC is a full-stack AV over IP System-on-Chip (SoC) capable of transmitting HDMI 2.0 4K 4:4:4 video, audio, and control data over a 1Gb network. The ME10 SoC comes in a small 23 x 23 mm BGA package. Its key features are interoperability and superior performance.

IPMX is a collection of open standards and specifications for AV over IP, specifically developed for the professional AV market. The ME10 is ready for AIMS IPMX. Interoperability standards include device discovery and registration, USB and serial extension, resilient network performance, sub-frame latency, and JPEG-XS compression. IPMX is an industry-awaited standard.

Built on this interoperability platform, the ME10 provides a complete AV over IP solution for all your product lines. The included web stack features a RESTful configuration API, providing full access to ME10 functionalities. The customizable web interface is clean and intuitive. JPEG-XS compression delivers a truly immersive and transparent desktop experience. For manufacturers with high-volume professional AV applications, we offer ME10-based modules and development boards.

ME10 SoC Highlights

**Open Standards Give You Control**
The ME10 SoC is built for the AIMS IPMX open standard. IPMX is a collection of open standards from independent standards organizations, designed to provide a full-stack AV over IP solution that is interoperable among Pro AV industry manufacturers. Because these standards are independent, you retain control regardless of industry developments.

**4K60p 4:4:4 over 1G — Superior Video Quality on Low-Cost Networks**
Thanks to JPEG-XS based compression technology, the ME10 SoC delivers exceptional video quality over standard and low-cost 1 Gigabit networks, which are widely available in IT environments.

**Sub-Frame Latency — A Key Feature for KVM Applications**
The compression and timing technologies used in the ME10 SoC ensure end-to-end sub-frame latency. Human-to-mouse interaction is responsive, and gaming is immersive.

**Integrated Control — Customizable Web UI to Control SoC Behavior**
The ME10 SoC comes with a customizable Web UI and a RESTful API for controlling and configuring SoC behavior. This includes node registration and discovery, video and audio mode settings, and stream control.

**ME10 SoC Features**
• Low-latency synchronous transmission based on AIMS IPMX open standards, combining SMPTE ST 2110, AMWA NMOS, and JPEG-XS compression standards
• AES128 Encryption
• RESTful API and Lua Scripting Environment
• Web UI

**ME10 Applications**
1. HDMI Extenders
2. Video Walls
3. Remote Screen Switching, KVM
4. Digital Signage
5. Long-Distance Audiovisual Teaching/Presentations
