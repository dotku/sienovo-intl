### ******Functional Overview******

This module primarily features FPD-Link III / CML transceiver signals, HDMI / SDI / USB signals, and Gigabit Ethernet signals, supporting customer customization based on specified functions.

The current default function is FPD-Link III / CML conversion to HDMI / SDI / UVC signals.
