### LRM Airborne Recording Module

1 Introduction

1.1 Product Overview

The LRM Airborne Recording Module is an LRM data storage module built around a domestic VX690T FPGA processor, a domestic Phytium CPU FT2000/4, and a storage module. This module utilizes high-reliability LRM connectors and is centered on a processor group comprising a CPU and an FPGA. It offers various communication interfaces including Aurora, RapidIO, Gigabit Ethernet, fiber optic ports, RS422, CAN, and PCIe. Its capabilities include partitioned and classified storage of data files (Class A, B, and C), real-time recording, automatic indexing, playback, data loading/unloading, data encryption, data destruction, as well as Gigabit network routing and switching, configuration, time synchronization, visual media access, and self-testing. The total storage capacity can reach 2TB, with a real-time data recording and transfer bandwidth of no less than 1.6GB/s. The operating temperature range is -55℃ to +70℃.

The recording module is primarily designed for high-speed real-time recording and high-speed loading/unloading of system data, and for applications demanding high reliability in harsh environments.

2 Key Features and Performance

● Supports independent dual storage area mode: 2TB mass real-time data storage area 1 and 32GB file storage area 2;

● Supports real-time data recording function: Enables real-time recording of multiple data types via the Aurora interface, with a total read/write bandwidth of no less than 1.6GB/s;

● Supports data loading/unloading and transfer functions: Achievable via RapidIO, Gigabit Ethernet, fiber optic ports, RS422, and other interfaces;

● Supports AES-256 file data encryption and soft/hard key destruction functions;

● Aurora interface: 11 channels, supporting 3.125Gbps, 2.5Gbps, 1.25Gbps transmission rates;

● RapidIO interface: 2 channels, supporting 3.125Gbps, 2.5Gbps, 1.25Gbps transmission rates;

● Gigabit Ethernet ports: 5 channels, supporting full switching functionality and Network File System (NFS) functionality;

● Fiber optic ports: 4 transmit, 4 receive, MT type, supporting RapidIO communication protocol;

● Other interfaces: Supports 2 CAN channels, 1 RS422 channel, and other external communication interfaces;

● Power supply: +28V, power consumption less than 60W;

● Cooling method: Conduction;

● Operating temperature: -55℃～+70℃.
