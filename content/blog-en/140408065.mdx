A high-performance, high-reliability application system platform based on the 6U VPX bus architecture, supporting VPX and OpenVPX protocol standards. This system platform utilizes a high-performance 6-slot 6U VPX backplane with a highly optimized topology, capable of providing users with various high-speed serial bus interconnections. The platform supports standard VPX plug-in power supplies and high-power module power supplies, offering maximum design support for VPX system power. This platform supports rear plug-in card design, facilitating user IO expansion. This system platform can be widely used for real-time data acquisition, processing, transmission, and storage in harsh environments such as laboratories, field operations, and aircraft cabins, making it suitable for aerospace, aviation, marine, and other application scenarios.

*   Standard 4U height VPX system platform, supporting rackmount and desktop applications
*   Compliant with OpenVPX/VPX standards, conforming to VITA46/VITA65 specifications
*   6-slot 6U VPX backplane, with 5HP slot spacing
*   Supports standard 6U VPX plug-in power supplies or power modules
*   High-strength aluminum-magnesium alloy
*   Supports 6U horizontal plug-in cards, supports rear IO

#### Technical Specifications

*   Standard 4U VPX system platform, supporting rackmount and desktop applications;
*   Compliant with OpenVPX/VPX standards, conforming to VITA46/VITA65 specifications;
*   Chassis Specifications:
    1.  Chassis Material: High-strength aluminum-magnesium alloy;
    2.  Surface Treatment: Black powder coating;
    3.  Dimensions (W x H x D): 483.6mm x 177mm x 310mm;
    4.  Supports 6U horizontal plug-in cards, supports rear IO;
    5.  Cooling Method: Air cooling;
*   Backplane Specifications:
    1.  6-slot 6U VPX backplane, 5HP slot spacing;
    2.  Highly optimized backplane topology;
    3.  Slot0: Reserved Power Slot, Slot1: Controller Slot, Slot2~5: Peripheral Slots;
    4.  VPX J1: Slots 2-4 support x8@10Gbps Full Mesh between 3 slots;
    5.  VPX J2: Slot1 has 4 lanes of x4 PCIe@8Gbps connected to Slots 2-5 respectively;
    6.  VPX J3: Slots 2-4 support x8@10Gbps Full Mesh between 3 slots;
    7.  VPX J4: Slot5 supports 4 lanes of x4@10Gbps connected to Slots 2-4 respectively;
    8.  VPX J5: Slots 2-5 support 4-slot LVDS X8 full interconnection;
    9.  VPX J6: Slot1 has 2 channels of RS422/RS485 connected to Slot2 and Slot3;
*   Power Specifications:
    1.  System supports standard 6U VPX plug-in power supplies or power modules;
    2.  Max Power Supported: 1100W;
    3.  Input Voltage: AC 90~264V, 47~63Hz;
    4.  Output Voltage: +12V@max 60A (Ripple ≤120mV);
    5.  +5.0V@max 40A, +3.3V@max 10A (Ripple better than 1%Vpp), -12V@max 5A (Ripple ≤120mV);
*   Physical and Electrical Characteristics
    1.  Shock Performance: 3000 Bumps, 10G (16ms) half sine
    2.  Vibration Performance: 1.5G, 10~200Hz
    3.  Cooling Method: Air cooling
*   Environmental Characteristics
    1.  Operating Temperature: -40°C to +85°C, Storage Temperature: -55°C to +125°C;
    2.  Operating Humidity: 5%~95%, non-condensing

#### Application Areas

*   Software Defined Radio (SDR);
*   Radar and Baseband Signal Processing;
