Board Introduction:

This board is the Master Control Processing Unit (MCPU), primarily responsible for functions such as logic control, data processing, board communication management, system safety protection switching, and data storage.

Specifications:

Power Supply: DC 5V

CPU: MPC8377

Core Count: Single-core 32-bit

Clock Speed: 667MHz

MCU: MK60DN512VLL10

FPGA: XC6SLX16-2FT256I

Memory: DDR2 256Mb (for CPU)

PROM 16MB (for FPGA)

NVSRAM 512KB (shared by CPU and FPGA)

NORFLASH1 512Kb (shared by CPU and FPGA)

NORFLASH2 32Mb (shared by CPU and FPGA)

Communication Interfaces (Front Panel Side): USBx1 (USB-A); RS422x1 (D-SUB9); RS232x1 (D-SUB9); ETHx1 (M12-D)

Communication Interfaces (Backplane Side): CPCI Bus

Dimensions: 220mm x 100mm x 1.6mm

Weight: 0.160kg

Operating Temperature: -25℃ to 70℃
