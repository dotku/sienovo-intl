Product Overview  
The COMe-D2000 core module complies with the PICMG COM Express® Type-6 Rev2.1 standard and adheres to the COMe Basic form factor. It features a ruggedized design with onboard-mounted memory and SSD storage, enhancing reliability while maintaining a compact footprint. The COMe-D2000 integrates the domestically produced Phytium D2000 octa-core processor, which includes eight 64-bit high-performance cores clocked up to 2.3GHz. It comes with 16GB of onboard domestic DDR4 memory, 64GB of domestic SSD storage, an integrated high-performance GPU, and provides rich peripheral interfaces.

 ![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/136870755/01_90fa1736c3c6971ed3d5dc183ae7a704.png)

The COMe-D2000 is a fully domestic, high-performance, high-density, low-power, and highly reliable COMe computer module. It supports multiple mainstream operating systems and is widely applicable in defense, government, scientific research, telecommunications, and other fields. Users can develop computing systems, communication systems, office terminals, and more based on this module. Compared to developing systems from scratch, it offers significant advantages in flexibility, speed, and robustness, enabling true plug-and-play functionality.

This product adopts a **fully domestic design**, featuring self-developed hardware, firmware, and domestic operating systems, ensuring real security, autonomy, and controllability.

The default configuration includes a **pre-installed Galaxy Kylin Desktop V10 operating system**; other OS options must be confirmed prior to ordering.

Product Features  
- High integration in standard Basic size (95 × 125mm)  
- Compliant with COM Express Rev2.1, Type\-6  
- 100% domestic component content  
- Phytium D2000 octa-core processor, 2.0~2.3GHz  
- Onboard 16GB domestic DDR4 memory chips  
- 4 SATA 3.0 interfaces, including onboard 64GB BGA SSD  
- 2 Gigabit Ethernet ports supporting Gigabit optical-electrical COMBO interface  
- Rich PCIe interfaces: supports 2× PCIe3.0 x8 or 1× PCIe3.0 x16, 1× PCIe3.0 x4, and 8× PCIe3.0 x1  
- 4× USB3.0 (USB2.0 compatible); 4× USB2.0 with OC protection  
- 1× DP1.4, supporting maximum resolution of 3840×2160@60Hz  
- 1× HDMI, 1× VGA; HDMI and VGA operate in mirror mode for simultaneous display
