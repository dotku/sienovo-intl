### 24-Bit, 32-Channel, 51.2 kS/s, ±10 V PXI Analog Input Module

The PXIe‑4303 is a filtered input module suitable for voltage, thermocouple, and current measurements. This module allows you to select the appropriate digital filter for each channel, thereby improving filtering performance. You can also simultaneously use four different sample rates and trigger configurations, ensuring that you can measure all required sensors. Each channel of the module is equipped with an analog-to-digital converter (ADC), enabling the highest simultaneous sampling rate within the input range. The included NI-DAQmx driver simplifies hardware configuration and measurements.

Shenzhen Xinmai provides customized NI localization solutions.

Product Specifications
Maximum Sample Rate: 51.2 kS/s/channel
Analog Input Voltage Range: -0.1 V to 0.1 V, -10 V to 10 V
Analog Input Isolation: -
Maximum Differential Analog Input Channels: 32
Analog Input Resolution: 24 bits
Independent Low-Pass Filters: 1 kHz, 2 Hz, 2 kHz, 20 Hz, 200 Hz
Filtering: Anti-aliasing
