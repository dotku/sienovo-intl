    1. Basic Parameter Information

Basic Information

32 x 10/100/1000Base-T RJ45 ports, 8 x 1G/10Gig SFP+ ports

1 x RJ45 Console management port

1 x USB interface, 1 x reset button

Dimensions

482 mm (L) × 300 mm (W) × 44 mm (H) (including rack ears)

Installation Method

1U rack-mountable

Weight

≤3 kg

Operating Voltage

Dual redundant 220V AC power supply

Power Consumption

≤60 W

Operating Temperature

-40°C to +65°C, 5%–95% relative humidity, non-condensing

Storage Temperature

-40°C to +85°C, 5%–95% relative humidity, non-condensing

Relative Humidity

5%–95%, non-condensing

Protection Rating

Fan-cooled, metal enclosure

1.1 Software Specifications

**Main Software Features**

Software Features

Redundant Networking: Supports STP/RSTP/MSTP, EAPS, ERPS

Multicast Support: Supports IGMP Snooping V1/V2/V3

Layer 3 Features: Supports static routing, RIP, OSPF, VRRP, and other Layer 3 routing protocols

VLAN: Supports IEEE 802.1Q 4K VLAN, Access, Trunk, Hybrid modes

Link Aggregation: Supports manual aggregation, static/dynamic LACP protocol

Diagnostics & Maintenance: Supports port mirroring, Syslog, Ping command, etc.

Alarm Management: Supports one relay alarm output, RMON, TRAP alerts

Security Features: Supports DHCP Snooping, Option 82, 802.1X authentication

QoS: Supports CoS, DSCP, 8 queues, WRR, SP, WFQ scheduling modes, policy-based QoS

Management: Supports CLI, WEB, SNMP management, and integrated network management systems

Traffic Control: Supports ingress/egress port rate limiting, half-duplex backpressure and 802.3X full-duplex flow control, broadcast, multicast, and unknown unicast storm suppression
