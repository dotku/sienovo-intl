• 8-bit Dual-Channel Digitizer
• Maximum Sampling Rate up to 5 GS/s
• Analog Bandwidth up to 500 MHz
• Utilizes PCIe 3.0 x8 Interface
• Based on Xilinx Kintex UltraScale, XCKU040
• Offers Hardware, FPGA, and Software Customization Services

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/139551951/01_e510da767642fa9028369f9b83474c4e.png)

The high-speed data acquisition card is an 8-bit dual-channel digitizer, featuring a maximum sampling rate of up to 5 GS/s and an analog bandwidth of up to 500 MHz. The card's main control chip uses a Xilinx Kintex UltraScale, XCKU040, allowing for the implementation of custom real-time processing algorithms. It features large-capacity DDR4 memory on board for long-duration data acquisition. It adopts a PCIe 3.0 x8 interface, offering very high data transfer rates while occupying only one PCIe slot in the host PC. This provides high performance in a compact form factor, making it an ideal platform for many commercial and industrial applications. Furthermore, it offers hardware, FPGA, and software customization services according to customer requirements.
