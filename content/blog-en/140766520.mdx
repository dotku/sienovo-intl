This is a PC/104 embedded motherboard based on the RDC 3308. The RDC 3308 X86-architecture CPU is widely applicable in industrial fields, especially in harsh environments. The EB-9460 features pin-header breakouts for all I/O interfaces, enhancing convenience and stability during use. To improve usability, the EB-9460 is equipped with three USB ports, one of which supports hot-swapping under DOS.

This is an RDC-based PC/104 embedded motherboard. The RDC X86-architecture CPU is a high-performance, highly stable processor. The RDC processor utilizes a graphics card with built-in dedicated video memory, supporting VGA, TTL, and LVDS. The EB-9446 features four USB ports, one of which supports hot-swapping under DOS, facilitating user maintenance operations.
