The Smart Self-service Terminal Motherboard IoT-3288A has launched a self-checkout terminal, primarily targeting store checkout scenarios such as shopping malls, supermarkets, restaurants, and retail stores. After customers scan items they wish to purchase at the barcode scanning area, they can choose QR code payment, facial recognition payment, and membership/shopping card payment, significantly improving checkout efficiency, greatly reducing labor and hardware costs, and enhancing the customer shopping experience.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138803814/01_ed1eb82de1fc11cac8b8618e34d150d2.png)

**Functional Requirements**

01/ Product Identification and Settlement:
Customers can scan product barcodes or use RFID technology to identify products on the self-checkout machine. The system automatically reads product information and calculates prices.

02/ Aggregated Multiple Payment Methods:
Supports facial recognition payment, QR code payment, bank card payment, etc. Customers can choose the appropriate payment method according to their needs.

03/ Receipt Printing:
After completing payment, the self-checkout machine automatically prints transaction receipts and shopping lists for customers to verify items and retain proof of purchase.

**Motherboard Solution Advantages**

The motherboard solution for this project includes "Smart Self-service Terminal Motherboard IoT-3288A + Firmware + Kit". The IoT-3288A features high performance and low power consumption, with rich onboard expansion interfaces, multiple display interfaces, and communication serial ports. It is mature and stable, supporting 7*24 hour operation of terminal devices. Industry-mainstream peripherals support plug-and-play, allowing partners to quickly implement projects.

**Motherboard Configuration**

Core Configuration

CPU: RK3288 quad-core processor, main frequency up to 1.8GHz

Operating System: Android 7.1/11.0

Memory: Standard 2GB (4GB optional)

Storage: Standard 16GB (32GB/64GB optional)

Display Interfaces

HDMI Output: 1 port, supports 1080P@60Hz, 4kx2k@60Hz output

LVDS Output: 1 single/dual channel, can directly drive 50/60Hz LCD screens

eDP Output: Can directly drive eDP interface LCD screens of various resolutions

Supports dual-screen asynchronous display

MIPI CAM: Reserved MIPI CAM interface, supports up to 1300W pixels

Expansion Interfaces

USB: 7 x USB2.0 ports

Serial Ports: 1 x TTL, 2 x RS232, 1 x Bluetooth multiplexed serial port, 3 x GPIO, 1 x ADC

Storage Card: Supports SD card

Network Support

Wireless Network: Built-in WiFi; Bluetooth 4.2

Wired Network: Supports 10/100M adaptive Ethernet

Mobile Network: Supports PCI-E interface for 4G internet and calls

Audio Configuration

Amplifier: Supports left and right channel output, supports 8Ω/10W, 4Ω/20W

Headphone: Supports one 3-segment/4-segment headphone jack

Others

Dimensions: 146mm*100mm

RTC: Supports RTC real-time clock

System Upgrade: Supports local SD/USB upgrade

Power/Remote Control: 12V 3A/Supports remote control
