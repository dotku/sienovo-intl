This high-density server is a high-performance, high-compute edge computing product, equipped with the intelligent visual deep learning processor BM1688, and optionally configurable with 12 compute units. It supports intelligent analysis of 192 channels of HD video (hardware decoding) and hardware encoding of 120 channels of HD video. It supports multiple network ports, HDMI, USB, and storage expansion. It can flexibly handle high-concurrency business scenarios such as public security, transportation, smart parks, manufacturing, and warehousing. ****Key Parameters****

Product Name

High-Density Server

Product Model

XM3588-BM1688

AI Compute Power

192 TOPS (INT8) peak compute power

Encoding/Decoding Capability

Video Decoding: H.264&H.265: 16x1080P@30fps, supports 8192\*8192/8K/4K/1080P/720P/D1/CIF

Video Encoding: H.264&H.265: 10x1080P@30fps, supports 8K/4K/1080P/720P/D1/CIF

Number of Nodes

12 compute nodes (up to 96 ARM cores) + 1 control node

Processor

Compute Nodes: Sophon BM1688 core (A53\*8) 64-bit processor, max frequency 1.6GHz

Control Node: RK3588 octa-core (4×Cortex-A76+4×Cortex-A55) 64-bit processor, max frequency 2.4GHz

Memory

8GB LPDDR4 × 12 (compute nodes), Storage 32GB eMMC × 8 (compute nodes), 2.5-inch SATA3.0 drive bays × 2

Network

Gigabit Ethernet RJ45 ports × 5 (1 control node port, 1 general-purpose port), SFP+ × 2, 4G/LTE/5G network (optional)

Display

HDMI 2.0, 4K@60Hz (main processor core board display)

USB

1 × USB3.0 HOST, 1 × USB2.0 HOST, 1 × Type-C (for BMC unit debugging)

Fans

5 high-speed cooling fans

Power Supply

300W AC power supply (Input: 100V AC～240V AC)

Linux BMC

Integrated BMC management system, providing a web-based management interface. The BMC management system supports secondary development.

Deep Learning Frameworks

TensorFlow / PyTorch / Paddle / Caffe / ONNX / MXNet / DarkNet

Dimensions

Standard 1U rackmount server: 490mm x 390mm x 44.4mm

Operating Temperature

0ºC - 50ºC, Operating Humidity 10%～90%RH (non-condensing)
