## Edge Computing Box

Dual-core A55 | 2.5 Tops Computing Power

● 2.5 Tops@INT8 computing power, supports INT8/INT4/FP16 multi-precision mixed quantization.

● 4 channels or more of 1080p@30fps video encoding and decoding, IVE module independently provides image basic operator acceleration.

● Supports Caffe, ONNX/PyTorch deep learning frameworks, provides basic AI algorithm examples such as ResNet50 and YOLOv5.

● MindStudio graphical development environment, integrates model conversion and quantization, and network performance optimization tools to improve application development efficiency.

● Supports Docker containerized deployment and Ubuntu 20.04 system environment, facilitating cluster management and application expansion.

● Supports multiple network communication methods (dual Ethernet, WiFi/4G networking, etc.), providing convenience for remote management and control of edge computing nodes.

● Expandable with SATA hard drive or TF flash card to meet large-capacity data storage requirements for video image caching and application operation and maintenance logs.

● Provides serial communication control (RS232/RS485, USB), reserves visual monitoring and audio-visual alarm interfaces (HDMI, Audio).

Edge computing intelligent workstation, equipped with a dual-core Cortex-A55 architecture AI processor; built-in efficient neural network inference engine, providing 2.5 Tops@NPU computing power; supports multi-channel video image recognition hardware acceleration. High performance, low cost, stable and easy to develop, multi-point wiring, networked management and control; it can meet the application requirements for deploying lightweight intelligent video analysis at the edge; it is a new generation upgrade solution for economical intelligent workstation products, widely applied in smart city scenarios such as smart security, smart parks, and smart urban management.
