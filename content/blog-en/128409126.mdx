The AX58100 is a cost-effective EtherCAT slave controller that is **certified by Beckhoff Automation** and **integrates two Fast Ethernet physical interfaces** that **support 100Mbps full-duplex and HP Auto-MDIX**. The AX58100 supports all EtherCAT systems, including standard EtherCAT devices (e.g., CoE, FoE, VoE, etc.). The AX58100 provides a highly cost-effective EtherCAT solution for applications such as drive/motor/data I/O control, digital-to-analog conversion, analog-to-digital conversion, variable frequency controllers, and image data acquisition!

The AX58100 offers **a three-channel PWM controller or a stepper controller**, along with an additional (ABZ)/Hall encoder interface for closed-loop management of control applications such as drives/motors; it provides an SPI **master** to manage DAC/ADC converter control applications and image data acquisition applications; and it also includes an I/O watchdog and an **emergency stop input** for industrial automation safety applications.

The AX58100 offers two PDI (Process Data Interfaces): **SPI slave and Local bus**, making it easier for system designers to implement standard EtherCAT protocols for traditional non-EtherCAT MCU/DSP industrial platforms. The AX58100 provides a configurable SPI bus to increase the bandwidth of the SPI slave.

**Advantages:**

Two built-in PHY Ethernet ports + one MII interface

Built-in SPI master interface (supports 8 SPI channels)

Built-in PWM or stepper/direction controller, ABZ/Hall encoder

Built-in 32 digital I/O control signals

Built-in 2 Process Data Interface (PDI) interfaces (SPI Slave/Local Bus)

Supports emergency stop input function
