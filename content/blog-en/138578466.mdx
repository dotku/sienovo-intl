The ARINC664 dual-port terminal network card is a high-performance ARINC664 network interface card (NIC) that supports ARINC664 network simulation, testing, and data analysis functions. It features two ARINC664 ports with RJ45 connectors, which can be used as two independent ports or as a pair of redundant ports. The card integrates an IRIG-B system clock, providing multi-node clock synchronization capabilities, facilitating its use in multi-node simulation systems. The ARINC664 communication specification fully complies with the ARINC664 Avionic Network PART7 protocol requirements.
