The triage and queuing display screens, powered by the XM-3288A smart terminal motherboard, are designed to show patient queue information and doctor work status. They offer advantages such as patient convenience, enhanced hospital image, improved management efficiency, better doctor-patient relationships, and cost savings. This system effectively addresses issues like disorganized patient visits, cumbersome consultation processes, noisy environments, and unbalanced doctor workloads, thereby significantly improving hospital service quality and efficiency.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/139091993/01_0d6b1307bc0faaafb1754eefbe16dad4.jpeg)

**Motherboard Solution Advantages**

The core motherboard of this solution is the XM-3288A smart terminal motherboard. It is a high-performance, graphically powerful, feature-rich, and mature and stable motherboard, enabling terminal devices to stably and smoothly implement functions such as triage information display and consultation broadcast reminders.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/139091993/02_705f41bf0b38a1e12bebf4d36dd70b2a.png)

**Core Configuration**

CPU: RK3288 quad-core processor, with a main frequency up to 1.8GHz

Operating System: Android 7.1/11.0

Memory: Standard 2GB (4GB optional)

Storage: Standard 16GB (32GB/64GB optional)

**Display Interfaces**

HDMI Output: 1 port, supports 1080P@60Hz, 4kx2k@60Hz output

LVDS Output: 1 single/dual channel, can directly drive 50/60Hz LCD screens

eDP Output: Can directly drive eDP interface LCD screens of various resolutions

Supports dual-screen asynchronous display

MIPI CAM: Reserved MIPI CAM interface, supports up to 13-megapixel

**Expansion Interfaces**

USB: 7 x USB2.0 ports

Serial Ports: 1 x TTL, 2 x RS232, 1 x Bluetooth multiplexed serial port, 3 x GPIO, 1 x ADC

Storage Card: Supports SD card

**Network Support**

Wireless Network: Built-in WiFi; Bluetooth 4.2

Wired Network: Supports 10/100M adaptive Ethernet

Mobile Network: Supports PCI-E interface 4G internet access and calls

**Audio Configuration**

Amplifier: Supports left and right channel output, supports 8R/10W, 4R/20W

Headphone: Supports one three/four-segment headphone jack

**Other**

Dimensions: 146mm*100mm

RTC: Supports RTC real-time clock

System Upgrade: Supports local SD/USB upgrade

Power/Remote Control: 12V 3A / Supports remote control

####

**Application Examples**

This project has been successfully implemented in many locations across the country, including university affiliated hospitals, regional capital hospitals, military hospitals, and community health centers.
