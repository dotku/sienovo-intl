The smart waste recycling project launched by the all-scenario smart terminal motherboard AIoT-3588A aims to address the increasingly prominent waste disposal issues in the process of urbanization. Smart waste sorting bins feature multiple functions such as touchscreen operation, automatic weighing, categorized disposal, and electronic voice prompts. Residents can deposit waste through various methods, including sorting points cards, mobile phone QR code scanning, and facial recognition, enabling more efficient waste sorting, improving sorting accuracy, and effectively saving human resources.

#### Project Feature Requirements

1. Supports card/QR code/facial recognition for waste disposal
2. Supports waste weighing
3. Waste bin overflow alert
4. Supports voice prompts
5. Smart anti-pinch (hand protection)
6. Advertising screen displays waste sorting promotional ads, community service information, etc.
7. Surveillance camera

Motherboard Solution Advantages

The core motherboard for the smart waste recycling project is Xinmai's all-scenario smart terminal motherboard AIoT-3588A. This motherboard is their most powerful flagship high-end motherboard, offering significant advantages:

_**01/Powerful Computing Core**_

The AIoT-3588A is equipped with the RK3588 core, possessing powerful processor performance and AI computing power. It fully supports the application of technologies such as AI vision and big data models in the waste recognition process. It intelligently identifies waste types and controls the opening of corresponding bin doors, guiding correct disposal behavior and helping users easily tackle sorting challenges.

_**02/Powerful Expansion Capability**_

It features rich expansion interfaces, supporting USB/TTL/RS232/RS485/GPIO/ADC and other interfaces. It can expand to peripheral modules such as facial recognition cameras, barcode scanners, ID card readers, and various sensors. This facilitates resident identity verification and supervision of sorting behavior.

_**03/Rich Network Support**_

Supports Gigabit Ethernet, integrates high-speed wireless communication technologies such as 4G/5G (optional), WiFi, and Bluetooth, fully meeting the device networking requirements for various application scenarios.

**Motherboard Parameter Configuration**

**Core Configuration**

**CPU: RK3588 octa-core processor, main frequency up to 2.4GHz, 6 TOPS AI computing power**

**Operating System: Android 12.0 (supports Linux/OpenHarmony)**

**Memory: Standard 4GB (8GB optional)**

**Storage: Standard 32GB (64GB/128GB optional)**

**Display Interfaces**

**HDMI Output: 1 channel, standard Type-A female connector, supports resolutions up to 7680x4320@60Hz**

**LVDS Output: 1 channel, supports single/dual 8-bit, can directly drive 50/60Hz LCD screens**

**MIPI Output: 1 channel, can drive MIPI LCD screens of various resolutions**

**eDP Output: 1 channel, can directly drive eDP interface LCD screens of various resolutions (customization required)**

**eDPx Output: 1 channel, can directly drive 4K resolution eDPx interface LCD screens**

**Supports four-screen heterogeneous display**

**HDMI Input: 1 channel, standard Type-A female connector, supports 4K@60Hz resolution input**

**Camera**

**Supports USB single/dual-lens cameras and MIPI cameras. MIPI interface supports 1x2-lane/2x2-lane@2.5Gbps/lane, with a maximum support of 5M pixels (1x2-lane design limitation).**

**Peripheral Expansion**

**USB: 1 channel USB 3.0 (Type-C), compatible with DP output support**

           **8 channels USB 2.0**

**Serial Ports: 2 channels RS232, 2 channels TTL, 1 channel RS485**

**TP Interface: 1, connectable to IIC interface TP screens or peripherals**

**Storage: Supports U disk, TF card, M.2 SSD, SATA expanded storage**

**Network Support**

**Wireless Network: Built-in WiFi; Bluetooth 5.0**

**Wired Network: Supports 10/100/1000M adaptive Ethernet**

**Mobile Network:**

**1. Supports 4G internet via PCI-E interface;**

**2. Supports 5G internet via M.2 5G interface (optional)**
