Product Overview:

This is a fiber optic board that converts LVDS signals into optical signals. Its external communication interface enables high-speed communication with the main control board via the fiber optic board.

Performance Specifications:

Power Supply: DC 5V

CPLD: XC2C256-7TQ144I

Communication Interfaces (Backplane Side): RS485 x 1; I2C x 1

Communication Interfaces (Front Panel Side): SFP x 12

Dimensions: 160mm x 233.5mm x 1.6mm

Weight: 0.145kg

Operating Temperature: -25℃ to 70℃
