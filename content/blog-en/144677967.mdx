8-Channel Vibration/Voltage 5 Msps High-Speed Data Acquisition Board

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/144677967/01_b70aec4b57604f7786661b58592e2eee.png)

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/144677967/02_3318ff6470cd4001bf9d2269d59ae9e6.png)

Utilizing an XC7A35T-2FGGI model FPGA, this is a multi-channel dynamic signal acquisition device with a maximum sampling rate of 5 Msps, primarily applied in scenarios requiring high sampling rates.

It supports an external synchronization clock device, compatible with GPS synchronization, enabling synchronized data acquisition across multiple boards. Acquired data can be stored on an external SD card and a built-in SATA interface hard drive.
