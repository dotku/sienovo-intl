![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/139566662/01_d66bdebfcee33d076b46fcc13114ecdd.png)

is an executable test execution management software used to organize, control, and execute your automated prototyping, validation, or manufacturing test systems. Archon is the core software of a domestic test operating system.

****Software Interface****

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/139566662/03_cceab48b378e2c1543ebd048f2a02ddc.png)

Automatically discovers instruments and provides simple control and management; rich system APIs, including connection management APIs and test result judgment APIs, improve development efficiency; graphical drag-and-drop operations and auto-completion reduce development difficulty; test cases are saved as CSV files and Python scripts for easy editing and version management.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/139566662/04_2fe2b14d16ed2c5d314926af1e1f1241.png)

Supports distributed deployment, sequential execution, and parallel execution modes, flexibly adapting to various production test scenarios. Test data results can either be stored in a database for visual analysis or exported and saved locally using custom templates.
