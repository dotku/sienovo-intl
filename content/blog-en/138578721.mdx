The FC node card primarily uses LC or MPO fiber optic interface forms. It can be used as 4/2 independent ports or as 2 pairs/1 pair of redundant ports, supporting communication rates of 1.0625Gbps, 2.125Gbps, 4.25Gbps, and 8.5Gbps. The node card fully complies with FC fiber optic communication protocol specifications such as FC-LS, FC-FS, FC-AE-ASM, and FC-AE-1553B.

**Key Features:**

Ø Supports 4/2 independent or redundant FC ports for point-to-point and switched communication modes

Ø Supports 1G/2G/4G/8G communication rates

Ø Complies with FC-PC, FC-FS protocol standards

Ø Supports 256 non-data blocks and 16 data blocks

Ø Supports implicit and explicit login

Ø Complies with Class 3 service

Ø FC frame length supports 36～2148 bytes;

Ø Supports Class 3 services, supports multicast, broadcast functions;

Ø Supports at least 8 Buffer-to-Buffer credits, credit value configurable;

Ø Supports up to 256 receive channels and 256 transmit channels;

Ø Supports primitive-level clock synchronization, resolution 100ns;

Ø 4Gb high-speed data receive buffer, transmit buffer, data buffering time 1s

Ø Supports hardware encryption function

Ø Supports error correction for data frame header and payload, supports port redundancy switching

Ø Physical layer data transmission bit error rate less than 10^-12

Ø Supports various User Upper Layer Protocols (ULPs), including FC-AE-ASM, FC-AE-RDMA, FC-AE-1553, AC-AV, ARINC818

Ø Supports Windows 7, VxWorks 6.9, and Linux operating systems
