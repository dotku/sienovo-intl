FPGA solution: HDMI IN receives the original HDMI signal, HDMI OUT outputs the HDMI signal with overlaid characters/images.

Customer Application: XBOX game console monetization

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138607817/01_27a678c71236a03bf26852007ceb149e.png)

Key Features:

1. Supports multi-resolution format display
2. Supports OSD overlaying multiple layers
3. Supports flexible font size configuration
4. Supports flexible font format configuration
5. Zero latency, instant output upon input
6. Supports independent audio signal output
7. Customizable functions, extensible applications
