### 102.4 kS/s, 24-bit, 8-Channel PXI Strain/Bridge Input Module

The PXIe-4331 high-speed input module provides integrated signal conditioning and data acquisition for dynamic strain, load, and pressure measurements. The module features higher accuracy, high data throughput, and synchronization capabilities, making it an ideal choice for high-density measurement systems.

To eliminate noise, each channel of the PXIe-4331 provides anti-aliasing and digital filters. Each channel also features an independently programmable excitation voltage. Additionally, the PXIe-4331 offers remote sensing, internal bridge completion, and shunt calibration options for each channel.
