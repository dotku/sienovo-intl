**Board Overview**
The FMC_XM131 is a 4-channel, 12.6GSPS sampling rate, 16-bit DA playback FMC daughter card module. This board adheres to the FMC+ standard, complying with the VITA57.4 specification. It can serve as an ideal I/O module for coupling to an FPGA frontend. Its 16-lane JESD204B interface connects to the FPGA's high-speed serial port GTH via the FMC+ connector. The board supports an onboard programmable sampling clock and an external reference clock. Multiple boards can achieve output synchronization through trigger (input/output) signals. The board's four analog signal outputs are routed through 50Ω characteristic impedance SSMC RF connectors and utilize AC transformer coupling. This board can be widely applied in scenarios such as radar simulators, communication and navigation simulators, and jammers.

**Technical Specifications**
*   Performance Indicators:
    *   Supports 4-channel RF signal output;
    *   Utilizes AC transformer coupling;
    *   Output signal power range: ≤0dBm;
    *   AD9172: Maximum complex input data rate 1.54GSPS;
    *   AD9176: Maximum complex input data rate 3.08GSPS;
    *   Each channel has 1 independent NCO;
    *   Supports 8-lane, 15.4Gbps JESD204B interface;
    *   48-bit NCO, supporting frequency synthesis up to 6GHz;
*   FMC Interface Specifications:
    *   Standard FMC+ daughter card, complying with VITA57.4 specification;
    *   Board supports 1 EEPROM;
    *   FMC connector model: ASP-184330-01, HPC interface;
    *   Board powered by +12V, typical total board power consumption 6W;
*   Clock Distribution:
    *   Supports external clock mode;
    *   Onboard 1 high-precision clock chip HMC7044;
*   Other Functions:
    *   Supports external trigger;
    *   Onboard status indicator;
*   Physical and Electrical Characteristics:
    *   Board dimensions: 84.1x69mm
    *   Board power supply: 3Amax@+12V (±5%)
    *   Cooling method: Natural air cooling or metal conduction cooling
*   Environmental Characteristics:
    *   Operating temperature: -40°C ~ +85°C;
    *   Storage temperature: -55°C ~ +125°C;
    *   Operating humidity: 5% ~ 95%, non-condensing
