## CPU Module

This is an SOM Express module based on the RDC 3306. The RDC 3306 X86-architecture CPU is a high-performance, highly stable processor.
It is a flexible and compact mainboard (measuring 91.8mm × 68.6mm) that can be flexibly integrated into a user's baseboard, saving development costs. The module's connectors use a pin header design to ensure stability during use.

## CPU Module

This is a Baytrail-based SOM Express module. It uses an SOM Express form factor (dimensions: 105mm × 69mm).
The pin header design mitigates the disadvantage of unstable module operation. Furthermore, this module integrates high, mid, and low-tier CPUs under the Baytrail architecture
(J1900, N2930, N2807), offering users more choices.
