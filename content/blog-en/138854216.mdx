The A429 is a standard PXI/PXIe x1 multi-protocol bus adapter card. This product supports up to 36 ARINC 429 channels, or 32 ARINC 429 channels plus 4 ARINC 717 channels. Each ARINC 429 and ARINC 717 channel can be software-configured for reception or transmission, meeting the communication, testing, and data analysis application requirements for ARINC 429 and ARINC 717 buses.

Each ARINC 429 channel of this product supports standard ARINC 429 data rates and custom rates, with custom data rates reaching up to 300kbps. It supports functions such as bus signal acquisition, error injection, IRIG-B DC encoding/decoding, IRIG-B AC hardware decoding, automatic data rate detection, data playback, and Label or SDI message filtering.

Transmit channels support periodic and aperiodic messages, with period size and message interval time being software programmable. Receive channels simultaneously support three message caching modes: queue, sampling, and multi-channel buffering. Interrupt queues are used to achieve software-hardware interaction, ensuring lossless message buffering when all channels operate simultaneously.

Each ARINC 717 channel of this product supports polling and/or interrupt operations. Transmit channel rates support multiple options: 64wps, 128wps, 256wps, 512wps, 1024wps. It supports standard ARINC 717 and non-standard ARINC 717 data transmission, with word length and inter-word gap time being software programmable. Receive channels add status tags and time tags to each received word.

This product features a high-speed host interface, onboard 512MB large-capacity DDR cache, and self-developed protocol processing IPCORE. These ensure real-time data transmission and reception for multiple ARINC 429 and ARINC 717 channels. Bundled with FlightPack A429 software, it provides a complete hardware and software solution for ARINC 429 and ARINC 717 bus development and testing.
