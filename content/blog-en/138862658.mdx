It is a standard full-height, half-length PCI specification, high-performance ARINC664 network interface card. This product supports application requirements such as ARINC664 network simulation, testing, and data analysis; it supports 2 ARINC664 ports, using RJ45 connectors, with each port having up to 64K buffer. The two ports can be used as two independent ports or as redundant ports. The product integrates an IRIG-B system clock, providing multi-node clock synchronization, which facilitates its use in multi-node simulation systems.

Transmission supports configuration of up to 128 transmit Virtual Links (VLs) and 1024 transmit ports; reception supports configuration of up to 512 receive VLs and 4096 receive ports. It not only supports Queuing, Sampling, and SAP ports, but also SNMP and 615A loading protocols, among others.

As a standard full-height, half-length PCI card, it can be conveniently applied in various PCs and industrial computers with PCI slots. This product supports high-speed forwarding ports, reducing transmission delay and ensuring real-time communication during simulation and testing. Coupled with FlightPack software, it can achieve simulation, testing, and data analysis.
