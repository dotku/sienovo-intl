1.  **Product Introduction**
    1.  

The XM3568-EP is an edge computing gateway (hereinafter referred to as: gateway) integrating multiple interfaces such as serial ports, CAN interfaces, digital inputs, digital outputs, and AI voltage acquisition. It can also be additionally expanded with 4G modules, SATA hard drives, and other plugins. The system is based on an arm64 architecture, features a highly reliable industrial-grade processing core, boasts high-performance 4-core concurrent processing capability, and runs the Linux operating system.

1.  1.  Product Features

Powerful Performance

The gateway has four high-performance cores with a 1.5GHz main frequency and a built-in standard Linux operating system. It can be used not only for data acquisition but also as a conventional server. Users can freely download and install application software such as MySQL, Java, Chrome, and programming language SDKs according to their needs.

High/Low Temperature Resistance and Electromagnetic Compatibility

The gateway complies with national standards such as GB/T2423, GB/T9254, and GB/T17626, meeting various high/low temperature, electromagnetic interference, and magnetic field immunity test levels. It can operate stably in ambient temperatures ranging from -20℃ to +70℃ and in various complex electromagnetic environments, making it suitable for diverse application scenarios such as high-speed, outdoor parks, and industrial computer rooms.

Rich Interfaces

The gateway integrates multiple acquisition and control interfaces, including RS485, digital input interfaces, and digital output interfaces. These rich interfaces allow a single gateway to connect to various industrial devices without the need for external expansion of other acquisition modules.

4G Communication

The gateway has a built-in 4G communication module and uses DHCP by default, requiring no configuration tools. The gateway automatically connects to the network upon power-on, offering convenience, quick setup, and significant labor cost savings.

Video Output

The gateway features an HDMI high-definition multimedia interface and provides several common video resolutions. The default resolution is 1920*1080P@60Hz. Simply plug it into a screen to display high-definition images. It automatically adapts to mouse, keyboard, and touch screen, providing users with the same experience as a desktop PC.
