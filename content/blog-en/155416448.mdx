ARM Platform:

Ultra-compact EtherCAT Motion Controller, developed based on ARM architecture, configured with a dual-core ARM processor, featuring 1 x LAN, 1 x EtherCAT, 1 x MicroSD, and Slice IO expansion.

*   ARM-based EtherCAT Motion Controller supporting up to 32 axes
*   Dual-core ARM processor
*   1 x Ethernet LAN, 1 x EtherCAT, 1 x MicroSD
*   Supports Modbus TCP, TCP/IP communication
*   Provides a low-code software development environment - MotionNavi Designer
*   Provides Advantech BASIC programming language for multi-task development
*   Retain memory (FRAM)
*   Supports PTP, linear and circular interpolation, path tables

x86 Platform:

Ultra-compact EtherCAT Motion Controller, developed based on PC architecture, configured with an Intel Atom processor, featuring 64GB eMMC, 2 x LAN, 2 x USB, and Slice IO expansion. PC-based EtherCAT Motion Controller supporting up to 32 axes. Intel® Atom® Quad-core x6413E Processor. 4 GB DDR4 RAM. 64GB eMMC storage. Equipped with 2 x GbE, 2 x USB, 1 x HDMI. M.2 B expansion slot for flexible expansion. Built-in TPM2.0 for hardware-based security considerations. Supports PTP, linear and circular interpolation, path tables. Supports C/C++, VS.Net, LabVIEW programming languages.
