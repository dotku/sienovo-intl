This 4-channel 10G Fibre Channel adapter, based on PCIe bus architecture, features four SFP+ 10 Gigabit optical interfaces and an x8 PCIe host interface. It includes one group of 64-bit DDR3 SDRAM as high-speed cache, enabling high-speed acquisition, real-time recording, and broadband playback of 4-channel optical network data.
The card also features four SATA interfaces, enabling high-speed storage of optical fiber data. It supports the SATA 3.0 standard, with a maximum storage bandwidth of 1.6 GByte/s.
This card is a standard full-height, half-length PCIe form factor, suitable for current mainstream servers or workstations. It can be widely applied in data centers, server acceleration computing, and other scenarios, and is also suitable for hardware-in-the-loop simulation systems.

#### Main Features

*   Standard PCI Express half-length card;
*   Compliant with PCI Express Gen2.0 specification, with optional x1, x4, or x8 modes, 40Gbps bandwidth;
*   Optical Transmission Performance: Supports 4-channel SFP+ 10 Gigabit optical fiber, 10Gbps/lane line rate;
*   Optical Transmission Protocols: Supports multiple high-speed protocols such as Aurora and RapidIO;
*   Dynamic Storage Performance: One group of 64-bit 2GByte DDR3 SDRAM, theoretical bandwidth 12.8GByte/s, efficiency up to 90%;
*   SATA Storage Performance: Supports x4 SATA 3.0;
*   The card features 4-channel opto-isolated digital discrete inputs and 4-channel opto-isolated digital discrete outputs;
*   Onboard 1 PMCIO interface for GPIO expansion;
*   Onboard 1x 128MByte BPI NorFlash for FPGA loading;

#### Interface Features

*   Front panel supports 4-channel SFP+ Cages (max 10Gbps/lane);
*   4-channel SATA interfaces on board;
*   1 PMCIO interface on board;
