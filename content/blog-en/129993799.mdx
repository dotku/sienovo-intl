The AVS Intelligent Vision Recognition System's main function is positioning. By using the independently developed PC software ADTvision, users can quickly build their own vision positioning system with just a few simple parameter configurations. The positioning results can be rapidly transmitted back to PLCs and various types of controllers via popular interfaces such as Ethernet, serial port, Modbus, and I/O outputs. It can be applied in various industrial equipment and robotic systems that require high-precision positional information.

The entire system mainly comprises two parts: hardware and software.

Hardware: Vision Controller + Camera + Lens + Light Source + Light Source Controller

Software: ADTvision

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/129993799/02_98741cf5ec34dae7696286fc720ade20.png)

**Features**

1.  Supports cameras with various interfaces such as Gigabit Ethernet and USB.
2.  Supports area scan CCD cameras.
3.  Provides 1 opto-isolated input, 1 opto-isolated output, and 2 GPIOs.
4.  Supports online or offline inspection.
5.  Supports various common communication interfaces such as RS232 and Ethernet.
6.  Complies with OPC standards.

Xinmai provides customized vision controller solutions.
