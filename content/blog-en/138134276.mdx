****Product Features****

QC PXIe-7108

Number of Channels

8 Channels

Communication Modes

Supports RS232, RS422, RS485, DIP switch selectable

Signal Isolation Method

Input/Output Isolation

Baud Rate

RS232 MAX 1 Mbps

RS422/485 MAX 10 Mbps

FIFO

256 MBytes Transmit and Receive FIFO

Data Format

Parity Bit: None, Even, Odd, Space, Mark

Data Bits: 5, 6, 7, 8

Stop Bits: 1, 1.5, 2

I/O Address/IRQ: BIOS Assigned

Connector

Port0-7: VHDCI68-1

PXIe_CLK100

Supported

Interface

PCI Express Gen1 X1

Serial Port Protection

Onboard 15 KV ESD Protection and Surge Protection

System Support

Windows2000/XP/2003/Vista/Win7/Win10, Linux 2.4/2.6

LabVIEW, C, QT, etc.

Programming Method

Virtual COM Port, Supports Standard Serial Port Debugging Assistants
