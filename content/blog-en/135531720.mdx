Overview

The USB-XM2404 board is a high-speed, high-precision 4-channel synchronous data acquisition card with a USB 2.0 bus, offering up to 24-bit resolution and a 156 KSPs sampling rate. This card supports external triggering and software-timed triggering. Four analog signals and one external trigger signal (falling edge trigger) are input via the IDC-20 connector P1 on the board. 16 digital inputs (DI) and 16 digital outputs (DO) are input/output via the IDC-40 connector P2.

The analog input signals of the USB-XM2404 use differential input for ADC conversion. The AD conversion results are stored in large-capacity SDRAM (4M words). The AD chip itself includes a differential amplifier and a digital filter.

The USB host controller chip of the USB-XM2404 card uses CYPRESS's CY68013A. This board is a high-speed, high-precision data acquisition board with a USB 2.0 bus, supporting hot-swapping and plug-and-play functionality. It utilizes Field-Programmable Gate Array (FPGA) design to enhance reliability.

Drivers and Dynamic Link Libraries (DLLs) for Win10/Win8/Win7/Vista/XP, as well as programming guides (DEMO programs), are provided at the factory. Sample acquisition programs for VB/VC/LABVIEW are available, and a two-year warranty service is also provided.

Product Applications

The USB-XM2404 is a data acquisition card based on the USB 2.0 bus. It can be directly connected to a computer's USB port to form data acquisition, waveform analysis, and processing systems in various fields such as laboratories and product quality inspection centers. It can also form industrial production process monitoring systems. Its main application areas include:

Electronic Product Quality Inspection
Medical Testing
High-Precision Signal Acquisition
Zero Phase Difference Acquisition
I/O Control

Performance Features

Utilizes four independent AD7765 (156KHz) chips
High-performance, 24-bit Σ-Δ ADC, no missing codes
Signal-to-Noise Ratio (SNR): 112 dB min @ 156 KHz;
User only needs a single 5V power supply (in cases where USB port power is insufficient)
Bipolar input (±5V*, ±10V)
AD Trigger Modes: Software-timed trigger, external trigger (falling edge trigger)
High-precision 4.096V reference voltage
Hardware-integrated digital filter
4-channel analog differential inputs, with software selection for simultaneous operation of any number of channels
Sampling frequency is software-configurable (range: 1.95KHz～156.25KHz)
Power management chip automatically determines whether to use external power or USB bus power
External power reverse connection protection: if the external power supply is connected in reverse, the diode will conduct forward, and the onboard 1A fuse will provide protection
Onboard large-capacity memory (4M words) supports continuous acquisition for real-time monitoring
4KB user EEPROM space for convenient storage of user parameters
Board ID recognition (0-15), supporting multiple USB-2404 cards in one machine
16 digital inputs, 16 digital outputs. Outputs feature latching functionality and automatically clear to zero upon power-up
USB 2.0 bus, CYPRESS FX2 low-power host controller chip
GPIF (General Purpose Interface) for data transfer
Provides drivers and dynamic link libraries for Win10/Win8/Win7/VISTA/XP
Dimensions (excluding casing): 91(W) × 139(L) (mm)

\* indicates factory default settings
