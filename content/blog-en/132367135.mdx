The structure of the distributed data acquisition system is shown in Figure 2.1, primarily consisting of a sensor network, a data acquisition and transmission subsystem, a data aggregation platform, and a data acquisition system control terminal.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/132367135/01_574a1f164321dbe734633b97b5de5cdb.png)

1) Sensor Network: The sensor network includes multiple sensors and analog-to-digital converter devices. After sampling the signals to be acquired and converting them into digital signals via analog-to-digital conversion, they are sent to the data acquisition and transmission subsystem.

2) Data Acquisition and Transmission Subsystem:
Upon receiving digital signals sent by the sensor network, the data acquisition and transmission subsystem transmits them to the FPGA for data processing. The data acquisition and transmission subsystem communicates via Ethernet to complete the reception of commands and the transmission of data.

3) Data Aggregation Platform:
The data aggregation platform sends commands via an Ethernet interface and receives sampled data from the subsystem. Furthermore, the data aggregation platform communicates with the data acquisition system control terminal via an Ethernet interface to execute commands from the terminal and upload sampled data.

4) Data Acquisition System Control Terminal:
The data acquisition system control terminal communicates with the data aggregation platform via Ethernet to control the entire distributed data acquisition system, view its operational status, and analyze and display sampled data.
