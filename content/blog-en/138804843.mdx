Smart running tracks, launched by our partners and based on the IoT-3288A smart terminal motherboard, represent a healthy exercise method leveraging intelligent technology. Utilizing technologies such as the Internet of Things (IoT), facial recognition, and big data analysis, they monitor and collect data on people's exercise, display workout statistics including mileage and calorie consumption, and generate performance leaderboards, making public fitness more convenient, effective, and intelligent.

**Core Motherboard Solution Advantages**

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/138804843/03_21f86ed5b3a8d8a00c74419db331ee01.png)

The core motherboard of this solution is the IoT-3288A, based on the RK3288 platform, offering the following significant advantages:

1.  **Powerful Performance:** The IoT-3288A utilizes the Rockchip RK3288 quad-core processor, with a main frequency of up to 1.8GHz. It boasts powerful multi-threaded computing, graphics processing, and hardware decoding capabilities, providing robust core support for smart running tracks.

2.  **Mature System:** With over a decade of system R&D experience, Xinmai has refined its solutions through extensive testing. The seamless integration between the system, hardware, and application software ensures outstanding compatibility. Additionally, Xinmai offers system customization services, providing API reference code for system calls to fully support customers' upper-layer application (APP) development.

3.  **Reliable Quality:** The IoT-3288A features an immersion gold process, ensuring corrosion resistance for the core board's gold fingers. Secured by double studs, it has passed various stress tests and supports 7x24-hour stable, fault-free operation. It is reliable, market-proven, and more cost-effective.

**Motherboard Configuration**

**Core Configuration**

CPU: RK3288 quad-core processor, main frequency up to 1.8GHz
Operating System: Android 7.1/11.0
RAM: Standard 2GB (4GB optional)
Storage: Standard 16GB (8GB/32GB/64GB optional)

**Display Interfaces**

HDMI Output: 1 port, supports 1080P@60Hz, 4kx2k@60Hz output
LVDS Output: 1 single/dual channel, can directly drive 50/60Hz LCD screens
eDP Output: Can directly drive eDP interface LCD screens of various resolutions
Supports dual-screen asynchronous display
MIPI CAM: Reserved MIPI CAM interface, supports up to 13-megapixel

**Expansion Interfaces**

USB: 7 x USB2.0 ports
Serial Ports: 1 x TTL, 2 x RS232, 1 x Bluetooth multiplexed serial port, 3 x GPIO, 1 x ADC
Storage Card: Supports SD card

**Network Support**

Wireless Network: Built-in WiFi; Bluetooth 4.2
Wired Network: Supports 10/100M adaptive Ethernet
Mobile Network: Supports PCI-E interface for 4G internet access and calls
