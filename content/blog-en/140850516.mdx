Xinmai VS Weintek, close partners of **CODESYS**, have once again launched a significant solution. It not only supports IoT communication protocols like MQTT / OPC UA, which facilitate digital factory transformation, but also enables a control solution where HMIs can replace PLCs at the field execution layer.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/140850516/01_54c49b851b8b924dd4978754e5e43b8b.jpeg)

**Distributed I/O Application**

In factory automation, whether for product manufacturing, processing equipment, or entire flexible production lines, designers typically utilize HMIs, PLCs, and I/O modules for development, testing, and program debugging to control various digital and analog components like sensors, motors, and signal lights.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/140850516/02_0f500271900eccaf5b288b987613c1c2.jpeg)

**One Machine, Multiple Tasks: Your HMI is Your PLC**

To meet market demand, we offer a programmable control solution that supports **CODESYS** integration, embedding PLC control logic into the HMI. This enables the HMI to control remote I/O based on **CODESYS**.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/140850516/03_620a4907e04adc1b4b3bbd8a3ef9ed77.jpeg)

**CODESYS Solution Advantages**

**1**

**Distributed Control**

In factory production lines, Weintek I/O modules can be installed closer to the signal distribution points. Under permissible network conditions, the distance between modules can reach approximately **100 meters**. When the production line needs to be expanded, only one network cable is required to add an additional module.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/140850516/04_df47a01d207280f48044e02518783704.jpeg)

**2**

**Weintek Function Library**

To facilitate user programming, Weintek **independently develops and provides common function blocks for free**, such as motion control function blocks, PID control function blocks, and more. These function blocks not only simplify your programs but also improve readability.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/140850516/05_07be8af92a85030f9797caff17b5af65.jpeg)

**3**

**Remote Monitoring**

The HMI itself supports features such as the Weincloud cloud platform, EasyAccess 2.0 remote monitoring, Webview web browsing, and cMT Viewer centralized monitoring. These capabilities allow you to view data in real-time and maintain production line equipment.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/140850516/06_cacb7dcf3c7d3bed4d82a5889d9b7c01.jpeg)

**4**

**IIoT Communication Protocols**

IIoT support will also be one of Weintek's core competencies. While the Industrial Internet of Things (IIoT) in the PLC market has been developing slowly, using the HMI as a carrier allows all device data to be seamlessly integrated and connected to cloud platforms with ease.

![图片](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/140850516/07_8729e383724d48bc52a8c244828be6ba.jpeg)

**Application Example**

Xinmai's Shenzhen office precisely utilizes the HMI's **CODESYS** functionality combined with independently developed remote I/O to achieve automated control of all lighting, air conditioning, power, and other equipment.

Xinmai provides CODESYS-related hardware and software customization services. Among these, Allwinner T3, A40i, and RK3568 + CODESYS are all readily available and mass-production validated solutions.
