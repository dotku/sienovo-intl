### Computing and I/O Interface Module

1 Introduction

1.1 Product Overview

The computing and I/O interface module is primarily composed of the Fudan Microelectronics JFM7VX690 FPGA, domestically produced Ethernet transceiver YT8521, domestic BMC chip GD32F450, domestic CPLD chip EF2L45BG256B, and domestic memory components, designed in a standard 6U VPX form factor.

This computing and I/O interface module is mainly used in applications requiring high-speed, compute-intensive processing, such as radar signal processing and data processing. It provides a 10G Ethernet interface for service data and Gigabit Ethernet interfaces for management data. As a domestically developed, independently controllable solution, this high-performance, low-power, and highly reliable computing interface module can be widely applied in \*\*\*, \*\*\*, scientific research, communications, and other fields.

2 Key Features and Performance

● High integration with dimensions of 233.35mm × 81.5mm × 25.1mm;  
● Onboard DDR3 memory with total capacity of 6GB;  
● PCIe interface: PCIe 3.0 ×8;  
● Provides 2 Gigabit Ethernet interfaces and 1 MPO 10G optical fiber interface;  
● Onboard industrial-grade FPGA JFM7VX690 from Fudan Microelectronics;  
● 5 RS422 interfaces;  
● Supports online loading and upgrading of FPGA programs, as well as remote debugging;  
● Capable of detecting presence information and monitoring system status;  
● Power consumption: ≤50W.
