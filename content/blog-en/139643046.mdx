****II. Product Features****

Supports 4-channel AHD HD input at 1080P*30FPS, 720P, D1, CIF resolutions, etc.; supports connecting IPC via network port;

Video files use H.265 & H.264 codecs, saving storage space and transmission bandwidth;

Supports 4-channel audio input, 1-channel audio output; 1-channel voice intercom; external speaker support;

Supports 1-channel CVBS video output (optional 1-channel VGA video output);

Supports manual or scheduled recording to 2 SD cards, with a maximum capacity of 512GB;

Supports 1 USB 2.0 interface for exporting video files and mouse operation;

Supports 1 SIM card for 4G communication and internet access; supports Beidou 3 / GPS dual-mode positioning; supports built-in Wi-Fi function;

Supports emergency alarm function and 4-channel switch signal input, 1-channel switch output, 2-channel RS232 serial ports (optional 1-channel RS232, 1-channel RS485);

Built-in AI intelligent algorithms, supporting ADAS, DMS, BSD, 360°;

Supports mobile APP, client software, and platform software for video preview and parameter configuration;

Proprietary file system to improve disk read/write efficiency and ensure more stable recording;
