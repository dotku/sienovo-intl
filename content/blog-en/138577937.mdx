The ARINC429 Interface Simulation Card is a high-performance, low-power 429 bus protocol processing board. It complies with ARINC429 specifications, supporting 16 receive and 16 transmit 429 bus interfaces, enabling customers to quickly implement ARINC429 interface terminals. In addition, the product offers an IRIG-B time-code synchronization interface, suitable for multi-board collaborative environments. The card is equipped with standard driver libraries for various VxWorks and Windows versions, and provides powerful configuration application software, facilitating quick user adoption and product secondary development.
