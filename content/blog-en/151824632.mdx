The industrial control board solution, based on the Rockchip RK3588 processor and Fudan Micro JFM7K325T FPGA, combines the high-performance computing of ARM with the real-time processing capabilities of FPGA. It is suitable for industrial applications such as data acquisition, image processing, and AI inference. This solution supports 100% domestic localization, featuring high scalability and low latency.

**Core Hardware Configuration**

Processor and FPGA Collaborative Architecture‌

RK3588J‌: 8nm process, quad-core Cortex-A76 + quad-core Cortex-A55, 2.0GHz main frequency, built-in 6TOPS NPU, supports 8K video encoding/decoding and multi-channel AI algorithm acceleration.
JFM7K325T FPGA‌: Fudan Micro domestic FPGA, compatible with Xilinx XC7K325T, provides 326K logic cells, supports high-speed parallel processing and hardware acceleration.![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/151824632/01_0dc0d9f9ca204673879ba8698690e030.jpeg)

Interfaces and Expandability‌

Communication Interfaces‌: Supports PCIe Gen2 x8 (3.2 GByte/s bandwidth), Gigabit Ethernet, RS422/RS485, CAN bus, etc.
Expansion Interfaces‌: FMC (HPC) daughter card interface, connectable to ADC/DAC, optical fiber modules, etc., supports applications such as SDR and radar signal processing.
Video Interfaces‌: Supports HDMI input/output, MIPI Camera, SDI video processing (optional).

Storage and Cache‌

Onboard DDR3/DDR4 memory (max 4GB), supports high-speed data caching; optional 32GB LPDDR4X expansion.
Typical Application Scenarios
Industrial Vision and AI Inspection‌
Supports models such as YOLO, Faster RCNN, enabling real-time object detection and tracking, suitable for defect detection, intelligent sorting, etc.
High-Speed Data Acquisition and Processing‌
Achieves multi-channel sensor data synchronous acquisition (e.g., radar, IF signals) via FPGA, combined with high-speed PCIe transmission to the host.
Real-time Control and Communication‌
Supports EtherCAT master/slave stations, multi-channel RS485/232, suitable for industrial automation and robot control.![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/151824632/02_eb08bb588afb41f689d2638e754ff361.jpeg)
Technical Advantages
Domestic Localization‌: Core chips (RK3588, JFM7K325T) and peripheral components have all achieved domestic substitution.
Heterogeneous Computing‌: ARM handles complex algorithms, FPGA handles real-time tasks, improving system efficiency.
Low-Power Design‌: 8nm process and dynamic power management, suitable for edge computing scenarios.
Development Support
Software Ecosystem‌: Supports Linux/Android systems, provides FPGA low-level drivers, PCIe DMA test demos, and custom algorithm development.
Expansion Compatibility‌: Can be paired with FMC daughter cards (e.g., ADC, DAC) or AI accelerator cards, flexibly adapting to different requirements.
