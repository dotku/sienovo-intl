#### **CODESYS Control Linux ARM SL**

A Soft PLC compliant with the IEC 61131-3 standard and suitable for ARM & Linux hardware platforms.

![](https://pub-048dcb96257f476697b113fcb5939cb9.r2.dev/blog/126839996/01_95a703d7e8ab6c947dd31a6efe93eead.jpeg)

**Product Description**

CODESYS Control Linux ARM SL is a CODESYS Runtime for ARM & Linux hardware platforms. The Runtime's functionality can be extended through the system's hardware properties. After installing the accompanying CODESYS Deployment Tool plugin in CODESYS IDE, users can install this product on the target controller's Linux operating system via CODESYS IDE. CODESYS Runtime will automatically start after each reboot. CODESYS Runtime supports running for two hours without functional limitations and without a license.

**Product Features**

CODESYS Runtime supports a rich set of I/O interfaces, such as local I/O or bus-based I/O, and an integrated IEC 61131-3 protocol stack. Fieldbuses can be configured directly in CODESYS IDE without the need for third-party tools.

· CODESYS IDE can program and debug the Runtime online.

· Supports logic control functions and web-based HMI functions.

· Supports operation of I/O systems and fieldbuses.

· This product supports CODESYS multi-core functionality.

For detailed information, please refer to CODESYS Online Help (CODESYS documentation: https://zh.helpme-codesys.com/).

The RK3568 chip is a mid-to-high-end general-purpose SoC, manufactured using a 22nm process technology, integrating a quad-core A55 processor and a Mali G52 GPU. It has a built-in independent NPU with 0.8 TOPS computing power. The chip features rich peripheral interfaces, supporting display peripherals such as HDMI, eDP, MIPI, and LVDS. It can connect to PLC programmable controllers, optoelectronic conversion modules, sensors, and other devices via interfaces such as GPIO, RS485, RS232, and ADC. In industrial applications, the RK3568 is primarily used as the processor for products such as IoT gateways, NVR storage, industrial panel PCs, industrial inspection equipment, and industrial control boxes. It is widely applied in scenarios such as industrial inspection, process control, data acquisition, control computers, and centralized IPC management.
