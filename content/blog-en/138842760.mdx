It is a main controller specifically designed for PXIe hybrid test systems, compliant with 3U 12HP PXIe specifications. This product features Intel® Core™ i3/i5/i7 4th generation high-performance processors and supports up to 16GB DDR3L memory. The system's PXI Express link configuration uses a common 4-port 4-lane mode, offering data throughput up to 8GB/s.

The CX786x provides rich and flexible I/O interfaces, including two DisplayPort interfaces capable of connecting two monitors simultaneously, two USB 3.0 interfaces for high-speed external devices, two Gigabit Ethernet ports, and four USB 2.0 interfaces for other external devices or USB-interfaced instruments. The product design has undergone rigorous testing and is mature for application, capable of operating stably and reliably for extended periods. It is widely applicable in industrial automation control and military computing fields.

Product Features:

· Supports Intel® Core™ i7-4700EQ 2.4 GHz/ i5-4400E 2.7 GHz/ i3-4100E 2.4GHz processors, with a maximum frequency of 3.4GHz in single-core Turbo Boost mode.

· Supports dual-channel DDR3L SODIMM 1600MHz memory, up to 16 GB.

· Maximum system bandwidth support: 8 GB/s.

· PXI Express Link Capability:

· Quad-lane configuration: x4 x4 x4 x4.

· SATA Storage Configuration: 500 GB (or larger) HDD or 240 GB (or larger) SSD, supports AHCI mode.

· I/O Interfaces:
                    2x USB 3.0 interfaces
                    2x Gigabit Ethernet ports
                    4x USB 2.0 interfaces
                    2x DisplayPort display interfaces
                    Trigger I/O interface: for PXI trigger functions
                    1x RS-232/422/485 D-SUB9 interface

System

· Processor: Intel® Core™ i3/i5/i7 4th generation high-performance processor

· Chipset: Intel® QM87 Express Chipset

· Memory: DDR3L SDRAM up to 16GB

· PXIe Express Bus: Supports 4-port 4-lane link mode

· DP Interface: 2x DP interfaces (resolution up to: 3840x2160@60Hz)
