#### Introduction

We operate in an [SoC](https://so.csdn.net/so/search?q=SoC&spm=1001.2101.3001.7020 "SoC") mode, meaning we perform model compilation, quantization, and program cross-compilation on an x86 host using tpu-nntc and libsophon. For deployment, the compiled programs are copied to the SoC platform (1684 development board / SE [microserver](https://edu.csdn.net/cloud/sd_summit?utm_source=glcblog&spm=1001.2101.3001.7020 "microserver") / SM module) for execution.

Note: All the following operations are performed on an Ubuntu 20.04 system. Of course, Ubuntu 18 and 22 are also acceptable, as we primarily use the official Docker environment for configuration.
